Adding Cash Payout

The Cash Payouts window is as shown.

Note: The provision to capture the customer code is activated  by checking the parameter Display Customer Field in Cash Payouts/Receipts under Customer category.

 

Document Prefix: Select the document prefix.

Document No.:Starts with 1 for each series and gets incremented automatically.

Reason Code:Select a valid Reason Code, which is catalogued in general lookup, by pressing F2 key.

Narration: Enter the text quoting the reason for the payout.

Against Credit Note: This field is enabled if the Reason Code selected is Credit Note. Select this option if the payout is made against a credit note. For more details click here.

On selection of this option, two text boxes appear for entering Credit Note Prefix and Credit Note Number respectively.

 

Srl No.: The Serial Number is displayed

Mode: Select the mode of payment from the drop down list.

Code: Select the currency code from the drop down list.

Description: Enter the optional description, if any.

Units: Enter the number of units of the currency.

Exchange Rate: The predefined rate of exchange of the currency is displayed automatically based on the currency code selected. For example, in the case of Rupees, the exchange rate is 1.

Value: The product of Units and Exchange Rate is displayed here.

Amount: Displays the total amount of cash to be paid out.

Add: Click to add new details of Cash Payouts

Edit: Click to edit the entered details of Cash Payouts

Delete: Click to delete the entered Cash Payouts details

Reprint: Click to reprint the Cash Payouts details

Ok: To accept the details keyed in. Once you click Ok, it will give you an option to take the print out of the details entered.

Cancel: To undo all the details keyed in.
