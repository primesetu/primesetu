Sales Personnel

This option is used to store salesman information/ details like salesman type and address.

How to reach: Catalogue > Sales Personnel

The sales personnel window is displayed. In this window you can execute the following functions.

Add: To record a new Sales Personnel. Click here for more information.

Edit: To edit the details of an existing Sales Personnel. Enter the Code to display the Sales Personnel details. Make necessary changes  and click Ok to confirm modification.

Delete: To delete the details of an existing Sales Personnel. Enter the Code to display the Sales Personnel details. Click Ok to confirm deletion.

Form: To accept/ enter the details of a Sales Personnel.

Grid: To display/ list the Sales Personnel code and name

The navigation buttons help in viewing the records in sequence, in both, Form and Grid window. In Add mode these buttons are not activated.

Click the relevant button to display First, Previous, Next and the Last record.

Exit: To quit this option
