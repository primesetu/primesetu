Editing Item Master

The Item Master Entry window is as shown.

Select the data edit format as Classic View or Grid View. If you select Classic View, data edit form will accept details of a single item. In Grid View you can edit details of one or multiple items. You can use cut/ copy and paste to fill the columns. If the updated records are available in MS-Excel sheet, with the same order of the fields, then the whole data can be copied and pasted into this grid.

Fields to display: Select the fields to be displayed for editing item details.

Unselected fields: Displays the fields that are not selected for data edit.

Selected fields: Displays the fields that are currently selected for data edit. This list is based on the values selected during installation or on the settings you have chosen on this window and saved earlier.

In Unselected fields click to select the fields you want to add for data edit. Multiple selections are possible. Click to move the selected fields to the Selected fields list. You can use to remove the field names from the Selected fields list. You can rearrange the order of the fields in the Selected fields list using and .

Save field selections: Click to save the selections you made. If you do not save the selections, the fields list will be available only for the current session. Even if you save these selections, they can be changed in subsequent sessions, if required.

Click Item details to display the tab. Alternatively, press Alt+3. A message is displayed as shown.

 

Yes: To display data in Item details tab.

No: To display Item details tab without data. You may choose to apply filter to select required item details.

The Item details tab is as shown, if you have selected Grid View. Non editable fields are displayed with a grey background.

Note: Click here to view a sample of this tab, if you have selected the Classic View. Use and to view the previous and next records respectively.

Data Filter: The default selection is No Filter. If you select Apply Filter, the Selector window is displayed. Click here to know more about how to apply filter.

The item records that can be edited are displayed in the grid. The non editable fields are displayed with a grey background. Make necessary changes.

Replace Data: Select to change selected field values to the specified ones for all items displayed. When you click this, the Replace Data window is displayed. Click here for more details.

Ok: To save the item details modified

Cancel: To undo all the changes made

Print: To print the item details modified in the current session

 

Note:

1. You can browse through the tabs and make changes, if required. Either click the corresponding tab name or press Alt+1 for View and Alt+3 for Item details.

2. During data entry, if you want to choose any field value that is already catalogued, press F2. The Select Codes window will be displayed, a sample of which is shown.

Make necessary selections and click Accept Codes.
