Adding Payment Mode

The Mode of Payment Catalogue window is as shown.

 

Mode of Payment: Select the mode of payment from the list. Various modes possible are Cash (local or foreign currency), Credit Card, Discount Card, Gift Coupons, Advance Receipt, Cheques and Credit Note.

If you have selected Cash, Credit Card, Discount Card or Gift Coupon, certain fields corresponding to these payment modes are displayed. If mode of payment selected is Advance Receipts, Cheques or Credit Note, you just need to specify if the selected payment mode can be used at POS.

Common fields for all payment modes are Payment Code and Enable in POS Transaction.

Payment Code: Enter a meaningful code for the currency being catalogued.

Enable in POS Transaction: Select to indicate that the instrument can be allowed for use at POS.

Ok: To save the payment mode details.

Cancel: To undo the values chosen. Click this button a second time to cancel the functionality chosen.
