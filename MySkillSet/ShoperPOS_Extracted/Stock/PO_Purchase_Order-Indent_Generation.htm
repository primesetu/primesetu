Purchase Order/ Indent Generation

This option addresses the requirement of the generation of Purchase Order at HO, distributor and POS, based on business requirements. You have the flexibility of generating a PO based on:

  • Stock Number

  • Class1

  • Class2

  • SubClass1

  • SubClass2

  • Size

  • Analysis Codes

  • Any combination of the above attributes

By default, PO generation is based on stock number. However, you can change this and rearrange the attribute columns according to the requirement.

Other features include:

  • Specification of delivery dates for the PO or for individual items

  • Specification of order quantity for each item separately

  • Specification of order quantity size wise for single

The item rate in a PO based on stock number can be modified, provided the parameter Allow PO/Indent SKU Rate Alteration is enabled.  If the details are based on any other attribute, then you can enter the rate in the grid.

How to reach: Stock > Purchase Order > Generation

The Purchase Order/Indent Generation window is displayed. In this window you can execute the following functions.

Add: Click to create a new purchase order. Click here for more information.

Edit: Click to modify an existing PO.  Select Type, Prefix and enter the Number to display the existing purchase order. Follow the steps as in adding a new purchase order to make necessary changes.

View: Click to see an existing PO. Select Type, Prefix and enter the Number to display the existing purchase order.

Exit: To close the window
