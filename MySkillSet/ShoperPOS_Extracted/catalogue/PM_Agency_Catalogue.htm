Agency Catalogue

The Agency Catalogue window is as shown.

 

Agency ID: Enter ID of the agency.

Agency Name: Enter appropriate description for the agency.

Account ID: Enter the account number/ code of the agency.

Online Authorisation: Select if the agency allows online submission using Credit Card Authorization terminals.

Credit Card Honoured: Select if the agency honours and services the credit cards used during billing.

Note: An agency can honor either Credit Card or Gift Coupons. But not both.

Gift Coupons: Select if gift coupons accepted in billing are serviced by the agency.

OK: To create the new Agency Catalogue entry.

Cancel: To undo the values entered. This is also used to close the window.
