Stock Number Generation Methodology

This option is to design the pattern for stock numbers that are to be generated by Shoper 9 during item master addition. Various parameters that help in defining the pattern of stock numbers are to be provided.

Shoper 9 supports two standards for stock number generation:

  • Non-GS1 Standard—for following user specific numbering system

  • GS1 Standard—for following the global numbering system

How to reach: Setup > General > Stock Number Methodology

The Stock Number Generation Methodology window is displayed.

Select Non-GS1 Standard or GS1 Standard. Click the links to learn more about the corresponding methodology.
