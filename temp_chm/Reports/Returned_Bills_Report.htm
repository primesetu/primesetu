Returned Bills Report

The Returned Bills Report provides information of sales return made for the selected date range. It can be generated based on transaction type and grouped by vendor and/ or Consignment/ Non Consignment Items. You can generate the report either date-wise or reason-wise. The report can print item codes or descriptions, according to your requirement.

How to reach: Reports > Sales > Returned Bills

The Returned Bills Report window is displayed.

Date Range: By default, the From and To dates are the current Shoper date. You can enter any date range but the To date cannot be greater than the Shoper date.

Transaction Type: Select Both (Cash/Credit), Cash Transaction or Credit Transaction to include the corresponding transactions.

Report Criteria: Select Bill Returns - date Wise or Reason Wise to group the corresponding transactions.

Reason: This gets activated when Reason Wise option is selected as Report Criteria. Choose All or Specific. If you choose Specific, select the reason from the list of reasons accepted during the time of return.

Break-Up:

Vendor: Select if the report has to be grouped vendor wise. Choose All or Specific. If you have chosen Specific, select the vendor from the list.

Consignment / Non Consignment: Select to group the consignment items and non consignment items in the report.

Choose Code or Description to specify which value should be printed in the report.

View: To view the report

Exit: To close the window
