Item Tracker

This report allows you to track stock item using stock no., serial number, etc, and get details of different transactions recorded for the item.

How to reach: Sales > Item Tracker

The Item Tracker window is displayed.  

   

Search On: Select the attribute based on which the stock item will be searched. The attributes provided  based on which items can be searched are Serial No./IMEI No., Stock Number, Style, Party Name and Party Code.

Search value: Enter a value for the attribute to search the item

Transactions: Select All to search for all transactions of the selected stock item or only particular transaction from the list provided.

Date Range

From: Select the beginning date for the period to search from

To: Select the end date for the period

Search in Head Office: Select to track the item details from HO data, if enabled.

Search: To start the search and display the report.

Display/Search Selection: Use the check boxes to add/remove columns to the search results. In the value field, type the condition to further narrow down the search results.

Exit: To close the window.
