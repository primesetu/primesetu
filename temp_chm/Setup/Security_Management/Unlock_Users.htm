Unlock Users

When the user enters wrong password more than the maximum allowed count of invalid password configured in password configuration, then the user gets locked. The Super User will be able to unlock such users using this window.

How to reach: Setup > Supervisory Functions > Security Management

On clicking Unlock Users, the Security Management – Locked Users window is displayed.

User ID: Displays the user ID.

User Name: Displays the user name.

Status:Displays the status of the user. Un-check to unlock the user.

Ok: To save the applied changes

Cancel: To clear the applied restrictions

Exit: To close the Security Management window
