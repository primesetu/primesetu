Entry Panel Column Configuration

This option is used to configure the columns in the item details and direct entry grids of the billing, slips and goods outward screens.

How to reach: Setup > General > Entry Panel Column Configuration

The Entry Panel Column Configuration window is as shown.

Transaction Type: Select the type of transaction.

Accept Details

Displays the columns in the direct entry grid of the billing, slips and goods outward screens. The Accept Details tab is selected by default.

Column Details Configuration

The details of the columns to be displayed in the direct entry grid are displayed here.

  • Configure/ change the caption and the width of the columns

  • Rearrange the sequence of the columns by either clicking on the row number, dragging and dropping or, by clicking the column caption/ column width and moving the row using the and arrows.

Show Column:  Select the columns to be displayed (visibility) in the items details grid.

Value Type:  Displays the type of value such as Date, Text, Currency, Combo box, etc.

Rec Id: Displays the Rec Id configured in the General Lookup.

Add New Column Details: This option is used to configure custom columns in New Bill, Sales Return (with and without reference), Inwards and Outwards.

The configured custom columns are displayed in the corresponding transaction UI and the values captured are updated in the database.

Caption: Enter the desired caption to be displayed in the transaction grid.

Value Type:  Select the type of value such as Date, Text, Currency and Combo

Column: Select the column to be added from the list

Rec Id: Enter the Rec Id configured in the General Lookup when Combo is selected.

Add: Click to add the configured column

Display Details

Displays the columns in the item details grid of the billing, slips and goods outward screens.

Save: To save the configuration setting

Clear: To reset the selections

Exit: To quit the option
