Personnel Incentives Definition

Incentives

This option is used to set personnel incentives for the employees doing sales. Incentives can be defined for all sales staff or specified people. Incentives can be for selected period and selected item classifications, based on cumulative sales value or quantity for the period, specified bill value or highest bill value in the period.

How to reach: Catalogue > Personnel > Incentive Definition

The Personnel Incentives window is displayed. In this window you can do the following.

Add: Click to define a new incentive scheme. Click here for more information.

Edit: Click to edit an existing incentive scheme. Select the Incentive Name. The catalogued incentive details are displayed. Make necessary changes and save.

View: Click to view an existing incentive scheme. Select the Incentive Name. The catalogued incentive details are displayed.

Delete: Click to delete an existing incentive scheme. Select the Incentive Name. The catalogued incentive details are displayed. Click Ok to confirm deletion.

Exit: To close the window
