Dashboard Reports

Dashboard reports are available in Shoper 9 opening screen. These reports can be configured as per your requirements. View up to five consolidated reports at a time, in graphical representation.

How to reach: Click the link next to Dashboard on Shoper 9 POS Info Panel.

The Dashboard window is displayed.

 

Summary: Click to view the summary information.  Click here for more information.

The Summary window is displayed.

Close: Click to exit

 

Configure: Click to set the display configuration for the report. Click here for more information.

The Configuration window is displayed.

Chart Area: Six charts/ graphs can be configured and displayed at any given time.

Report: Select the time period from the drop-down list.

Value Type: Select either quantity or value to be represented in the chart/ graph.

Chart Type: Select the type of chart to be displayed.

Display Type: Select the type of display, either 3 dimensional or 2 dimensional chart/ graph.

Show Graph Values: Select Yes to display the graph values.

Ok: Click to save the settings

Exit: Click to close the configuration window

 

Exit: To close the window
