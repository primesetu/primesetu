The Mailing Address window is displayed.

The number of address columns on this screen depends on the number of parameters configured under the category Types of Address in General Lookup. The default is Billing Address. You may modify the existing one or add any other address parameters as required. In the given Mailing Address window, the default parameter is modified to Mailing Details.

The captions Address1, Address2, Address3, Address4, Address5 or Default Address can be changed by setting the corresponding system parameters under the category Customer, if required.

The Customer Code and Customer Name are displayed in the Code and Name fields respectively. The Code and Name displayed cannot be altered.

Enter the remaining address details as required.

You may set this address or one of the addresses (if you have more than one parameter configured), as Default Address. This address would be used for all correspondences.

Ok: To confirm and save the mailing details of the customer

Cancel: To clear the details entered

Adding Customer

The Customer Catalogue window is displayed. This window has three tabs, Form, Retail Details and Additional Details.

 

General Details

Customer Code: Enter if the system parameter Customer Code User Assigned under the category Customer is enabled. If the parameter is not enabled then the customer code is generated automatically.

Customer Name: Enter the name of the customer. The same can be entered in the Mailing List form which is then displayed here automatically.

Customer Price Group: Select from the list already catalogued in the General Lookup.

Mailing List Info

Add: On clicking this option, the Mailing Address window is opened to enter the contact details of the customer.

Classification Details and Profile Details

Shoper 9 provides five customer classification fields and five customer profile fields. The captions that are displayed like Religion, Age Group, Ethnicity etc., are user defined. Also, you have the freedom to use none, few or all of these. These settings have to be done during installation or by using System Parameter option under Setup. The values for these customer attributes can be catalogued under General Lookup.

Details of Shoper

The information under this heading is applicable if the customer whose details are being catalogued is also a Shoper user. These details help in making the data exchange easier.

Comp Code: Enter the Shoper company code

Environment: Select Retail or Distributor depending on the environment in which the customer operates.

Flat File Format: Select the appropriate format from the list in which the flat file has to be created.

Delimiter: Enter the delimiter to be used for creation of data

Buying Factor/ Selling Factor: Enter the factor (e.g., 10) to be applied as a rate or amount on different prices while creating flat files for any type of transaction. The factor set here is used in the option Setup > General Setup > Flat File Value.
