In the Buy Items grid, select the first set of items.

Quantity: Specify the quantity for the first set of items to avail the offer.

Set No.: Auto assigned depending on the condition selected in the next column.

Condition: Select And or Or to specify the criteria. Last row will always have End. When you select And, the offer is applicable if the buyer picks the items in both the current row and the next row. Selecting Or indicates the offer is applicable if the buyer picks the item either in the current row or the next row.

If the condition selected is And the item in the current row and next row will be in two different sets and the sets will be numbered accordingly. If the selection is Or, the items will be in the same set and both rows will be assigned the same set number.

In case you are selecting Or, the quantity for both item combinations in the set will remain the same and hence the quantity column in the next row will be non editable.

Notice that set numbers are assigned to the rows based on your selection in Condition.

Item Rate/Worth: Select if the definition is based on item rate or item worth. When you select this, the possible comparison operators that can be used to specify the selection criteria are filled in the list. The choices are Rate =, Rate >, Rate <=, Rate Range, Worth =, Worth >, Worth <= and Worth Range. Select the appropriate one and the Buy Items grid changes accordingly.

Items at Fixed Value

This promotion type helps you to define schemes that can be applied as fixed discounted value on specified items at the item level during billing. Using this sales promotion type, you have a choice to define sales promotions on items or item combinations.  

The details to be captured can be divided into two parts, namely, Buy Items and Discount Details.

Buy Items

This section allows you to select the items to be sold with discount.

The Buy Items section is as shown.

Bundle Items: Select if the definition is based on item combinations. On selecting this option, some more columns are added to the Buy Items grid, where you need to specify the required item combination by grouping the items into different sets. Click here for more information.

Configure: Click to configure the options for item selection. Click here for more information.

The Buy Items grid displays the item classification columns as configured. On pressing F2 in any item classification column, the corresponding browse window is displayed with the list of catalogued values to select from.

Note: In the Buy Items grid:

   · You can press F2 anywhere in the grid, irrespective of the order in which the item classification columns are present, to select values of any attribute. Click here to learn more about value selection.

   · You have the flexibility to select either one or multiple attributes in a row, to specify the required items.

   · If Stock No. of an item is selected, the remaining item classification columns for that stock number will be filled automatically.

Discount Details

This section allows you to capture discount details for the items selected in Buy Items grid. The discount details captured vary according to the selection of items (i.e., based on item combination or not) in Buy Items grid.

The Discount Details section is as shown.
