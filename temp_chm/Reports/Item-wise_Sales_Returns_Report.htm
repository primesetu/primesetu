Item-wise Sales Returns Report

The Item Wise Returns Report provides information on items that were returned for various reasons during a given period. Grouping of the report can be done based on Super Classification 1, Super Classification 2, Items Classification 1 or Items Classification 2 for the goods returned. Vendor selection enables you to analyse items returned by different vendors.

How to reach: Reports > Sales > Item-wise Sales Returns

The Item Wise Returns Report window is displayed.

Report Grouping: Select Department, Buyer, Product or Brand to group the lines in the report.

Item Details: Select All Items, Consignment Items or Non Consignment Items to include the corresponding details in the report.

Group On Consignment Items: Select to group all consignment items.  

Report: Select Detail or Summary. Detail report will show detailed item level information along with reason for return. Summary report shows only the values like item class, return quantity and return value.  

Date Range: By default, the From and To dates are the current Shoper date. You can enter any date range but the To date cannot be greater than the Shoper date.

Vendor Details: Select All Vendors or Specific Vendor. If Specific Vendor is chosen, select the vendor from the list.

Group On Vendor: Select to group the report vendor wise.  

View: To view the report

Cancel: To cancel the selections made

Exit: To close the window
