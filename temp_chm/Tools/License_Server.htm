License Server

You can designate a selected License Server as the license provider for the chosen product. The License Server service needs to be running while doing the activation or reactivation process.

How to reach: Start > Programs > Tally License Server > Tally Admin

The License Server window is as shown.

Product: Select the required product from the list.

License Server(LS) list: Displays the details of the license servers available. The details include the System Name, IP Address, Port number, Service Name, Version and License Status of the License Server.

Show All Licenses in LAN: Select to display the details of all the compatible License Servers available within the network.

Select the License Server that has to be designated as the license provider.

Activate: To activate a Single Site or Multi Site license. Click here to learn about license activation process.

Re-Activate: To reactivate a Single Site or Multi Site license. Click here to learn about license reactivation process.

Refresh LS list: To refresh the list of license servers.

Connection List: Displays the list of computers connected to the license server.

Proxy Configuration: To configure the proxy server to connect through the Internet, if required. Click here to learn more.
