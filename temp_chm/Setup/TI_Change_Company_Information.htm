Change Company Information

After setting up Tally Interface Mapping, you might need to change the Tally company information like server or port details or change the name of the Tally Company (in case you have renamed the company in Tally) to which the data need to be posted. This option can be used for the same.

How to reach: Setup > General > Tally Interface Utilities > Change Company Information

The Tally Interface Utilities – Change Company Information window is displayed as shown.

All existing mapping information for the selected showroom is displayed in the grid. Select the mapping for which the company information has to be altered.

Ok: To proceed with the change information process

Exit: To abort the change information process

The Change Company Information window is displayed as shown.

The existing Tally Information is displayed in the grid.

Server IP: Enter revised the IP address of the machine where Tally is installed.

Port No.: Enter the revised port number where Tally is configured.

Company: Select the Tally Company Name, to which Shoper data has to be posted. It is not advisable to choose a new Tally Company here. This option has been provided to manage the situation where the name of the existing mapped company has been changed in Tally or where the Tally Company has been split.

Apply: To save the changes. A confirmation message is displayed, if the changes are saved successfully. The changed data is displayed in the New column.

Exit: To abort the changes.
