Menu Access Control – User

This option helps to restrict users from accessing relevant menu options.

The Security Management – Menu Access Control user window is displayed.

User ID: Enter / Browse the User ID for which the permissions have to be restricted.

User Name: The name corresponding to the ID is displayed.

Company Code: Enter / Browse the Company Code for which the permissions have to be restricted.

Company Name : The company name corresponding to the company code is displayed.

Note: To display Company Code and Company Name enable Activate company-wise restrictions in Manage Menu Access in Security Management – Configuration window.

Menu Descriptions: Displays the Shoper 9 menu tree. The options that can be currently accessed by the selected user are checked while the inaccessible items are unchecked.

You may click to expand all options, to shrink the tree structure, to select operations or double click to clear the selections made.

Ok: To confirm selection made

Cancel: To cancel the selection

Exit: To close the Security Management window
