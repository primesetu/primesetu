Customer Offtake Product-wise Report

The Product-wise report provides detailed information of sales made to different customers. This report is used to keep track of the items purchased by customers for a given period.

The report can be generated for

  • A specified date range.

  • Transactions pertaining to sales, sales returns, void sales, etc.

  • All or a selected customer.

  • All or selected product classifications.

  • All or selected product analysis codes.

  • Customers of all or selected customer classifications.

How to reach: Reports > MIS > Customer Offtake > Product-wise

The Customerwise Productwise OffTake Report window is displayed.

Date Range:  By default, the From and To are the current Shoper date. You can enter any date range but the To date cannot be greater than the Shoper date.

Transaction Type: Select the transaction types for which the report is to be generated. The types are Sales, Sales Returns, Misc. Issues, Void Sales, Void Sales Returns, Misc. Receipt.

Item Details: Select All or Range. If you have chosen Range, select the required classifications. The corresponding values for the selected item classification is displayed in the next box to select from. Highlight each classification and select All or Range. If you have chosen Range, select the required values from the list.

Customer Details: Select All, Specific or Custom to identify the customer list to be included in the report. If you have selected Specific, choose the list of customers to be included in the report. If you have chosen Custom, Customer Details grid is activated. In the grid, select All or Specific to identify the categories to be included in the report. If you have selected Specific, choose the categories to be included in the report. Highlight each of the selected categories and select All or Specific to include the corresponding values from the list on the right.

Select Code or Description to be displayed in the report.

Order By: Select CustomerCode, Qty (quantity) or Value to sort the report accordingly. Additionally, select Asc (ascending) or Desc (descending) to specify the sort order.

Show by MRP: Select MRP to include the MRP values in the report.

View: To view the report

Exit: To close the window
