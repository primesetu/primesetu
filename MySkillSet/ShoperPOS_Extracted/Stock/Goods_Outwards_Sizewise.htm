Goods Outwards Size-wise

Goods Outwards Size-wise refers to transactions such as purchase return, inter-showroom transfers, and miscellaneous issues based on size. You can use this option to record details of size-wise outward transactions.

How to reach: Stock > Goods Outwards Size-wise

The Goods Outwards Size-wise window is displayed. In this window you can execute the following functions, based on configuration.

Add: Click to do a new goods outwards size-wise transaction. Click here for more information.

Edit: Click to modify an existing goods outwards size-wise transaction details. Select Transaction Type, Doc. No. Prefix and Doc. No. to display the existing outward transaction document. Follow the steps as in the Add option to make necessary changes.

You can edit the documents only if the document date is same as Shoper date. When you edit a document, the existing document number is voided and a new document number with the changes is generated.

Delete: Click to delete an existing goods outwards size-wise transaction details. Select Transaction Type, Doc. No. Prefix and Doc. No. to display the existing size-wise outward transaction document. Verify the details in the selected document and click Ok.

You can edit the documents only if the document date is the same as Shoper date.

Reprint: Click to reprint an existing goods outwards size-wise transaction details. Select Transaction Type, Doc. No. Prefix and Doc. No. to display the existing  outward transaction document. Click Ok.

Exit: To close the window
