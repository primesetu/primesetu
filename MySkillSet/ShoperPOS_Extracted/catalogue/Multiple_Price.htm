Multiple Pricing

Businesses follow different pricing strategies. This option caters to the businesses using multiple prices for items.

In order to define multiple prices:

1. Set the System Parameter Multiple Prices Applicable under category Item Classification.

     System Parameter values indicate the following:

          0 – multiple prices are not applicable to any item

          1 – multiple prices are applicable to some items

          2 – multiple prices are applicable to all items

2. If System Parameter is set to 1, identify the class1 + class2 (product + brand) combinations for which multiple prices are applicable.

    • Use the option Product + Brand Combo Settings

3. Define multiple prices using the option Multiple Price Setting.
