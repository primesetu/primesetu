Bill-wise Sales Report

The Bill-wise Sales report provides sales information bill wise for the selected period, based on types of sales Product, Service or Both. Bill wise sales and sales returns are included in Detailed Sales report as well as Tax Wise Break Up report. Cancelled bills can also be included in both reports, if required.

How to reach: Reports > Sales > Bill-wise Sales

The Bill Wise Sales Report window is displayed.

 

Sale Type: Select the type as Product, Service or Both (Product/ Service).

Date Range: By default, the From Date and To Date will be the current Shoper date. You can enter any date range but the To Date cannot be greater than the Shoper date.

Detailed Sales: Select to view detailed bill wise information. The details include bill value, tax, deductions, discounts and modes of payment for each bill.

Tax Wise Break Up: Select to view the total tax collection based on Tax code defined for the Items. It gives the various tax components information from tax component 1 to 5. It generates a summary information report based on mode of payment and tax rate. The details include sale value, tax, deductions, discounts and modes of payment for each bill.

Include Void Sales: Select to include bills pertaining to cancelled bills.

View: To view the report

Exit: To exit the window

 

Note: The Bill Value does not include the Round off values because the round off is done at bill level and this report is based on Item Level.
