Designing Label for Standard Printers

Barcode labels can be created as per the requirements and printed using LaserJet, Inkjet, etc. You can set dimensions of the labels, constant text, include fields from item master table, images, etc. The designed barcode label template is saved as a file with the extension btf.

How to reach: Stock > Barcode > Design Label for Standard Printers

The Label Design for Standard Printers window is displayed.

Open the .btf file to load the selected label settings.

Font Name: Select a font name for the constant text/fields added to the label designer.  This is disabled in the case of a barcode label.

Font Size: Select a font size for the text. Font size plays a major role in deciding the size of the label.

Top, Left, Width and Height: On adding the selected controls, these boxes are populated and show the top, left, width and height of the control in the label designer. The values of Top and Left can be modified to relocate the control. Also, relocation of the control changes these values automatically.

Label Parameters:

Unit of Measurement: Select the unit of measurement from the list as Millimeter, Centimeter or Inches. Millimeter is selected, by default.

Height: Displayed as per the selection made in Font Size.

Width: Displayed as per the selection made in Font Size.

Apply Label Settings: To apply the specified height and width to the label.

Select Fields: Select the type of the control to be added to the designer, from the list. The field parameters you need to specify vary, with the type selected here. The types are Constant Text, Barcode Label, fields from item master table and Images.

Note: One or more of these types can be selected in the same label.

 

Add Item(s): To add the selected field in the label designer. Alternatively, use down arrow key and Enter at Select Fields.

Field Parameters: The values to be accepted here vary with the field type selected in Select Fields. Click the links to learn more about Field Parameters.

  • Constant Text

    If the field type selected is Constant Text, the Field Parameters group is as shown.

    Constant Text: Enter the text to be used in the barcode label.

  • Barcode Label  

    If the field type selected is Barcode Label, the Field Parameters group is as shown.

    Symbology: Select the symbology of the barcode to be printed. Six symbologies are currently available. To create small labels, Narrow symbologies equivalent to these six are provided. It is recommended to have bigger font sizes for the narrow labels to avoid readability issues caused by poor printing quality. The recommend font sizes are

    Font Size

    I2O5, I2O5-H, UPCEAN

    18 - 24

    Barcode Content: Select a field from the list. The value of this field will be printed as barcode. Generally, Stock Number is selected.

    Sample Text: The value of the field selected in Barcode Content is displayed and the sample of the barcode for that value is displayed in the label designer. As you change the sample text, the sample barcode changes accordingly. The value can contain a maximum of 32 characters.

  • Fields from Item Master table

    If the field type selected is Item Description, the Field Parameters group is as shown.

    Full Width: Select to print the complete field.

    From Left: Select to mark the starting position and number of digits from left side.

    From Right: Select to mark starting position and the number of digits from right side.

    Field Selected: The field in item master table selected under Select Fields.

    Starting Position: Enter the position from where the value has to be picked for printing.

    Sample Text: Enter a sample text here. The label designer will display all the digits that are entered starting from the left or the right (as selected).

    No. of digits: Enter the number of digits to be printed for the selected field.

  • Images

    If the field type selected is Images, the Field Parameters group is as shown.

    Image Path: Enter the image file name. Alternatively, click to browse for the image file and select it.

Note: If Batch / Grade / Location are enabled then the corresponding fields will be displayed in Select fields.

 

Apply Field Settings: To apply the parameters/ control attributes specified for the barcode label.

Delete: To delete an added control. Alternatively, you can double click the control to be deleted.

Clear All: To clear the values entered.

Exit: To close the label design window.
