Incentive Analysis Report

 This report provides the incentive details of sales persons.

How to reach: Reports > MIS > Incentive Analysis

The Incentive Analysis window is displayed.

Incentive Definition

All: Select to generate report containing all defined incentives.

Specific: Select to generate report for the incentives selected from the drop down list. Click … to browse defined incentive.

Sales Person

All: Select to generate report for all sales persons eligible for the selected incentives.

Specific: Select to generate report for the sales persons selected from the drop down list.  Only the sales person eligible for the selected incentives will be available in the drop down list.

Date Range: By default, the From Date and To Date will be the current Shoper date. You can enter any date range but the To Date cannot be greater than the Shoper date.

Group by

Incentive: Select to display report based on incentives.

Sales person: Select to display report based on sales persons.

Display Config. : Click to view Display details screen. Select the columns to be displayed in the report.

View: To view the report in spread sheet format.

Clear: To clear all the values entered

Exit: To close the window
