The Mailing Address window is displayed.

The number of address columns on this screen depends on the number of parameters configured under the category Types of Address in General Lookup. The default is Billing Address. You may modify the existing one or add any other address parameters as required. In the given Mailing Address window, the default parameter is modified to Mailing Details. The captions Address1, Address2, Address3, Address4, Address5 or Default Address can be changed by setting the corresponding system parameters under the category Customer, if required.

The Vendor ID and Vendor Name are displayed in the Code and Name fields respectively. The Code and Name displayed cannot be altered.

Enter the remaining address details as required.

You may set this address or one of the addresses (if you have more than one parameter configured), as Default Address. This address would be used for all correspondences.

Ok: To confirm and save the mailing details of the vendor

Cancel: To clear the details entered

Company Code: Enter the Shoper company code of the vendor

Environment: Select Retail or Distributor depending on the environment in which Vendor operates.

Flat File Format: Select the appropriate format from the list in which the flat file has to be created.

Delimiter: A semicolon is set as the default delimiter for creation of data files. You can change the delimiter, if required. This field will be disabled if the Flat File Format selected is either DOS Format or GUI Format.

Buying Factor/ Selling Factor: Enter the factor (Eg.,10) to be applied as a rate or amount on different prices while creating flat files for any type of transaction. The factor set here is used in the option Setup > General Setup > Flat File Value.

Tax Inclusive: You can opt for cost price of items to be inclusive of tax or exclusive of tax, depending on the vendor, while executing goods inwards under Stock > Goods Inwards. You can set the parameter Cost Price Inclusive of Tax under the category Inwards at Setup > General > System Parameters.   

Possible Parameter Values:

0 - For Cost Price to be tax inclusive irrespective of the Vendor

1 - For Cost Price to be tax exclusive irrespective of the Vendor

2 - To set Cost Price to be tax inclusive or exclusive depending on the Vendor from whom the purchase is being made

So, if this field (Tax Inclusive) is checked, then the goods inwards option will consider the items received from this vendor as Tax Inclusive when the parameter explained above is set as 2.

Vendor

This option is used to maintain Vendor details like mailing list information, vendor code, vendor type, TIN etc. Shoper 9 Point-Of-Sale maintains the mailing information captured in various catalogues commonly.

How to reach: Catalogue > Vendor

The Vendor Catalogue window is displayed.

 

Initially the window is in browse mode, by default and you can move across the entries in the catalogue using the navigation buttons in the window. You can also toggle between record-by-record view and grid view using the Form and Grid buttons respectively.

Note: Choose the required functionality by clicking the corresponding button located at the bottom of the screen.

Add: To record the details of a new vendor

Edit: To edit the details of an existing Vendor

Delete: To delete the details of an existing Vendor

Form: To accept/ enter vendor details

Grid: To display/ list the catalogued vendor code and vendor name

The navigation buttons help in viewing the records in sequence, in both, Form and Grid window. In Add mode these buttons are not activated.

Click the relevant button to display First, Previous, Next and the Last record.

Exit: To exit the vendor catalogue option

The vendor catalogue form is invoked from the corresponding option.

Vendor ID: Displays/ Enter vendor Id. An alphanumeric value can be entered

Vendor Name: Displays Vendor name which is entered in the mailing list window

Vendor Type: Select from the list of vendor types already catalogued in General lookup.

Vendor LST/TIN: Enter vendor's LST number or TIN

Vendor CST: Enter vendor's CST number

Mail List Info: Displays mailing information for the vendor ID, if entered already

Add: On clicking this option, the Mailing Address window is opened to enter the mailing details of the vendor

Partially Supply against Purchase Orders Allowed: Select to allow partial supply of goods against the purchase orders placed on this vendor

Details of Shoper

This section is used to enter the Shoper details of the vendor, if the new vendor is a Shoper user. These details help in data transfer. Click here to see the Shoper details to be entered.

Ok: To confirm and save the vendor details entered

Cancel: To clear the details entered
