Creation of a New Secondary Database

Select the option Create Secondary Database to proceed.

The database information is displayed.

Server Name: The name of the server is displayed.

Database Name: The name of the secondary database is displayed.

Previous: To go to the previous window

Cancel: To cancel the selected information for the secondary database

Next: To go to the next window

On clicking Next, the secondary database is created and the following window is displayed.

Server Name: The name of the server is displayed.

Database Name: The name of the secondary database is displayed.

Database Type: The database type, for example SQL Server, is displayed.

Finish: To complete the creation of the secondary database
