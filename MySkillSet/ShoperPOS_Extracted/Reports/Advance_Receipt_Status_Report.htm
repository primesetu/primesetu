Advance Receipt Status Report

This option provides details about the current status of the advance receipts recorded in the system. These advance receipts can be open advances, advance receipts consumed/ used in sales or advance receipts paid-out via cash payouts. This report also provides details of prefix, sales reference number, balance amount, used amount and other necessary details.

How to reach: Reports > Cash > Advance Receipt Status

The Advance Receipts Status window is displayed.

Date Range: By default, the From Date and To Date will be the current Shoper date. You can enter any date range but the To Date cannot be greater than the Shoper date.

Customer

Code: Enter the customer code or browse.

Name: Enter the customer name.

View in spread sheet format: Select to view the report in a spreadsheet format.

View: To view the report

Exit: To close the window
