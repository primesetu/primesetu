About Extensions Types

Generally, customised add-ons/ extensions are used in the base applications to perform specific functions as per requirement.

Shoper 9 Extension Framework supports two types of extensions, namely, Pre-update Extension and Post-update Extension.

  • Pre-update Extension – is executed before saving the corresponding standard Shoper 9 POS functionality

  • Post-update Extension – is executed after saving the corresponding standard Shoper 9 POS functionality
