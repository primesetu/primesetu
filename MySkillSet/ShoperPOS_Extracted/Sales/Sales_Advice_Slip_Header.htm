Sales Advice Slip Header

The header part of the sales advice slip is as shown.

Doc. Ref: The document reference and the number of the sales advice slip are displayed.

The current Shoper date is displayed, which is considered as the date of generation of the sales advice slip.

Customer: Enter the customer code. Alternatively, pressing F2 displays the customer Browse window from which the required customer code can be selected. The name of the customer according to the selection of the customer code is displayed in the next field.

Alt+M: To change the customer at any stage of generating the sales advice slip

Trial Date:Enter the date on which the trial of the item took place

Delivery Date: Enter the date on which the item was delivered to the customer

Sales Staff: Select the appropriate sales staff code from the drop down list

Remarks: Enter remarks, if any

Import: This option allows to import the details of the items to be included in the sales advice slip. The PDT Import window is displayed on selection of this button. There are two ways of importing details — from file or transaction

The PDT Import options appear as shown when transaction is chosen for import.

Import from: Select the mode of import of item details as File or Transaction. When Transaction is selected, the following values need to be entered.

Transaction:

Type: This allows to import items detail from various types of transactions. The available transaction types are 1100 - Purchase, 1200 - Transfer In, 1300- Sales Return, 2100 - Sales, 2200 - Transfers Out and 3200 - Stock Adjustments.

Bill Prefix: Enter the prefix of the selected transaction type.

Bill No: Enter the number of the document for the selected transaction type.

OK: Click to accept and save the entered details

Clear: To remove the entries made

Close: Click to close the screen

The PDT Import options appear as shown when file is chosen for import.

Import from: Select the mode of import of item details as File or Transaction. When File is selected, the following values need to be entered.

Field Template: Select the details of the items to be included in the sales advice slip. The available formats are Stock Number, Stock Number + Qty + Rate, Stock Number + Rate + Qty and Stock Number + Qty.

Note: The default delimiter of the formats is ~. You can set the default delimiter to your choice by changing the value of the system parameter Delimiter Character for PDT Import in Sales Transaction under the category Billing.

Path: Enter the path where the file containing the items is found. Alternatively, browse and select the filename.

OK: Click to accept and save the entered details

Clear: To remove the entries made

Close: Click to close the screen

Recall: This option allows to recall the details of the items to be included in the sales advice slip. This can be done by entering the transaction details in the opened window
