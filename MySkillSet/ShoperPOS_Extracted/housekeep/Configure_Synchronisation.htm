Configure Synchronisation

This option is used to configure the modes of communication that can be used for synchronisation between the showroom and the corresponding head offices or principals. The same option is useful in configuring the modes used for communication between POS locations.

How to reach: Housekeeping > Synchronisation > Configure Synchronisation

The Configure Synchronisation window is displayed. In this window you can configure the following.

  • HO Communication Details: Click here for more information.

In case you are a multi-brand outlet doing business of items from multiple principals and you have a MBO Configuration file present in the In folder, the Communication Setup - Principal Company window opens. Click here for more information. If a principal re-sends the configuration details, opening this window displays a message that a new configuration file is found and asks for confirmation to overwrite the existing  item classifications. It overwrites the existing configuration, if chosen and imports newly received classifications, if any.

Note: If a configuration file is present in the In folder, even if multi-brand outlet is not enabled, the Communication Setup - Principal Company window opens. On saving the item classification, multi-brand outlet is automatically enabled.

  • POS to POS Communication: Click here for more information.

Exit: To quit the Configure Synchronisation option.
