Goods Inwards based on Stock No.

The details of goods received are recorded here, based on the stock number.

In this window you can execute the following functions.

Add:Click to record a new inwards document. Click here for more information.

Edit:Click to correct or edit the existing inward document. Click here for more information.

Select the required Transaction Type.

Select Internal Doc Prefix/ External Doc Prefix as required.

Enter the Doc. Prefix, Doc No. in the Edit window and click Ok.

The document details are displayed in the grid. Make necessary changes and click Ok.

Delete:Click to remove an existing inward document. Click here for more information.

Select the Transaction Type.

Select Internal Doc Prefix/ External Doc Prefix as required.

Enter the Doc. Prefix, Doc No. in the Delete window and click Ok.

The document details are displayed in the grid. Click Ok to confirm deletion.

RePrint:Click to print an existing goods inward entry. Click here for more information.

Select the Transaction Type.

Select Internal Doc Prefix/ External Doc Prefix as required.

Enter the Doc. Prefix, Doc No. in the Reprint window and click Ok.

Note:  A pop-up message is displayed to clear entries made during Edit/ Delete and continue.

 

Ok:To accept the entries

Cancel:To quit the entries

Exit:To exit the goods inward option.
