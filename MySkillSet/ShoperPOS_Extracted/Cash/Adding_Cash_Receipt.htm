Adding Cash Receipt

The Cash Receipts window is displayed.

Note:The provision to capture the customer code (Party Code) is activated by checking the parameter Customer Mandatory in Cash Payouts/Receipts under Customer category.

Document Prefix: Select the document prefix.

Document No.:Starts with 1 for each series and gets incremented automatically.

Reason Code:Select a valid Reason Code, which is catalogued in general lookup, by pressing F2 key.

Narration: Enter the text quoting the reason for the receipt to be recorded.

Mode: Select the mode of payment from the drop down list. The mode of payment can be Cash, Cheque or Credit Card. If it is cheque, credit card or debit card then you will have to enter the details.

The Screen for entering Credit Card details is displayed as shown.

cardno: Enter the Credit Card number.

authorization:Enter the authorisation slip number generated by the swipe machine.

Select the type of card and enter the amount.

Note: The caption of the field (for example: cardno, authorization) is displayed according to the caption catalogued Payment Mode.  Click here for more details.

Code: Select the currency code from the drop down list.

Description: Enter the optional description, if any.

Units: Enter the number of units of the currency.

Exchange Rate: The predefined rate of exchange of the currency. This will be displayed automatically based on the currency code selected. For example, in the case of Rupees, the exchange rate is 1.

Value: The product of Units and Exchange Rate is displayed here.

Amount: Displays the total amount of cash received.

Add: To make a new cash receipt entry. Click here for more information.

Edit: To edit an existing cash receipt

Delete:To delete an existing cash receipt

Reprint:To reprint an existing cash receipt

Exit:To exit from the cash receipt window

Ok: To accept the details keyed in. Once you click Ok, it will give you an option to take the print out of the details entered.

Cancel: To undo all the details keyed in.
