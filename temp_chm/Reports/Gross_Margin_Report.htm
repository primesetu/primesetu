Gross Margin Report

The Gross Margin report provides you details of gross margin in the specified period. Report can be generated for products, brands, styles or a combination of these either in terms of cost price or purchase price. You may select the exceptional transactions to be included in the report and how those are to be treated.

How to reach: Reports > MIS > Gross Margin

The Gross Margin Report window is displayed.

Period: By default, the From and To are the current Shoper date. You can enter any date range but the To date cannot be greater than the Shoper date.

Select Product, Brand or Style or any combination of these. For each item of Product/ Brand/ Style, select All or Specific to identify the corresponding items to be included in the report. If you have selected Specific, select the required values from the corresponding list.

Gross Margin: Select Cost Price or Purchase Price to generate the report based on the corresponding values.

Report View Options: Select Closing Balance, if it has to be included in the report.

Display Profit %: Select Cost Price, Selling Price or Both to display the gross margin percentage in terms of the specified price.

Report Display By: Select Code or Description to include the corresponding values in the report.

Transactions to Include: Select Approval Returns to include such returns in the report. In case you want such transactions to be treated as sales returns, select Treat As Sales Returns. Similarly, select Approval Issues to include such issues in the report and Treat As Sales to treat such transactions as sales.

Select Misc Issues to include such transactions in the report. In case you want such transactions to be treated as sales, select Treat As Sales or select Treat As Purchase Returns if you want these transactions to be treated so.

Select Misc Receipts to include such transactions in the report. In case you want such transactions to be treated as sales returns, select Treat As Sales Returns or select Treat As Purchases if you want these transactions to be treated so.

Summary: To view the report without the stock numbers of the items involved in the transactions included for the report generation.

Detail: To view the report with the stock numbers of the items involved in the transactions included for the report generation.

View: To display the Report

Exit: To close the window
