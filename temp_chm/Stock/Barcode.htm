Barcode

Barcodes provide detailed information on key aspects of the item sold, enabling sales activity to be executed quickly. The functionality in this group are useful in stores where the items are bar coded in-house. Stores using vendor assigned bar codes need not use this feature.

The printer independent barcode label printing features of Shoper 9 make it possible to print the barcode labels in the available printers like LaserJet, Dot matrix, Inkjet, etc.

In Shoper 9 you can:

  • Design Label for Barcode Printers

  • Design Label for Standard Printers

  • Design Layout for Standard Printers

  • Print Labels
