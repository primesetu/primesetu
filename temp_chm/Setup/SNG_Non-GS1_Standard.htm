Non-GS1 Standard

A stock number can be defined as a combination of a maximum of seven segments, each of which can be of different but specified length, optionally delimited with the required character.

A default stock number generation methodology is defined as an auto incrementing number. You may edit this to define any methodology of your choice. Stock number generation methodology can be defined only if there are no item masters. Click Edit to start defining a methodology.

Number of Segments: Select the number of segments required in the stock number.

Note: The number of segment tabs (Seg 1 to Seg 7) enabled depends on your selection in Number of Segments.

Segments Are:

Delimited: Select if the segments are separated by a delimiter.

Delimiting Character: Enter the delimiter. It can be a comma, full stop or hyphen.

Fixed: Select if the segments are of fixed length.

Seg 1

Number of Digits: Enter the length of the segment under consideration.

Type of Segment: Select the type of the segment from the drop down list. The choices available are Auto-incremented, User Assigned and Derived. If there are more than one segment in the stock number pattern, Auto-incremented is available only on the last segment.

Note: There can be only one Auto-increment segment and if present, should be the last segment.

Field Derived From: Select the source, for example, Class1, Class2, SubClass1, SubClass2 or SizePackUOM, if the segment type is Derived. This field is not enabled, if the segment type selected is either Auto-incremented or User Assigned.

Starting Digit Of The Field: Enter the starting digit of the field mentioned in Field Derived From.

Click the enabled tab names to enter the required details for the corresponding segments.

Ok: To save the changes made to the definition

Cancel: To cancel the changes made to the definition

 

Exit: To close the window
