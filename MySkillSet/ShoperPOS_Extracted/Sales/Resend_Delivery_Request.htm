Resend Delivery Request

This option is used to filter and resend the delivery requests in case they are loaded/not loaded in the service location. Those loaded/not loaded delivery requests can be resent in case they are required by the service location due to some reason.

How to reach: Sales > Delivery Management > Resend Delivery Request

The Remote Location Delivery Request window is displayed.

  

Delivery Request Selection: Select the required delivery request to be resend

Not Loaded: This option is selected by default.

Note: The delivery requests may not be loaded if there is a problem in the internet connectivity. Such unloaded delivery requests are displayed on selecting this option.

Loaded: Select this option to display details of loaded delivery requests to service locations for the selected time period.

From: Select the check box and enter the starting date.

To: Select the check box and enter the end date.

Doc No.: Type/enter the bill number to display the loaded delivery request.

Note: If the Doc No. field is left blank all loaded delivery requests for the period are displayed.

   

Select: Select the required delivery request to be resent.

Note: The Resend button is enabled after the selection of the displayed delivery request.

 

Resend: To resend the selected delivery request to the service location.

Exit: To close the window.
