Activity Log Report

Activity Log Report option provides details about the time when the user logged in and logged out. It also provides the logged-in duration and the status of all users.

How to reach: Setup > Supervisory Functions > Security Management

On clicking Activity Log Report, the Security Management – Activity Log Report window is displayed.

User ID: Displays the user ID.

Company: Displays the company code.

Company Name: Displays the company name.

Node ID: Displays the node ID.

Node Name: Displays the node name.

Date: Displays the date of activity.

Program Name: Displays the program name.

Login Time:  Displays the logged-in/ activity time.

Logout Time: Displays the logged out time.

Duration: Displays the logged-in duration of users.

Status:  Displays the status of the users.

Remarks: Displays the remarks in any.

Clear log upto: Select the date up to which the activity log has to be cleared.

Ok: To clear the log.

Export to Excel: To export the data to Excel

Exit: To close the Security Management window
