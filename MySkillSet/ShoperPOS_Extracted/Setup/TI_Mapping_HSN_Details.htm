Mapping HSN Details

The HSN tab is as shown.

Selected item details are displayed in the grid. If HSN codes are already assigned to any of the selected items, that is displayed. If HSN codes are catalogued in General Lookup, select the required HSN code from the drop down list.

Copy to All: To copy the selected HSN code to all items

New HSN Code: To define a new HSN code

Save: To save the HSN mapping

Exit: To close the Add or Update HSN Code window
