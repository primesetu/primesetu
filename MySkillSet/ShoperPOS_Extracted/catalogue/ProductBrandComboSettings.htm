Multiple Pricing Product + Brand Combo Settings

This option is available only when the system parameter Multiple Prices Applicable under the category Item Classification is set to 2 – multiple prices are applicable to some items. You have to use this option to identify the  class1 + class2 (product + brand) combinations for which multiple prices are required.

How to reach: Catalogue > Define Multiple Prices > Product + Brand Combo Settings

The Pricing Applicability for Items window is as shown.

Item Classification: Displays the list of class1 + class2 (product + brand) combinations defined in the system.

Multiple Price:

Applicable: Select if multiple prices are required for the class1-class2 combination.

Save: To save the selection made.

Exit: To close the window. If the changes are not saved, changes are discarded.
