Configuration

Permitted Operations Configuration for Goods Inwards

To enable a functionality, set the parameter Permitted Operations Configuration for Goods Inwards under the category Inwards accordingly.

Transaction Types

To add values to the transaction types list, set the system parameter Valid Transactions Configuration in Goods Inwards under Inwards.

Reason Code

To add values under reason code, insert values under the category Reason for Misc Issues in General Lookup.

Load From PDT File

To enable loading from PDT file and to specify the format of PDT files, set the system parameter Allow PDT File Loading in Goods Inwards/Goods Outwards/Physical Verification under Inwards.

 

Load from PT File

To enable loading details from PT File, do not set the system parameter Disable PT File Loading in Goods Inwards under the category Inwards.

 

Load Outward to Inward

To enable loading details from inwards document to outward documents, set the system parameter Import Goods Outwards Into Goods Inwards under the category Inwards.

Transporter Details

To add the transporter details, set the system parameter Capture transporter details under the category Inwards.
