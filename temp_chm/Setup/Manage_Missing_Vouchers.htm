Manage Missing Vouchers

This option will help you to restore the vouchers, if any, that were deleted, possibly by mistake in Tally. This utility will re-post only those vouchers that had been posted from Shoper and are not currently found in Tally.

How to reach: Setup > General > Tally Interface Utilities > Manage Missing Vouchers

The Tally Interface Utilities – Manage Missing Vouchers window is displayed as shown.

All existing mapping information for the selected showroom is displayed in the grid. Select the mapping for which you want to manage the vouchers missing in Tally.

Ok: To proceed with the missing vouchers management process

Exit: To abort the missing vouchers management process

The Manage Missing Vouchers window is displayed as shown.

The transactions posted from Shoper, in the specified date range, will be displayed.

From Date: Select the date from which the posted transactions need to be selected.

To Date: Select the date until which the posted transactions need to be selected.

Show Vouchers: To list the vouchers in Tally, corresponding to the transactions posted from Shoper for the specified period.

Compare Vouchers: To compare the vouchers against transactions in Shoper.

Select All: To select all transactions posted from Shoper.

Re-Post: To repost the selected transactions to Tally. In the grid, mark the vouchers to be reposted.

Exit: To quit the Manage Missing Vouchers option.
