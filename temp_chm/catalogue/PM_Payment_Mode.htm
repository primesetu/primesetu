Payment Mode

The Payment Modes are used by sale, purchase and cash transaction functions of Shoper 9 POS and Shoper 9 Distributor. You can decide whether the currency can be used for billing, purchase orders, tenders and payouts.

How to reach: Catalogue > Payment Mode

The Mode of Payment Catalogue window is displayed. In this window you can execute the following functions.

Add: Click to catalogue a mode of payment. Click here for more information.

Edit: Click to modify an existing mode of payment. Select the Mode of Payment and enter the Payment Code to display the catalogued details. Make necessary changes as explained in Add option. Click Ok to confirm modifications.

Delete: Click to delete an existing mode of payment. Select the Mode of Payment and enter the Payment Code to display the catalogued details. Click Ok to delete the selected Payment Code.

Exit: To quit the Payment Mode option.
