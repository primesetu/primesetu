Counter Summary across Cashiers

The Counter Summary across Cashiers report provides cashier-wise transaction information for all types of cash inflow and outflow, for the selected period. The report generated is based on tender-wise or currency-wise information. A detailed or summarised report can be generated, considering all cashiers or a selected one. Detailed report shows each transaction processed by individual cashiers. Summary report shows the cumulative values of all transactions done by each cashier.

How to reach: Reports > Cash > Counter Summary across Cashiers

The Counter Summary Report Across Cashiers window is displayed.

Report Option: Select Detail or Summary to specify the type of report to be generated.

Criteria Selection: Select Tender Report to include the information based on different modes of payment, for example, Cash, Credit Card, etc. Select Currency-Detail Report to include the information based on the type of currency used, for example, Rs., USD, etc.

Date Range: By default, the From Date and To Date are the current Shoper date. You can enter any date range but the To Date cannot be greater than the Shoper date.

Cashier: Select All or Selected to generate the report for all cashiers or for selected cashiers. If you have chosen Selected, select the cashier/s from the list.

View: To view the report

Exit: To close the window
