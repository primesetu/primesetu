Manage Master Changes

After setting up Tally Interface Mapping, you might need to change the masters’ name or alias in Tally. Having made the changes in Tally, you might want to change the settings for Tally Interface Mapping accordingly. This option can be used for the same.

Note: Tally Interface will continue to function without errors, even if you do not update the map settings.

 

How to reach: Setup > General > Tally Interface Utilities > Manage Master Changes

The Tally Interface Utilities – Manage Master Changes window is displayed as shown.

All existing mapping information for the selected showroom is displayed in the grid. Select the mapping for which you want to manage masters that are renamed in Tally.

Ok: To proceed with the change masters process

Exit: To abort the change masters process

The Manage Master Changes window is displayed as shown.

The existing masters’ information is displayed in the grid. If any changes were made to the ledgers, which are posted by this mapping, in Tally, those changes are highlighted. These changes will be updated in the map settings in Shoper.

Update Stock: To update the stock information in Tally 9.

Apply Changes: To save the changes to the map settings in Shoper. A confirmation message is displayed after completion of the process.

Exit: To abort the changes.
