Item Classification Browse

Item classification browse is applicable in the following options:

  • Physical Verification – Size Wise Entry

  • Physical Stock Take – Item Class

  • Physical Stock Take – Size Wise Entry

  • Goods Inward – Size Wise Entry and

  • Goods Outward – Size Wise Entry

On pressing F2, the Item Classification Browse window is displayed. Further, press F4 to view the details in the grid.

The Item Classification Browse window is displayed.

Enter the selection criteria under Product, Brand, Style or Shade and press Enter. The items with the given criteria get listed in the grid. Highlight/ select the required item and press Enter or press Alt+O or double click on it.

Alt + W: To toggle focus

Ctrl + PageUp: To search previous

Ctrl + PageDown: To search next

Ctrl + F5: To refresh/ load/ view/ browse the data from item master

F2: To browse the values and populate the current column. If the value for Subclass1 (Style) is entered and you press F2 when the cursor is on the stock number, then, only those stock numbers of items with the selected style are displayed in the browse. You can also use the browse facility in any sequence, for example, you can first select the brand, then the stock number.

 

Previous: Move to the previous page in the grid

Next: Move to the next page in the grid

Ok: To accept the selection
