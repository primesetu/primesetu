Unlock License

This option helps you to unlock the Shoper 9 license. The unlock key is sent to your e-mail. This option is available only after the activation request is sent.

The Manage License window with Unlock options is displayed as shown.

Unlock License: To unlock the license.

Proxy Configuration: To configure the proxy server to connect to the internet (if required).

List of License Servers: To list the License Servers accessible on the network.

Continue: To continue the unlock process.

The Unlock License window is displayed as shown.

 

Unlock Key: Enter the unlock key received by E-Mail.

Self Support: To retrieve the Unlock Key from the Tally website.

Submit: To activate the Shoper 9 License.

An activation successful message is displayed as shown.

 

OK: To return to Manage License screen.

 

Note: After license activation the Manage License displays the options Update License, Surrender License and Reset License.

To view the license details, close all open windows, start Shoper 9 and login.
