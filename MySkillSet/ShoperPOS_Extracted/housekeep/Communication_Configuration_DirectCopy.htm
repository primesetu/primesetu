Communication Configuration for Direct Copy

The Communication Configuration window is as shown.

Communication Type: Direct Copy is selected, based on the selection on the previous window.

Configuration Name: Enter the name for the direct copy configuration.

The grid displays the properties to be set to use direct copy for communication. The properties are listed under Names and the specified values under Value.

Destn. Folder: Enter the name of the folder where files (reports) are to be copied. By default, the Shoper application path is displayed. You need to have sufficient access permissions on this folder to create files.

Active: Select to make the communication type active.

Test Connection: Click to check the connectivity.

Ok: To save the configuration

Cancel: To cancel the values entered or to close the Communication Configuration window, as the case may be

 

 

Note: Here you cannot add or view any other communication configuration.
