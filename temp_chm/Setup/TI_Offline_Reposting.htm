Offline Reposting

In offline posting if response file or request file is lost, Tally Interface will not allow further exchange of data. To correct this problem the user can reset the last posting information which will force the interface to allow the user to create a new request file to be sent to Tally.

How to reach: Setup > General > Tally Interface Utilities > Offline Re-posting

The Tally Interface Utilities – Offline Re-Posting window is displayed as shown.

All existing mapping information for the selected showroom is displayed in the grid. Select the offline mapping to reset the posting information.

Ok: Click to proceed with the resetting process. A confirmation message is displayed after completion of the process.

Exit: Click to abort the process.
