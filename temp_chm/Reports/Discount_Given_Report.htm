Discount Given Report

The Discount Given report provides a list of discounts given in a period. Any unusual/ extraordinary discount can also be reported. Detailed discount information can be provided at the item-level or bill-level. Discount given on a particular day of the week can also be tracked.

How to reach: Reports > Sales > Discount Given

The Discount Given report window is displayed.

From: Select the date from when the report has to include data. Default date selected will be the current Shoper date.

To: Select the date to which the report has to include data. Default date selected will be the current Shoper date. You can enter any date but the it cannot be greater than the Shoper date.

Discount Type: Select Discount or Abnormal Discount. If you have selected Discount, the report will include bills where discount is a regular percentage. On selecting Abnormal Discount, you can specify an allowable percentage to view the report.

Bill Type: Select Cash, Credit or Both to include the selected type of bills in the report.

Applicability Days: Select Specific or All. If Specific is selected you need to choose the days individually.

Select Based on Bill to generate discount report based on bills or select Based on Item to generate the report based on individual items.

Select All to include all bills/ items with any discount percentage. Select Specific and enter Allowable Discount Percentage to include bills/ items with the specified discount applied.

View: To view the report

Exit: To close the window
