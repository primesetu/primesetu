Activate Site License

This option is used to activate license for single site, first site license in the case of a multi site license and promotional license.

In Shoper 9 start-up screen select Manage License > Activate License >  Single Site

 The Activate License window is displayed.

Serial Number/ Promotional Code: Enter the Serial Number provided with the Shoper 9 CD to activate regular Shoper 9 license. If you want to activate promotional license, enter the promotional code.

Activation Key: Enter the Activation Key provided with the Shoper 9 CD.

E-Mail ID of Administrator: Enter a valid e-mail ID.

Repeat (E-Mail ID of Administrator): Repeat the same e-mail ID.

Account Name: Enter the account name of your choice.

Note: It is important to provide a valid e-mail ID. Your Account ID is created using the e-mail ID provided. Unlock key, Tally.NET password and other account related information is sent to this e-mail ID. Tally.NET account is also created using the given ID.

 

Submit: To send the activation request.  An activation request message is displayed. The unlock key is sent to the e-mail ID displayed in the message.

Multi Site: To activate licenses at additional sites in a multi site account. Click here for more information.

 

Note: If Shoper 9 is not connected to the Internet, activation request is not processed. You can activate Shoper 9 in offline mode. Click here for more information.
