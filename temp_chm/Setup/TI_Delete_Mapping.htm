Delete Mapping

Tally Interface Mappings can be deleted, if required. Once deleted, the mapping cannot be restored. You may need to do this in cases where you no longer wish to post Shoper 9 data as configured by a mapping.

How to reach: Setup > General > Tally Interface Utilities > Delete Mapping

The Tally Interface Utilities – Delete Mapping window is displayed as shown.

All existing mapping information for the selected showroom is displayed in the grid. Select the mapping which you want to delete.

Note: A deleted map name cannot be reused as the map name is permanently stored in Shoper database.

 

Ok: To proceed with the deletion process. A message is displayed asking confirmation. Click Yes to delete. A message is displayed on successful deletion of the selected mapping.

Exit: To abort the deletion process.
