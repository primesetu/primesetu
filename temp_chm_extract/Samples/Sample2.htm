Sample 2 - Extension at Footer Detail Level of Billing

In billing, while confirming the bill by Cash, give a provision to enter the denominations of cash given by the customer. Additionally, display the change due, if any, to the customer.

The details of the extension to be enabled to facilitate the requirement are given below.

Extension Type: Pre-update extension at the Footer Detail Level of Billing.

Transaction: Sales

Exe Name: SR130000

Project Name: Project1

Class Name: clsDenomination

Function: ShowDllForm1

DLL Name: Denominations.dll

Records to be inserted in General Lookup (GenLookUp) table

SQL statements to insert records in GenLookUp table are:

Insert Into GenLookUp (Recid,Code,Descr,Flag) Values (14500,'2100+4+1','Project1.clsDenomination',1)

Insert Into GenLookUp (Recid,Code,Descr,Flag) Values (14600,'2100+4+1','Y1',1)

Output

The Billing window is displayed.

Here, Rs. 3840/- is paid by Cash. You need to enter the count under each denomination. The change due to the customer in this case is 10, which is displayed in a different colour.
