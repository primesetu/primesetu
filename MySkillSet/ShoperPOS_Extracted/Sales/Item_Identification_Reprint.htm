Item Identification - Reprint

The Item Identification Reprint window is displayed.

Prefix: Enter the IID prefix for printing.

No.: Enter the document number.

Print Item Identification: Select to print the item identification details.

Invoice Prefix/No.: Displays the sales bill prefix and number.

Billing location: Displays the billing location name, if enabled.

Print Invoice: Select to print Sales Invoice along with door delivery details.

Print: To print the selected document.

Exit: To close the window.
