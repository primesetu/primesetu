Group by

This option allows you to view the query list based on the Groups. The options available under Group by are Date, Site and Status wise. You can set the Default Grouping using this option.

Click on F3: Group by provided at the buttons toolbar or press the function key F3, to select the Group by option. Click here to view a sample.

Group by Date

Using this option the user can view date wise list of queries. Issues for the latest date are on the top followed by the issues from earlier dates.

Group by Site

This option allows the user to view the query list on the basis of the Site/ Location, i.e., the issues sent and received at different Sites belonging to the same or different Serial Numbers, under a single account. Click here to view a sample.

Click on F1: Condensed enabled in the buttons toolbar. This will display the Site wise query list in Condensed mode. Click here to view a sample.

Note: The option Site located under Default Grouping will not be listed for Single Site users.

Group by Status

This option allows you to view the query list based on the Status. Click here to view a sample.

The query list can further be viewed on the basis of the Status selected using the F9: Status Filter option.
