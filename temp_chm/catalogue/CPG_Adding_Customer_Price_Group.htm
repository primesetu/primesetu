Adding Customer Price Group

The Customer Price Groups window is as shown.

 

Code and Description: Enter customer price group code and a meaningful description for the entered code.

Item Classification-wise Price Factor Applicable: Select if item classification-wise price factor catalogued is applicable.

Allow Credit Invoice: Select to allow credit sales at the time of billing

Allow Cash Invoice: Select to allow cash sales at the time of billing

Allow Goods (Misc.,) Issue (Without Invoicing): Select to allow miscellaneous issues without invoicing   

Payment Terms: Select the type of payment terms from the list

Credit Days: Enter the number of days allowed for credit billing for the entered customer price group. Enter numeric values only.

Credit Limits: Enter the amount that can be allowed for credit billing.

Dest-Wise Tax Type: Select a destination-wise tax type code from the list. Its activation is set as per user requirement through parameter settings.

Ok: To confirm and save the entered transaction details

Cancel: To clear the details entered

 

Exit: To exit the customer price group screen
