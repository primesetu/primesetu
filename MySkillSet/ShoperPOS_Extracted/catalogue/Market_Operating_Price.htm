Market Operating Price

You can define and manage the market operating price (MOP) for various categories of stock items.  

How to reach: Catalogue > Market Operating Price

 The Market Operating Price window is displayed.

Define MOP: Click to define the market operating price.

Manage MOP: Click to view, activate, de-activate and archive existing MOP's.

MOP Category: Select the required category for cataloguing.

Configure MOP Category:

Active: Select/Deselect to activate the defined category.

MOP Category: Enter the required category. By default the General category is displayed.

Applicable: Select All/CPG (Customer Price Group) as required for configuration.

All/Specific: Select all customer price groups of a specific customer price group.

Specific: Select to open an additional window to display the specific customer price group for selection.

Apply: Click this button to accept the configuration.

Ignore: Click this button to stop the configuration.

 

Applicable:

From: Select/enter the starting date from which the MOP is applicable.

To:  Select/enter the end date till which for MOP is applicable.

Load Items: Click to load the MOP details to be catalogued from a file.

  

File Format: Stock Number is by default selected from the list. Select Min MOP and MOP options if the related information is available in the file. If the file contains only stock numbers, do not select the other options. After loading, the Min MOP and MOP can be updated in the Market Operating Price screen against the loaded Stock Numbers.

Delimiter: semicolon is selected by default.

Format: This is a non-editable field, which, displays the options selected for File Format.

File Name: Browse and select the file.

Load: Click this button to load the file.

Cancel: Click this button to close the window.

More: Click to display the MOP Category section in the grid. Click Less to hide the MOP Category section and display the grid.

Factors applied on: Select the appropriate factor to define the minimum market operating price and market operating price, Active MOP or Retail Price.

Apply: This button gets activated when the Select factor for Min MOP and Select factor for MOP are configured. Click to apply the configuration.

Stock No: Enter the stock number or press F2 to browse and select the stock item.

General: Enter/type the values under Min MOP and MOP respectively.

Allow Item Level Discount: Select if you want to allow discounts defined for the stock item to be applied in billing.

MOP Name: Enter/type the name for the MOP.

OK: Click to accept the definition.

Cancel: Click to clear the selected options.

Settings: Click to Configure the columns to be displayed in the grid.

Display: Select/deselect the item classifications to be displayed in MOP grid. The code for all classifications are selected by default. However, the code for stock number cannot be deselected while for other classifications it can be selected/deselected.

Active Minimum MOP: Select/deselect if required for display in Define MOP grid.

Active MOP: Select/deselect if required for display in Define MOP grid.

Accept:  Select to enter values for Minimum MOP (Min MOP).

Minimum MOP: Select/deselect if required to accept values in Define MOP grid.

Apply: Click this button to accept the configuration.

Ignore: Click this button to stop the configuration.

Exit: Click to close the window.
