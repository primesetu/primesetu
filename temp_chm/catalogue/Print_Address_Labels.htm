Print Address Labels

This option is used to generate address labels to send Customer Mailers.

How to reach: Catalogue > Print Address Labels

The Label Print window is displayed.

 

Printer Type: Select DOS or GUI based on the type of printer. DOS is selected by default.

Compressed: Check to compress the printing through DOS printers and gets inactive when GUI is selected

Select Customer

All: Selected by default, to print labels for all customers in the database.

Specific: Click to select specific customer.

This button is activated when Specific is selected.  On clicking this button, the Advanced Customer Search browse window is displayed.

Click on Advanced to display additional filter conditions to the window in addition to the default filter conditions displayed. Enter the required filter criteria and click Search or press F4. The list of customer records satisfying the filter criteria is displayed.

By default, all the records are checked. You may select Specific, and check the customers selectively for whom you want to print the labels or select the option All then click Select.

Print Details

Label Height (Lines): Enter the number of lines to be printed for each label.

Label Width (Char): Enter the number of characters to be printed for each label.

No. of labels in a row: Enter the number of labels to be printed in a row.

Print Customer Code: Select to include the customer code in the label.

Horizontal Gap (Char): Enter the gap between two lines of the address in terms of characters.

Vertical Gap (Lines):  Enter the gap between labels in terms of lines for DOS and Twips for GUI.

Print: To print the report

Exit: To quit the option
