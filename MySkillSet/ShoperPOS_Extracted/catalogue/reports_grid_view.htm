Reports Grid View

A report is used by various people for different types of analysis and planning. You might be using different tools for these. Grid view option displays the report in a spreadsheet format and there you have options to export the report to other file formats, copy to clipboard in order to paste data in various utilities, find a particular value, sort the report based on different attributes, filter data to derive relevant information and compute group totals or overall total for further analysis.

A report in default Grid View appears similar to a spreadsheet. By default, there are no totals or filters applied.

You can use the toolbar buttons to execute various actions on the report in the grid. The toolbar is as shown.
