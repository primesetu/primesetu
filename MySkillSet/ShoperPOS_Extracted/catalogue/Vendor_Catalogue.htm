Vendor

This option is used to maintain Vendor details like mailing list information, vendor code, vendor type, TIN etc. Shoper 9 Point-Of-Sale maintains the mailing information captured in various catalogues commonly.

How to reach: Catalogue > Vendor

The Vendor Catalogue window is displayed. In this window you can execute the following functions.

Add: Click to add a new vendor. Click here for more information.

Edit: Click to edit an existing vendor details. Enter the vendor ID to retrieve the relevant record. Make necessary changes as explained in add option. Click Ok.

Delete: Click to delete an existing vendor details. Enter the vendor ID to retrieve the relevant record. Click Ok to confirm deletion.

You can change the display mode of records and view different records by using the following buttons.

Form: Click to display the selected vendor from the grid in the form.

Grid: Click to display code and description of all the vendors available.

Click the relevant button to display First, Previous, Next and the last record.

Exit: To close the vendor catalogue window
