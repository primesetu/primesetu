Goods Inwards Size-wise

Recording goods inward based on the stock numbers will consume more time.

This option is provided to enable the user to save time and effort since it involves making less number of entries. Size wise quantity of items received for combination of Item Class1 (Product), Item Class2 (Brand), Item SubClass1 (Style), Item SubClass2 (Colour), can be entered in a single line. Size columns are activated based on the values entered for these classification fields.

You can enter style level discount without getting in to SKU level. It is possible to capture multiple Discounts/ Add-ons/ Deductions/ Tax rates at the line level.

The features enabled by setting the following system parameters will be applicable only for normal goods inwards and has no bearing in size wise goods inwards option.

  • To apply LSQ on item selection

  • Insertion of new items on the fly

  • Inward multiple purchase orders

How to reach: Stock > Goods Inwards Size-wise

The Goods Inwards Size-wise window is displayed. In this window you can execute the following functions.

Add: Click to record fresh goods inwards or to add an entry. Click here for more information.

Edit: Click to correct or edit the existing entry. Once the grid is populated, make the necessary changes and click Ok.

Delete: Click to remove an inwards document. Once the grid is populated, click Ok.

Reprint: Click to reprint an inwards document.

Exit: To quit the Goods Inwards – Size-wise window.
