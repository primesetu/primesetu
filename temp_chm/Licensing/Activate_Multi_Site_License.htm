Activate Additional Site License

This option is used to activate license at additional sites in a multi site account, after activating the first site license. Activating a license at an additional site also requires a unique Site ID and e-mail ID of the Site Administrator along with an Account ID.

In Shoper 9 start-up screen select Manage License > Activate License

The Activate Site License window is displayed as shown.

Serial Number: Enter the Serial Number provided with the Shoper 9 CD.

Activation Key: Enter the Activation Key provided with the Shoper 9 CD.

Account ID: Enter the Account ID provided by your Head Office. The Account ID is created by the concerned Tally partner or created during activation of the first site.

Site ID: Enter the Site ID.

E-Mail ID of Site Administrator: Enter a valid e-mail ID.

Repeat (E-Mail ID of Site Administrator): Repeat the same e-mail ID.

Note: The unlock key will be sent to the Site administrator's email address given in the form. It is important to provide a valid email Id. The Site administrators will administer the license for the particular site.

 

Submit: To send the activation request. An activation request message is displayed. The unlock key is sent to the e-mail displayed in the message.

Single Site: To activate license for the first time for your organisation. Click here for more information.
