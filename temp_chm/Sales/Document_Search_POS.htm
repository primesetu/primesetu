The document search window with the attributes provided for advanced document search is displayed.  

  

The Advanced search option allows you to search documents more efficiently. The required documents can be searched based on various attributes. The advanced search has attributes like StockNo, Item-Desc, Product, Brand, Style, Shade, Size and Node-Id in addition to the attributes available for General Search option.

Document Search

This option is used to search the required documents for the selected Transaction Type.

Note: The functionality of F2 Browse window for Sales Advice Slip, Sales Order, Service Order and Suspension remains the same.

The grid displays all the documents of the selected Transaction Type for the customer selected in the Billing window. If the Customer Code and Customer Name are not selected during Billing, then the Document Search window will display all the documents of the selected type regardless of the customer.

Note: The Transaction Type Suspension is used here to explain the Document Search function.

The Document Search window allows you to search documents based on various attributes. The attributes provided for searching a suspended bill are Susp.DocNo, Cust.Code, Cust.Name, Date, Slaes Staff, Total Items and Total Value. If you want to search the documents based on any of these attributes, specify the value of the attribute in the corresponding field.  

The Customer Code and Customer Name selected in the Billing window get displayed in the respective fields by default. You may edit the same, if configured.

Advanced: To display the list of attributes provided for advanced document search.

Search: To list all the documents based on the specified attribute values.

Document Details Grid:

The details of the documents are displayed in this grid. Click on any of the rows in the grid and press F2 to display the corresponding Item Details grid, if required. Select the required document/s for recalling.

Select All: Click to select all the documents listed in the grid. This button toggles between Select All and Remove All.

Remove All: To unselect all the documents.

Previous Page: Takes you to the previous page of the suspended bills list.

Next page:Takes you to the next page of the suspended bills list.

Ok: To recall the selected document/s and takes you to the Billing window.

If customer details (loyalty details) are recorded at the time of recall of a suspended bill, system will incorporate the same in the bill.
