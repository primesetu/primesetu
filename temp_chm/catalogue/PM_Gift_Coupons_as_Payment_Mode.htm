Gift Coupons as Payment Mode

Once you enter Payment Code, the following message is displayed.

 

If you click No, you can proceed entering the following details.

 

Serial No. Applicable: Select if the gift coupons have some serial numbers associated.

Rate Per Unit: Enter the value of each gift coupon.

If you choose Yes, the fields Serial No. Applicable and Rate Per Unit as shown need to be entered. Also the remaining details as required for a Credit Card need to be entered. Click the link for more information.

Note: The Issue Agency selected should have the field Gift Coupons Honoured marked as true.
