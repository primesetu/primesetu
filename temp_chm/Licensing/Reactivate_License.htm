Reactivate License

This option is used when Shoper 9 license is surrendered and the license has to be reactivated.

In Shoper 9 start-up screen select  Manage License > Reactivate License

The Reactivate License window is displayed.

E-Mail ID of Site Administrator: Enter the e-mail ID of the site administrator.

Password: Enter the Tally.NET password.

Submit: To reactivate Shoper 9 license.

 

 

Note: If the Site Administrator is linked to more than one site Reactivate License Resolution window is displayed.

The Reactivate License Resolution window is displayed as shown.

Surrendered Sites: Select the license to be reactivated.

Active Sites: Displays the list of active sites.

Submit: To reactivate Shoper 9 license for the selected site.
