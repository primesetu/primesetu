Gift Coupons as Payment Mode

Once you enter Payment Code, the following message is displayed.

 

Payment Code: Enter the code for gift coupon, e.g. GC.

Applicable: Select/Deselect as per requirement.

Serial Number Applicable: Select if the serial nos. of the gift coupons has to be captured. If this option is selected then the following screen will be displayed during payment settlement to enter serial number of the Gift Coupons.

Coupon Value: Enter the value of each gift coupon.
