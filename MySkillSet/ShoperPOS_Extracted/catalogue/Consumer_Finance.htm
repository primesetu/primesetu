Consumer Finance

 

Payment Code: Enter the code for consumer finance, e.g. CFIN.

Applicable: Select/Deselect as per requirement.

Component:

Number of Components: Used to set the number of additional components to be added to billing screen that will be required to capture scheme information in bills.

Billing: Enter the number of components to be added to billing screen.

Applied At: By default Billing is seleced.

Type: Select the information type to be captured in these components. By default Characters is selected.  The other options available are Date and Number.

Caption: Enter the caption to be given to the component.

Finance Scheme:

 

Scheme Code: Enter the code of the finance scheme under definition.

Scheme Name: Enter the name of the scheme.

Tenure (in months): Enter the period of the scheme in months, e.g. 6 months, 12 months, etc.

Down Payment: Enter the amount of initial payment to be made in terms of months for e.g. if the scheme tenure is 12 months. The Down Payment can be 2 months or 4 months as per the financier.

Start Date: Select the date of start of the scheme from the calendar.

End Date: Select the date of end of the scheme from the calendar.

Finance Scheme is Active: Select/Deselect to inform if the scheme is open or closed.

Additional Charges

Applicable: Select to include the additional charges.

Name: Enter the name of the additional charges

In terms of: Select Percentage/Amount to indicate the value of the additional charges

Value: Enter the appropriate value for the selection under In terms of field.

Note: The additional charges have to be catalogued in General Look-up.
