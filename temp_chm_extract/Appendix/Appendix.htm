Appendix

Creating Custom DLL

  1. Open Microsoft Visual Basic 6.0.

  2. Select ActiveX DLL as the new project.

  3. Rename Project and Class appropriately.

  4. Write the function PDTFile. This function will be called from the Extension Framework.

The PDTFile function returns a string value. If there are no values, NULL is returned.

  1. Write code for the required extension.

  2. Compile the Custom DLL (Select File > Make (Projectname.dll)).

Deploying Custom DLL

  • Copy the DLL and Register the DLL at Shoper 9 POS location.
