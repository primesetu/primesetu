Communication Configuration for SMS

The Communication Configuration window is as shown.

Communication Type: SMS (HTTP) is selected, based on the selection on the previous window.

Configuration Name: Enter the name for the SMS alert configuration.

The grid displays the properties to be set to use SMS for alerts. The properties are listed under Names and the specified values under Value. You may add any number of properties and their values required by the SMS service provider by appending them to the grid.

Provider Uri: Enter the Unique Resource Identifier (URI) for the SMS service provider.

Recipient Tag: Enter the tag name used by the SMS service provider to identify recipients.

Message Tag: Enter the tag name used by the SMS service provider to identify messages.

Active: Select to make the communication type active.

Test Connection: Click to check the connectivity.

Ok: To save the configuration

Cancel: To cancel the values entered

 

Note: Here you cannot add or view any other communication configuration.
