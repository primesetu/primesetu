License Service

This option is used to install, uninstall or restart the license service on your computer.

How to reach: Start > Programs > Tally License Server > Tally Admin > License Service

The License Service window is as shown.

Tally License Server Path: The path from where you have launched the TallyAdmin tool is displayed. Enter the License Server path or alternatively, browse to select the required path.

Port: Enter the port number where you like to install the license service.

Install: To install the License Service on the specified computer and start it as a Windows service. After installing the License Service and starting the service, a message Tally License Server Installed and Started Successfully is displayed.

Uninstall: To remove the License Service from the specified computer. This option is available only when you have installed a License Service. After removing the License Service, a message Tally License Server uninstalled successfully is displayed.

Restart: To restart the license service. After restarting the license service, a message Tally License Server restarted successfully is displayed.

Ping:

This option is used to check the License Server is running.

IP Address/System Name: Enter the name or IP Address of the computer to be pinged.

Port: Enter the required port number to be pinged.

Ping: Check if the computer at the specified IP address and port is reachable and active.
