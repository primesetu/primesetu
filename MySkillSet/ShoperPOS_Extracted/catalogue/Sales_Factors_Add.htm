Adding Sales Factors

The Sales Factors Catalogue window is displayed.

 

Record No: Displays a system generated unique number or manually enter a number to identify each discount rule.

Description: Enter an identification name for the discount rule.

Factor Type: Displays the types of sales factors to select from, i.e., Retail Price Factor, Retail Price Round-off, Add-on, Deduction and Bill Round-Off.

Category: Select the type of catalogue as Customers specific general, Customer Price Group specific and general and All Customers And item specific.

The option Used for Quantity-wise Discount Entry is not used currently.

Customer Details

Customer Code: Enter the customer

code or press F2 and select from the browse

Price Group No: Enter the price group number or press F2 and select from the browse

Period: Select the validity period of the discount rule in terms of From and To Dates and Time. You can select the period only if you have chosen Retail Price Factor as the Factor Type.

Item Details

Classification: Choose to select the Department, Buyer, Product and Brand

Stock No.: Choose to select the stock number as filter. Enter the stock number or press F2 to view the browse.

Application Details:

Computation: Select the point at which the sales factor is to be applied. This option is enabled if you chose Add-on or Deduction as the Factor Type. This can be Consider for Tax Calc. or Ignore for Tax Calc.

Computed On: Select the value on which the discount is calculated, if you have chosen Add-on or Deduction as the Factor Type. This can be Sale Value Before discount, Discounted Sale Value or Value inclusive of Tax.

Variable Y/N: Select if the sales factor is variable. Factor Types that can be variable are Add-on and Deduction.

Rate: Select if the discount is a percentage of the cost. This can be chosen if the Factor Types is not a round-off.

Amount: Select if the discount is an amount added or subtracted from the cost. If the Factor Types is a round-off, discount can only be an amount.

Value: Enter the quantum of the sales factor.

Ok: To save the discount rule.

Cancel: To clear the values entered, if required. Click a second time to change functionality.
