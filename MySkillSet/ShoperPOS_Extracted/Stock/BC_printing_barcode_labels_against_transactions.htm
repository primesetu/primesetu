Printing Barcode Labels against Transactions

To select the items, for which the labels are to be printed, from a purchase transaction file.

Select Transaction Type:

Doc No Prefix: Enter the prefix of Purchase Order.

Doc No From/ To: Enter the range of document numbers from which to pick the items for which the tags are to be printed.

OK: To display the stock number and its item attributes in the Selected Item grid.

The first item's details in the specified purchase orders are displayed in the Selected Item grid.  Use to view the first, previous, next or the last item in the list.

Print: To print the tags for the item displayed in the grid. If you are using old script file, the tags will be appended in the file.

Print All: To print the tags for all the items. If you are using old script file, this creates the tags file by deleting the existing file.

Clear: To clear all the values entered to enable a new session of tag printing.

Exit: To quit the tag printing process.

 

Note: On clicking OK, you will get the provision to select the printer on which the labels need to be printed, if you are using the new script file (.blf).
