Additional fields are:

Based on: Discount can be given based on item quantity, and/or item rate/ value.

Item Quantity: Select if the discount is based on item quantity.

Subject To Max. Discount Allowed: Enter the maximum percentage or amount that can be allowed as discount for the selected items. If you enter a number in Percentage, Amount will be disabled and vice-versa.

Fixed Percentage/ Amount Discount on Items

This promotion type helps you to define schemes that can be applied as fixed percentage/ amount discount at the item level (specified items) during billing. The discount percentage/ amount as per the definition will be displayed during billing. You cannot alter the values at the time of billing, as the discount type is fixed.

You can apply the discount type to all the items in the item master or the items selected based on item classifications.

The details to be captured can be divided into two parts, namely, Buy Items and Discount Details.

Buy Items

This section allows you to select the items to be sold with discount. Alternatively, you can load item details from a CSV file, if available.

The Buy Items section is as shown.

All Items: Select to apply the discount type to all items in the item master, including the new items added after defining the sales promotion. You can proceed to capture the details under Discount Details section.

Item selection can be done either by selecting the values for attributes displayed in the grid or by loading from a CSV file.

Selecting values for attributes:

Configure: Click to configure the options for item selection. Click here for more information.

The Buy Items grid displays the item classification columns as configured. On pressing F2 in any item classification column, the corresponding browse window is displayed with the list of catalogued values to select from.

Note: In the Buy Items grid:

   · You can press F2 anywhere in the grid, irrespective of the order in which the item classification columns are present, to select values of any attribute. Click here to learn more about value selection.

   · You have the flexibility to select either one or multiple attributes in a row, to specify the required items.

   · If Stock No. of an item is selected, the remaining item classification columns for that stock number will be filled automatically.

Loading from a CSV file:

Note: The Load option will be available only if you have selected the option Load Item Details from File under Configure.

Load: Click to load the item details from a CSV file, if available. Click here for more information.

Tip: To make loading faster, configure the item classifications according to the order of data values in the CSV file.
