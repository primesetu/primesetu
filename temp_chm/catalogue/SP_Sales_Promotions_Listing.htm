Sales Promotions Listing

The sales promotion listing gives the details of all the sales promotion schemes defined during the specified period.

How to reach: Catalogue > Listings > Sales Promotions

The Sales Promotion Definition View window is displayed.

Header Details: Enter any of the values in this group to recall the existing sales promotion definitions. You may enter values for multiple fields to filter and retrieve the relevant definitions that satisfy all the conditions specified.

Code: Enter the sales promotion code to retrieve a specific definition to view/ modify.

Description Like: Enter a part of the description of the scheme/s you want to view/ modify. All definitions that satisfy the given description would be retrieved.

Priority no: Enter a number to recall all definitions with the specified priority.

Def. Type: Select the definition type as Discount or Offer.

Def. Level: Select Item Level or Bill Level.

Active/In-Active: Specify whether the definitions are active or not.

Discount Type: Select Percentage/Amount, Buy X At Discounted Rate, Buy X Get At Y Value or Buy X Get Discount for Y.

Def. F/V: Specify whether the discount/ offer is fixed or variable.

Happy Hours: Specify whether happy hours are applicable or not.

Start Date: Select the start date of the discount/ offer schemes.

End Date: Select the end date of the discount/ offer schemes.

All definitions that satisfy the criteria mentioned will be displayed in the grid.

View: To view the Sales Promotions displayed in the grid.

Exit: To exit the window.
