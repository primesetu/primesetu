Load Item Details for Multiple Prices

The Multiple Price Catalog window is as shown.

File Format: This is to identify the fields in the PDT file. Stock Number is selected by default and cannot be unselected. You may optionally select Retail Price and/or Dealer Price.

Date: Click the list button to display a calendar and choose the date from when the multiple price has to be effective.

Delimiter: Specify the character/ symbol used to separate the fields in the PDT file. If you do not specify any character, the default value ; (semi-colon) is considered as the delimiter.

Format: The format of data in the PDT file is displayed here based on the selections you have done in File Format. The order of fields cannot be changed.

File: Enter the path and name of the PDT file (.txt file). Alternatively, select the file name by clicking the browse button.

Load: To select the item details from the PDT file

Cancel: To cancel the selections made
