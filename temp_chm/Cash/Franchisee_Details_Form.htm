Franchisee Details Form

The Franchisee Details window displays all the detailed information about the franchisee and HO. You can edit the details depending on the system parameters setting. This is a facility to create/ update franchisee details in a single form.

How to reach: Cash > Franchisee A/C > Franchisee Details Form

 

The Franchisee Setting Form is displayed. In this window you can execute the following functions.

Edit: To change the parameters of the franchisee. The saved values of all the fields is displayed in three tabs, Head Office - Franchisee Details, Bank Details and General Details.

Update: To update and save the changes made. You are prompted to provide the values for the mandatory fields before the information is saved into the database.

Exit: To exit from the Franchisee Details window.

 

Note: When used for the first time, Update button functions as an Add button.
