Edit Groups

The window for editing existing groups is displayed.

Group ID: Displays the existing Group Id’s. Cannot be edited.

Group Name: Displays the existing group names. Change if required.

User(s): Select to change the users assigned to the group.

Ok: To confirm the editing of the group(s)

Cancel: To abort editing of the group(s)

Exit: To close the Security Management window
