Communication Details for Direct Copy

The Communication Details window is as shown.

Configuration Name: Select the required configuration, if already created.

Create: Select, if you want to create a new communication configuration. On selecting this option, the Communication Configuration window is displayed.

Format: Select PDF, Excel, CSV or HTML, to identify the format in which the report is to be saved (sent).

Destination Folder: Specify the folder to which the report has to be saved.

Active: Select to specify that the current configuration is active and available for communication.

Ok: To save the details entered

Cancel: To close the Communication Details window
