Menu Access Control

This option helps to restrict Users /Groups / Nodes from accessing relevant menu options.

How to reach: Setup > Supervisory Functions > Security Management

On clicking Manage Menu Access, the Security Management – Menu Access Control window is displayed.

User: Select to set restriction for users. Click here to learn more.

Group: Select to set restriction for groups. Click here to learn more.

Node: Select to set restriction for nodes. Click here to learn more.

Ok: To save the applied restrictions

Cancel: To clear the applied restrictions

Exit: To close the Security Management window
