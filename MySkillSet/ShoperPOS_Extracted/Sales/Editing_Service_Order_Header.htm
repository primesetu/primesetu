Editing Service Order Header

The header part of the service order is as shown.

Doc.Ref.: The document prefix and the number of the service order to be edited  are displayed.

The current Shoper date is displayed, which is considered as the date of editing of the service order.

Customer: Enter the customer code. Alternatively, pressing F2 displays the customer Browse window from which the required customer code can be selected. The name of the customer according to the selection of the customer code is displayed in the next field.

Alt+M: To change the customer at any stage of generating the service order

Change Customer

The Change Customer window is displayed when you press Alt + M from the sales advice slip window.

Customer: Enter customer code in the first box. Enter the customer name in the next box. You can also select the customer code from the Browse window that displays on pressing F2.

Change Customer: To change the existing customer to the selected customer.

Cancel: To exit the Change Customer window without changing the customer.

Note: If you change the customer during service order generation, the sales factors and discounts relevant to the selected customer are applied in the service order.

Trial Date: Enter the date of trial of the item

Delivery Date: Enter the date of delivery of the item

Sales Staff: Select the appropriate sales staff code from the drop down list.

Import: This option allows to import the details of the items to be included for editing in the service order. The PDT Import window is displayed on selection of this button. There are two types of importing details - from file or transaction.

Import File

The PDT Import options appear as shown.

Import from: Select the mode of import of item details. The screen shows the options when Transaction is selected.

Transaction:

Type: This allows to import items detail from various types of transactions. The available transaction types are 1100 - Purchase, 1200 - Transfer In, 1300- Sales Return, 2100 - Sales, 2200 - Transfers Out and 3200 - Stock Adjustments.

Bill Prefix: Enter the prefix of the selected transaction type.

Bill No: Enter the number of the document for the selected transaction type.
 

Clear: To remove the entries made

Load Item(s): To load the service order transactions

 

Import File

The PDT Import options appear as shown.

Import from: Select the mode of import of item details. The screen shows the options when File is selected.

Field Template: Select the details of the items to be included in the sales advice slip. The available formats are Stock Number, Stock Number + Qty + Rate, Stock Number + Rate + Qty and Stock Number + Qty.

Note: The default delimiter of the formats is ~. You can set the default delimiter to your choice by changing the value of the system parameter Delimiter Character for PDT Import in Sales Transaction under the category Billing.

Path: Enter the path where the file containing the items is found. Alternatively, browse and select the filename.

Clear: To remove the entries made

Load Item(s):To load the service order entries contained in the file

Recall:This option allows to recall the details of the items to be included for editing in the service order
