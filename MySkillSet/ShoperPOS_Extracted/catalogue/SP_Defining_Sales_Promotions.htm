Defining Sales Promotions

The sales promotion catalogue definition window helps you to define a variety of promotion schemes as per your business requirements.

Each time you invoke the option Define Sales Promotion, last defined promo type will be selected. The definition details like header details, selection of the promotion type and configuration details pertaining to the previous definition are preserved for use in new definition. This reduces the user inputs in case of similar promo definitions.

The Sales Promotion Catalogue window is as shown.

Code/Description: The details for identifying a particular sales promotion definition. Additionally, this section provides you an option to copy an existing sales promotion definition.

Code: Enter the sales promotion code. Shoper can be configured to display the system generated serial number here.

Desc.: Enter a description for the new scheme you are defining.

Copy From: Click to copy the sales promotion definition from an existing one. This option is generally used to avoid re-cataloguing complex schemes when there are only slight variations from the existing one. Click here to learn more.

Note: Pre-configured prefix for the document code will be displayed, if it is configured during implementation.

Validity: Select the validity period of the scheme by selecting the start date and end date.

1. More>>: Click to capture more details, if required. This button toggles between 1. More>> and 1. Less<<.

The header section of the window changes as shown.

Priority No.: Enter a number here to set the priority of the scheme for application during billing. Smallest number is given the highest priority during billing.

Tax type applied for the items at the time of billing: Select As per Item Master, Tax Inclusive (Override of item master) or Tax Exclusive (Override of item master) to specify how MRP has to be treated while billing.

Happy Hours: Select, if the scheme is meant for some specific time interval in a day. Select the start time and end time.

All Days: Select to specify that the scheme is applicable on all days of the week or select the applicable days from the list.

Customer Selection: Click to select the type of customers who can avail the promotional scheme. Click here to learn more.

The Customer Selection window is displayed.

Selection Type: Choose All Customers, Customer Price Group, Customer Code or Customer Classification from the list. If you select All Customers, customer selection is completed at this step. If you have selected Customer Price Group or Customer Code the Prefill and Clear buttons are enabled.

Prefill: Click to fill the grid with appropriate customer details.

Clear: Click to clear the values displayed in the grid.

If you have prefilled the grid with customer details, delete the codes that should not be included in the list that can avail the scheme. If you have selected Customer Classification, select the required customer classification codes from the respective lists displayed.

Note: To delete any customer details listed in the grid, select the row and press F4.

 

OK: To accept the customer selections made

Cancel: To cancel the customer selections made

Sales Promotion Types: The sales promotion types supported by Shoper 9 is classified into four categories, namely, Item Level Discounts, Item Level Offers, Bill Level Discounts and Bill Level Offers. Each of these categories allows you to define a variety of sales promotion schemes.

  • Item Level Discounts: You can define seven types of sales promotion schemes under this category. The discount details captured vary according to the sales promotion type selected. Click the following links to get more information.

Percentage/Amount (Fixed)

Percentage/Amount (Variable)

Discounted Rate

Fixed Value

On Additional Item

On Last Piece

Bundle Discount

  • Item Level Offers: You can define three types of sales promotion schemes under this category. The offer details captured vary according to the sales promotion type selected. Click the following links to get more information.

Offer on Same Item

Offer on Different Item

Bundle Offer

  • Bill Level Discounts: You can define two types of sales promotion schemes under this category. The discount details captured vary according to the sales promotion type selected. Click the following links to get more information.

Percentage/Amount (Fixed)

Percentage/Amount (Variable)

  • Bill Level Offers: You can define offers at bill level using this sales promotion type. Click the following link to get more information.

Free Item

 

F2: To activate the corresponding item browse window if the cursor is in any of the item classification columns in the item details grid. Click here to learn more about value selection.

F4: To delete the current row from the item details grid.

F6: To copy details from the previous row in the item details grid.

 

Save: To save the sales promotion definition

Cancel: To clear the selections made
