Schedule for Reports

You can send Analysis Reports to specified recipients as per configured schedules. This option provides the flexibility to configure generation of selected reports, and despatch to specified recipients.

How to reach: Housekeeping > Schedule for Reports

The Schedule for Reports window is displayed. In this window you can execute the following functions.

Add: Click to define a new schedule for reports.

Edit: Click this button and select a Schedule Name. The corresponding values are displayed. Make necessary changes as explained in the Add option and click Ok.

View: Click this button and select a Schedule Name. The corresponding values are displayed.

Delete: Click this button and select a Schedule Name. The corresponding values are displayed. Click Ok to confirm deletion.

Exit: To quit the report scheduling configuration
