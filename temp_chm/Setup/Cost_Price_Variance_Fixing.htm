Cost Price Variance Fixing

This option is used to set right the abnormal cost price.

How to reach: Setup > Supervisory Functions > Cost price Variance Fixing

The User Authentication window is displayed.

 

User Name: Enter the user name.

Password: Enter the password.

Company: Select the company name from the list.

OK: To continue

Cancel: To exit

The Cost Price Variance Fixing window is displayed.

Filter Criteria: Specify the filter criteria to select the items for which cost price variances have to be fixed. To specify the criteria, select a combination of any of the following options by checking the box to the left of the field name as per requirement

Cost Price >= Retail Price: To select items with cost price greater than or equal to the retail price.

Cur. Stock: Select when you want to include items based on current stock quantity or value. Quantity filters available are – Qty Equal to 0, Qty Greater than 0, Qty Less than 0 and Qty Not equal to 0. Value filters available are – Val Equal to 0, Val Greater than 0, Val Less than 0 and Val Not equal to 0.

Cost Price: Select to include items based on the Cost price. Choose the comparison operation and enter the value as required. You can include items whose Cost price is Greater than, Less than, Equal to or Between the values provided.

Mark up%: Select to include items based on the required mark-up percentage between Cost price and Retails price. Choose the comparison operation and enter the value as required. You can include items whose mark-up percentage are Greater than, Less than, Equal to or Between the values provided.

List All Items: Select to include all items.

Product/ Brand/ Style/ Shade/ Size: Click All to select all values under the field or click Range and select the required values from the list.

List matching masters: Click to apply the criteria and display the items those satisfy the criteria, in the grid. A message No Records Found… is displayed when there are no records available to display for the selected criteria.

Revise the Cost Price: Select Based on Retail Price, Based on Cost Price or Fixed Value to indicate the basis of cost price variation.

If Cost Price Fixing revision is based on retail price or cost price then the cost price can be increased or decreased by a fixed amount or as a percentage of the selected price.

Otherwise, the cost prices of the selected items can be changed to fixed value.

Transaction Remarks: Enter appropriate remarks as to why the cost revision is being made.

Apply: To compute the revised prices and display the values in the grid.

Note: You can modify the revised cost price for individual items in the Revise Cost-Price column. The Closing Balance Value, Revised Value and Difference Value are displayed to indicate the impact of the cost price revision on the selected items.

 

1. Load from PDT File: Select for loading Stock Number and/ or Current Cost from PDT File.

Selecting Load from PDT File option the following screen is displayed.

Stock Number: Select this box to display the stock items based on the stock number.

Current Cost: Select this box to display the current Cost of each items.

Delimiter: Enter the delimiter, which can be a comma, full stop or hyphen. The segments are separated by these delimiters.

Format: This field displays the selected file formats with the delimiters.

File Name: Enter the file name with path from which data has to be imported. Alternatively, click to browse the directory and select the file.

Load: Click this option to load Item Details from Text file.

Cancel: To cancel and close the window.

 

Save: To accept the changes made and update the cost price as per the revised cost displayed.

Cancel: To undo all the changes made.

Exit: To quit the option.
