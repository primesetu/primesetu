Tax Register

The Sales Tax Register provides sales tax information for the selected period. The Summary option displays sales associated with the corresponding tax type and value, while the Detail option generates information on sales and tax at the bill-level.

How to reach: Reports > Sales > Tax Register

The Sales Tax Register - Report window is displayed.

From Date: Select the date from when the report details to be included. The default date selected is the current Shoper date.

To Date:  Select the date to when the report details to be included. The default date selected is the current Shoper date. You can enter any date but it cannot be greater than the Shoper date.

Select the type of report to be generated, i.e., Detail or Summary.

Detail: This report is at bill level. The bills are bifurcated under the respective groups and the respective bills are shown. It can also be grouped day wise.

Summary: To view only the summary of sales tax.  

View: To view the report

Exit: To exit the window
