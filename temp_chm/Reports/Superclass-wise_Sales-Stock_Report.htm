Superclass-wise Sales / Stock Report

The Superclass-wise Sales/ Stock report provides information on quantity and value of the sales made for a given period and current stock position at super classification level. The value of stocks and sales can be based either on cost price or on selling price.

How to reach: Reports > MIS > Superclass-wise – Sales/ Stock

The SuperClassWise Sales & Stock Summary Report window is displayed.

 

Dates: By default, the From and To are the current Shoper date. You can enter any date range but the To date cannot be greater than the Shoper date.

Super Class: Select Department or Buyer to identify the super class on which the report has to be generated. Select All or Selected to specify the super class values to be included in the report. If you have chosen Selected, select the required value from the corresponding list.

Class: Select Product or Brand to identify the class on which the report has to be generated. Select All or Selected to specify the class values to be included in the report. If you have chosen Selected, select the required value from the corresponding list.

Value At: Select Cost Price or SellingPrice as required, for the value computation of stock and sales.

Display: Select Sale Quantity Value to display the sale quantity and value.

View: To view the report

Clear: To cancel the entries made

Exit: To close the window
