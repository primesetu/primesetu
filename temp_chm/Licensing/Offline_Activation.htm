Offline Activation

This option is used when there is no internet connection to the computer where Shoper 9 license has to be activated. This option creates an activation request file and this file can be submitted using Shoper 9 product or TallyAdmin tool from a system with internet connection.

This window is displayed when you try to activate / reactivate Shoper 9 from a system without internet connection.

Retry: To activate license if internet connection is restored.

Continue in offline mode: To create the activation request file.

Submit: To submit the request.

 

Note: The request file Shoper_req.lic is created in the Shoper application folder.

 

Send External Request

To send external request, copy Shoper_req.lic file from Shoper 9 application folder of the offline machine to the system with active internet connection. You can use TallyAdmin tool or Shoper 9 product to send external request.

In Shoper 9 start-up screen select Manage License

The Manage License window is displayed as shown.

Proxy Configuration: To configure the proxy server to connect to the internet (if required).

List of License Servers: To list the License Servers accessible on the network.

Note: The option Send External Request is displayed only if  Shoper_req.lic file is available in Shoper 9 application folder.

Continue: To send the request file.

 

Note: A response file is created in the Shoper 9 application folder and the unlock key is sent to the e-mail ID given in the request file.

Transfer the response file back to the Shoper 9 application folder that has to be activated.

Unlock the license using the unlock key received by e-mail.  Click here to learn more.
