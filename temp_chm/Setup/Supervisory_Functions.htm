Supervisory Functions

Supervisory Functions submenu allows you to perform certain administrative activities to ensure security for various processes. This option also facilitates certain activities that need to be performed occasionally to ensure the smooth functioning of Shoper 9 POS.

Various activities that can be performed under this option are:

  • Node Management

  • Data Rebuild

  • Cost Price Variance Fixing

  • Year End Process

  • Re-Open Day

  • Database Archival

  • Manage DayEnd/ DayBegin Activity List

  • Security Management

  • Price Revision Authorisation

  • Item Reclassification
