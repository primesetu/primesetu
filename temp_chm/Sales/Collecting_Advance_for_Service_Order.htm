Collecting Advance for Service Order

This option allows you to collect advance payment for a service order made. The details of the advance payments are captured through the Cash Receipts window displayed on clicking Payment button.

The Cash Receipts window is displayed.

Document No.: The default document number of the Service Order is displayed.

The type of order is displayed in a separate box, i.e., Service Order

Net Payable: The total value of items entered in the Service Order is displayed.

The payment details grid displays the details entered like payment mode, description, units, value etc.

Mode: Select the mode of payment from the drop down list.

Note: If credit card is selected as the mode of payment, the card number and authorisation slip number are to be accepted. If the mode of payment is through cheque, then the cheque number and date are mandatory.

Code: Select the code for the selected mode of payment.

Description: The description of the selected mode of payment is displayed.  

Units: Enter the amount received. This is applicable only for Cash payment.

Exchange Rate: The currency exchange rate as configured is displayed in this field. This is applicable only in the case of Cash payment.

Value: Enter the amount received by each mode of payment. The amount (i.e., Units * Exchange Rate) is displayed in the case of Cash payment.

Ok: To save the payment details along with the service order

Cancel: To cancel and clear the screen for fresh entry

Exit: To exit from the payment window without saving payment details
