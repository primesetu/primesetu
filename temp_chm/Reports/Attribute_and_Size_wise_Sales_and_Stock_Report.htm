Attribute-wise Sales and Stock Report

The Attribute-wise Sales and Stock report gives you details of size-wise sales and stock based on all the item classifications for a selected period.

How to reach: Reports > MIS > Attribute-wise Sales and Stock

The Attribute & Sizewise Stock and Sales report window is displayed.

Filters

Date Range: By default, the From and To dates are the current Shoper date. You can enter any date range but the To date cannot be greater than the Shoper date.

Only Transacted Items: Select to generate the report including only the transacted items.

Skip Zero Balance: Select to skip the items with zero stock quantity.

Skip Stock Less Than: Specify the stock quantity, so that the items with quantity less than the specified quantity will be skipped. For example, if you specify zero (0), stock items with negative stock quantity will be skipped.

The left most list gives all the item classifications enabled in the system parameters. On selection of each classification, the corresponding lookup values catalogued are displayed under Code and Description. You can select/ deselect the values as required. If you have made partial selection of values, the corresponding item classification will be marked as half checked (grayed). However, you cannot have an empty list for any item classification.

Select All: To select all the values under the selected item classification.

Unselect All: To unselect all the values under the selected item classification.

Columns To Display

Select the columns to be displayed on the report screen. The position of the columns can be sorted as per requirement using the buttons and .

This list shows the following column names:

  • The Code and Description of all the item classifications listed

  • General columns like Retail Price and Cost Price of stock items

  • The sale and stock quantity and value columns of stock items

Note: The position of sale and stock columns cannot be changed while selecting the columns to display in the report. Selection of minimum one column from the item classification attributes is mandatory. Additionally, selection of minimum one column from the sale and stock attributes (the column names that are grayed and placed towards the end of the list) is mandatory for the report generation.

Report Display Format

There are three report display formats available for this report.

Pivot On Sizes: Select to display the report based on size groups.

Pivot On Selected Columns: Select to display the report based on the attributes selected under Columns To Display.

Pivot On Sizes And Selected Columns: Select to display the report based on both size groups and the attributes selected under Columns To Display.

Reset: To cancel the selections made

View: To view the report

Exit: To close the report selection window
