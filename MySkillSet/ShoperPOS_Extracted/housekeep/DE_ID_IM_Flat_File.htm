ID – IM Flat File

ID and IM files are used for the data consolidation at HO.

How to reach: Housekeeping > Data Export Masters > ID - IM Flat File

The Exporting Flat File window for ID FILE CREATION is displayed.

 ID/ IM File: Select any one as per the requirement

ID File:  ID file contains item-wise summary of the transactions for the day. You may create the ID file for a single day or any period by providing the Starting Date and the Ending Date.

Ok: To create ID File in the Shoper-Out directory with a naming pattern IDsccMMDD.seq where scc = Shoper Company Code, MM = Month, DD = Date.

Cancel:  To clear the selections made

Exit: To quit the option

 

IM File:  IM file contains item-wise summary of the transactions for the entire month. IM file is created at the first day of each month, by default. You can also create the IM file for a Month on a particular date.

 

Ok: To create IM File in the Shoper-Out directory with a naming pattern IMsccMMYY.seq where scc = Shoper Company Code, MM = Month, YY = Year

Cancel:  To clear the selections made

Exit: To quit the option
