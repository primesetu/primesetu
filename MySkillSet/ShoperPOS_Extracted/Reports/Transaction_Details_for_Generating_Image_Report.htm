The Transaction(s) tab on the Image Report Selection window is as shown.

Select Transaction Type: Select the required type of transaction from the list.

The fields Date From and To are enabled for report selection criteria, by default. Disable the date fields to generate the report based on some criteria other than date range. To generate reports based on other criteria deselect the check box next to the date fields.

Date

From: Select the date from which the report has to be generated. By default, the current Shoper Date is displayed. Change the date, if required.

To: Select the date to which the report has to be generated. By default, the current Shoper Date is displayed. Change the date, if required.

Doc No Prefix: Enter the document prefix of the transaction.

Doc No From: Enter the starting document number.

Doc No To: Enter the ending document number.

Party/Customer Code: Enter the customer code.

Party/Customer Name: Enter the customer name.

Click Display Fields to display the tab.
