Adding Payment Mode

The Mode of Payment Catalogue window is as shown.

 

Mode of Payment: Select the mode of payment from the list. Various modes possible are Cash (local or foreign currency), Credit Card, Debit Card, Discount Card, Gift Coupons, Advance Receipt, Cheques, Credit Note, On Account, Payment on Delivery, Consumer Finance and Exchanged Item Value.

If you have selected Cash, Credit Card, Discount Card or Gift Coupon, fields corresponding to these payment modes are displayed. If mode of payment selected is Advance Receipts you just need to specify if the selected payment mode can be used at POS. In Cash, Cheque, Credit Note and On-Account will be enabled by default.

If mode of payment selected is Exchanged Item Value, you can specify the code for that mode and is unique.

If mode of payment selected is Consumer Finance, you can specify the code and define the Billing Component and Finance Scheme details.

Common fields for all payment modes are Payment Code and Applicable.

Payment Code: Enter a code for the payment code being catalogued. Ensure that payment code entered is not similar to the names of default payment modes in Shoper. For e.g. Do not create a payment code - Cash, which is a name for default payment mode.

Applicable: Select to indicate that the instrument can be allowed for use at POS.

OK: To save the payment mode details.

Cancel: To undo the values chosen. Click this button a second time to cancel the functionality chosen.

Exit: To come out of the menu.

Settings: To include or exclude the payment modes for configuration.
