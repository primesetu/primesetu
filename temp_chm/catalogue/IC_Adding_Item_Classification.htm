Adding Item Classification

Combo Specific

The Item Classification Catalogue window is displayed.

Product/ Brand: Can either be keyed in or selected from catalogue browse window which opens on pressing F2. If the required ID is not already catalogued in the General Lookup, the same can be inserted by pressing Insert key. On pressing Insert key, the ID and description are accepted and inserted in the General Look Up.

Check if items combination is for sale: Select if the items combination is for sale. Unselect if the items such as gift items which are not for sale but whose inventory have to be tracked.

Check If Service Item: Select if the combination is a service item.

Check If Consignment Item: Select if the combination is a consignment item for which the payment is made to the vendor after selling the items.

Size Group: Select the size group from the list catalogued in General Look Up

Product Tax Type: Select the type of tax from the list catalogued in General Look Up

Preferred Vendors

Primary Id: Enter the Vendor Id already catalogued in Vendor Catalogue. You may press F2 to browse for vendors catalogued and select the required vendor.

Name: The name of the selected vendor is displayed

Mark-Ups Applied

For Retail Markup: Enter the percentage mark-up over cost price to arrive at the retail price

For Dealer Mark-Up: Enter the percentage mark-up over cost price to arrive at the dealer price

Department: Select the Super Classification-1 (the caption of which is Department in this installation as shown in the above screen) catalogued in General Look-Up

Category: Select the Super Classification-2 (the caption of which is Category in this installation as shown in the above screen) catalogued in General Look-Up

Next: To go to Size Catalogue

Cancel: To cancel the details entered
