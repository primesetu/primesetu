Selection of Schemes

Selection of discount/ sales promo schemes in billing can be either automatic or interactive (guided manual).

There is no user interaction required to select promos in the case of automatic selection.

In manual mode, you may select the required discount code or configure the system to select the discount code automatically. If auto selection of discount code during item entry is enabled, the highest priority promo will be chosen by default. However, you can change the promo, if required, before application.
