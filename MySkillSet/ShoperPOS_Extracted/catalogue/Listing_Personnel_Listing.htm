Personnel Listing

This report shows the details of the selected personnel including code, name, type and joining date of the personnels.

How to reach: Catalogue > Listings > Personnel Listing

The Personnel Listing window is displayed.

 

Period: Select the From and To dates.

Report Options

Select Specific or All to identify the personnel whose details are required. If you select Specific, you need to select the personnel from the list. Else, click to browse.

View: Click to view the personnel details

Exit: Click to close the window
