Proxy Configuration

This option is used when you have to configure the proxy server to connect to the internet (if required).

The Proxy Configuration window is as shown.

Use HTTP Proxy Server: Select to enter Proxy Server URL.

Proxy URL:

URL: Enter Proxy Server URL through which you can access the Internet.

Authentication Required: Select, if the proxy server requires authentication. This will be enabled only after entering the URL.

Username: Enter the user name.

Password: Enter the password.

Connection Configuration:

Connection Timeout (in seconds): Enter the duration in seconds. This is the duration for which the system would wait for internet connectivity.

Use absolute URL for HTTP actions: Select to use absolute URL for HTTP actions. If an external firewall is configured in the system then the firewall may expect full path from the application. In such cases, select this option to enable full path.

Ok: To save the proxy configuration.

Cancel: To cancel the  proxy configuration process.
