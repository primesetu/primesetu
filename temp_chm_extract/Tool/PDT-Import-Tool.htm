Script Generation – PDT/Import Extensions

The template PDT/ Import helps you to generate scripts for pre-update extensions enabled as new options in any existing functionality of the standard Shoper 9 POS.

On selecting 1.PDT/ Import tab, the relevant details captured are displayed.

Module: Select the module in which the new extension is to be enabled. The modules where the selected type of extension is allowed are listed in the drop down.

Button Caption: Give an appropriate caption for the extension

File Format: Select the required file format

Field Delimiter: Select the required file delimiter

Additional Parameter: Enter additional details, if any

DLL-Class Name: This grid gives the details of DLLs used to enable the extension

Select DLL: Click to select the required DLL

DLL Name: The name of the selected DLL is displayed

Class Name: The Class name of the selected DLL is displayed

Mandatory: Select, if the DLL is mandatory to enable the extension

Generate Script: To generate the script

Cancel: To cancel the selections made
