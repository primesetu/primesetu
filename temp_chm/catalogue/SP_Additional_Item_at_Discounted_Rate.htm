Additional fields are:

Based on: Discount can be given based on item rate/ value.

Item Quantity: This option is selected by default and cannot be altered.

Item Rate/Worth: Select if the discount is based on item rate or item worth. When you select this, the possible comparison operators that can be used to specify the selection criteria are filled in the list. The choices are Rate =, Rate >, Rate <=, Rate Range, Worth =, Worth >, Worth <= and Worth Range. Select the appropriate one.

Buy: Enter the criteria to be satisfied by item quantity and rate/ amount, to allow discount. For example, in the given definition, if you buy two items of rate more than Rs. 1000/-, you will get 20% discount on the MRP of the 3rd item.

Percentage/ Amount: Enter the percentage or amount to be given as discount for the item obtained by this scheme. The percentage/ amount of discount is calculated on MRP.

Additional Item at Discounted Rate

This promotion type helps you to define schemes that can be applied at the item level (specified items) during billing. This promotion type is used to define schemes such that when you buy a specified number of pieces of an item, you get discount on an additional item. You can apply the discount type to the items selected based on item classifications.

The details to be captured can be divided into two parts, namely, Buy Items and Discount Details.

Buy Items

This section allows you to select the items to be sold with discount.

The Buy Items section is as shown.

Configure: Click to configure the options for item selection. Click here for more information.

The Buy Items grid displays the item classification columns as configured. On pressing F2 in any item classification column, the corresponding browse window is displayed with the list of catalogued values to select from.

Note: In the Buy Items grid:

   · You can press F2 anywhere in the grid, irrespective of the order in which the item classification columns are present, to select values of any attribute. Click here to learn more about value selection.

   · You have the flexibility to select either one or multiple attributes in a row, to specify the required items.

   · If Stock No. of an item is selected, the remaining item classification columns for that stock number will be filled automatically.

Discount Details

This section allows you to capture discount details for the items selected in Buy Items grid.

The Discount Details section is as shown.

Buy: Enter the criteria to be satisfied by item quantity to allow discount. For example, in the given definition, if you buy two items, you will get 10% discount on the MRP of the 3rd item.

Percentage/ Amount: Enter the percentage or amount discount to be given on the additional item. The discount is given on MRP. If you enter Percentage, the Amount column will be disabled and vice-versa.
