Cash as Payment Mode

Currency Name: Enter description or name of the currency.

Local Currency: By default, the local currency is catalogued. Only one currency can be set as local currency and hence, for all subsequent entries the checkbox is disabled.

Decimal Description: Enter the name of the decimal component of the currency being catalogued.

Conversion Rate: The rate at which the currency is converted to the local currency. Thus, for the local currency this value is always 1.

Rate as On: Displays the date (Shoper Open Date) and time at which the currency details are entered.

Rate Change Frequency: Enter the frequency, in terms of hours or days, at which the conversion rates are expected to change.

Denomination Details: Click Add to enter the details of denominations of the currency being catalogued. This field is visible only if the system parameter Denomination Applicable under the category Setup is enabled.

Allow in PO: Select if this currency is allowed to be used in Purchase Orders.

Allow in Tender: Select if this currency is allowed to be used in Tenders.

Allow in Payouts: Select if this currency is allowed to be used in payments. Disabling this enforces balance return in local currency.
