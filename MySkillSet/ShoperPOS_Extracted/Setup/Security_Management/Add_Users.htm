Add Users

The window for adding new users is displayed.

User ID: Enter login ID of the new user to be created.

User Name: Enter name of the new user. This name is used in greeting the user and referring to an ID.

Password: Enter a new password for the new user.

Confirm Password: Enter the password again to confirm.

Admin User: Select to provide administrator rights (Full rights) to this user.

Group: Select to assign the user to groups.

 Group Id: Select to assign the user to group.

Add: To add new group(s)

Ok: To confirm selected group(s)

Cancel: To cancel the selection

 

Company: Select to assign the user to companies.

 

Comp Code: Select to assign user to the company

Applicable for All Companies: Select to assign user to all the companies created and also the companies created in future.

Ok: To confirm selected company

Cancel: To cancel the selection

Account Expiry Date:Enter the expiry date for user login.

Country Code: Enter the Country code.  +91 appear by default. Change if required.

Mobile No.: Enter the mobile number.

E-mail ID: Enter the e-mail ID.

Ok: To confirm the authorisation process

Cancel: To abort the authorisation process

Exit: To close the Security Management window
