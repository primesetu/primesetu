The PDT Import option is displayed.

Import from: Select File as the mode of import.

Field Template: Select the appropriate format. The available formats are:

    • Stock Number

    • Stock Number + Qty + Rate

    • Stock Number + Rate + Qty

    • Stock Number + Qty

Note: The default delimiter of the formats is ~. You can set the default delimiter to your choice by changing the value of the system parameter Delimiter Character for PDT Import in Sales Transaction under the category Billing.

Path: Enter the path of the file containing the required item details. Alternatively, browse and select the file name.

Clear: To clear the entries made

Ok: To load the item details from the specified file

The PDT Import option is displayed.

Import from: Select Transaction as the mode of import.

Transaction: This option allows you to import item details from various types of transactions.

Type: Select the required transaction type from the list. This list includes all transaction types (expected transactions excluded) made in Shoper application.

Note: If you select a sales related transaction (Sales/ Sales Return/ Void Sales), the item rates are taken from the respective transaction. In other cases, the item rates are taken from the item master according to the environment (Retail/ Distributor).  

Bill Prefix: Enter the document prefix of the required transaction.

Bill No: Enter the document number of the required transaction.

Clear: To clear the entries made

Ok: To load the item details from the specified transaction

Bill Header

The header part of a bill is displayed.

Select either Product or Service, as the type (configurable) of bill.

Select Cash or Credit as the transaction type (configurable) of the bill. In the case of Credit billing, customer selection is mandatory.

The consecutive boxes display the Bill Doc Prefix (Document Prefix of the bill) and the Bill Doc Number (Document Number of the bill), as per your Bill Prefix settings. The bill date and time are displayed by default and cannot be edited. The date displayed is the current Shoper date.

Customer: Enter customer code (configurable) in the first box. Enter the customer name in the next box. Alternatively, press F2 and select the customer code from the Browse window (Customer Search window). If the customer code is selected from the Browse window, the customer name is displayed in the bill.

Sales Staff: Select the sales staff Id (configurable) from the list, if all the items to be billed are handled by the same sales person.

Import: Click to load/ import the item details. The PDT Import window is displayed on selecting this option. There are two ways of importing item details as follows.

  1. Importing from file

  2. Importing from transaction

Recall: Click to recall the saved documents. The documents that can be recalled during billing are:

  • Sales Advice Slips

  • Sales Orders

  • Service Orders

  • Suspended Bill
