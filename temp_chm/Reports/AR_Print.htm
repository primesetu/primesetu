Title: Name of the report appears by default. You may change it, if required.

Sub-Title: Enter the sub-title, if any.

This title and sub-title appears on the print configurations sub-screen.

Additional Information:

Print with Company Name: Select Yes or No, as per your requirement. If you select No here, the field  Print with Company Address is skipped.

Print with Company Address: Select Yes or No, as per your requirement.

Print Date Range of Report: Select Yes or No, as per your requirement.

Print Page Numbers in Report: Select Yes or No, as per your requirement.

Page Numbering starts from: Enter the page number to be printed on the first page of the report printout.

Page Range: Enter the page number/s separated by comma (to print discontinuous pages) or enter starting and ending page numbers separated by hyphen (to print continuous pages) or enter All.

Print Mode: Select the required print mode from the Print Mode list. If you select Dot Matrix Format, a new field Compatibility appears on the Print Mode sub-screen.

Compatibility: Select the printer name from the Dot Matrix Printers list.

On the print configurations sub-screen, against Method, the selected print format appears. Number of columns and lines for printing changes based on the print mode selected.  If Dot Matrix Format or Quick / Draft is selected, preview option is disabled.

If the selected Print Format is Neat Mode, in Printer Name field, select the required printer name from the List of Printers.

If the selected Print Format is Dot Matrix Format or Quick / Draft, the Select Printer Type sub-screen appears.

Print to a File: Select Yes or No. If you select Yes, a new field File Name appears on the Select Printer Type sub-screen.

File Name: By default, the file name appears as <report name>.prn. Change the file name, if required. On the print configurations sub-screen, a new field, Print to File, appears with the specified file name against it.  

Printer Name: Select the printer name from the List of Printers.

The corresponding printer properties window opens. Make necessary selections and click OK.

Print

The print configurations sub-screen is as shown.

Yes: Select to print the report

No: Select to go back to the report view

You may change the print configurations using the buttons on the Right Button Bar. The following table explains the functionality of these buttons.

Button

Short-cut

Functionality

Alt+L

To change language. On selecting this option, the Language Configuration sub-screen appears.

Print Language: Select the required language from the List of Languages.

Alt+I

To enable preview of report. On selecting this option, on the print configurations sub-screen, (with Print Preview) appears at the bottom. The button label changes to No Preview.This is a toggle button.

Alt+T

To set report titles. On selecting this option, the Report Title sub-screen appears. Click here to learn more.

Ctrl+G

To set pages to print.  On selecting this option, the Page Range to Print sub-screen appears. Click here to learn more.

Alt+R

To select stationery. On selecting this option, on the print configurations sub-screen, against Method, to the right of the print format name, on Pre-printed Paper appears. The button label changes to Plain Paper. This is a toggle button.

Alt+F

To select print format. On selecting this option, the Print Mode sub-screen appears. Click here to learn more.

Alt+C

To set number of copies.  On selecting this option, the Number of Copies sub-screen appears. Enter the number of copies to be printed.

Alt+S

To select printer.  Click here to learn more.
