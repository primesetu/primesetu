Physical Stock Report

This option gives a report of the items for which physical stock is recorded during Physical stock taking.

How to reach: Stock > Stock Take > Physical Stock Report

Select the required options in the following window which comes up on invocation and click View to generate the report.

Batch Ref. No.

All: To select all the batches.

Specific: To select specific batches from the drop down list.

Product

All: To select all the products.

Specific: To select specific products from the drop down list.

Brand

All: To select all the brands.

Specific: To select specific brand from the drop down list.

Report By

Select to Product, Brand or Batch Ref.No. to specify the format in which the report has to be generated.

The screen after selection is displayed as shown.

 

View: To view the report

Print: To Print the report

Exit: To exit the report generation screen.
