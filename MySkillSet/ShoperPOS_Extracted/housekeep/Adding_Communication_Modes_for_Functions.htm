Adding Communication Modes for Functions

The Alert Management window is displayed.

Function: Select the required functionality, from the list.

E-Mail: Select, if the alert is to be sent as an E-Mail. Click here to know how to set E-Mail alerts.

On clicking E-Mail, the E-Mail Alert Definition window is displayed.

Configuration Name: Select the required configuration, if configured already.

Create: Select, if you want to create a new communication configuration. On selecting this option, the Communication Configuration window is displayed.

Specify the address details (To address, Cc & Bcc) to which the E-Mail alert is to be sent.

Subject: Specify the subject of the E-Mail.

Message: Specify the appropriate message to be sent.

Ok: To save the details entered

Cancel: To clear the entries made

SMS: Select, if the alert is to be sent as an SMS. Click here to know how to set SMS alerts.

On clicking SMS, the SMS Alert Definition window is displayed.

Configuration Name: Select the required configuration, if configured already.

Create: Select, if you want to create a new communication configuration. On selecting this option, the Communication Configuration window is displayed.

Mobile No.: Enter the mobile number to which the alert is to be sent.

Message: Specify the appropriate message to be sent.

Ok: To save the details entered

Cancel: To clear the entries made
