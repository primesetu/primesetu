Flat File Mapping — Detail tab

The Detail tab is as shown.

Destination Table Column Details: Displays the list of columns present in the selected destination table.

Column Details in Flat File: Select the columns of the destination table for which corresponding values are present in the flat file. These columns, based on the order of selection, are mapped to the corresponding values in the flat file.

All columns in the table need not be filled by values from the flat file and flat file may contain values that need not be used for filling the table.

When all columns in the selected destination table need not filled by values from flat file, select only the relevant columns for mapping to flat files.

In case there are some columns in the text file that need not be mapped to any column in the table, add blank field to the corresponding position in the list. For example, if you are mapping first and second values in the text file to some columns in the destination table and the third value in the text file has to be ignored, after selecting columns one and two, add a blank field. Then continue adding the remaining columns.

Use the buttons between the two lists to move the columns in or out of this list. See the table for functionality of the buttons.

Button

Description

Copy all column names in Destination Table Column Details to Column Details in Flat File

Copy current column name in Destination Table Column Details to Column Details in Flat File

Remove current column name from Column Details in Flat File

Remove all column names from Column Details in Flat File

Add blank field to Column Details in Flat File

 

The columns that are mapped to the values in the flat file are shaded with a green colour.
