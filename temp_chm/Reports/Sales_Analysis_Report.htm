Sales Analysis Report

The Sales Analysis report provides the sales information, based on item classification. This report helps you to understand the sales of each product-brand (class1-class2) combination. It also helps you to maintain a combination of items in your store.

How to reach: Reports > MIS > Sales Analysis

The Sales Analysis Report window is displayed.

Report Options: Select Entire List, A Product (class1), A Brand (class2) or A Product & A Brand (class1 & class2) to specify the items to be included in the report.

If you have selected:

Entire List: You can select either All Product or All Brand as per the requirement

A Product: You can select the required product from the list of products

A Brand: You can select the required brand from the list of brands

A Product & A Brand: You can select the required product and brand from the respective lists

Select Range of Months: By default, the From and To are the current Shoper date. You can enter any date range but the To date cannot be greater than the Shoper date.

Select Code or Description to view the respective values in the report.

Based On Qty/Val: Select Quantity or Value to show sales details in terms of the corresponding figures in the sales analysis.

View: To view the report

Exit: To close the window
