Component 1:

Name: Enter name of the tax component.

Desc.: Enter the description for the component.

Type: Select Fixed or Derived. If the Type selected is Fixed, enter the values in Comp. On and Tax Rate fields. Otherwise, these values need to be entered in the grid later.

Comp. On: Select the item based on which the tax component is computed.

Tax Rate: Enter/ modify the rate of tax for the component.

Repeat the values for the remaining components.

Ok: To copy the common values to the grid
