Size Management

Size group attached to each item and the order of sizes present in that group plays an important role in catalogue management. Size Management allows you to modify the existing size groups according to the changing requirements of your business.

Note: To create a new size group name, use General Lookup. To add sizes to the newly created size group name, use Item Classification.

 

How to reach: Catalogue > Size Management

The Size Catalogue Management window is displayed. In this window you can execute the following functions.

Merge Size Group: Click to merge different size groups into one. Click here for more information.

Split Size Group: Click to split an existing size group into multiple size groups. Click here for more information.

Rearrange Sizes: Click to change the order of size codes in size groups. Click here for more information.

Mark Applicable: Click to choose the relevant size codes in size groups. Click here for more information.

Exit: To quit the Size Catalogue Management option
