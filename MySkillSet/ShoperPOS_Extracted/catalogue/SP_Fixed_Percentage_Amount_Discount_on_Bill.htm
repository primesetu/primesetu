Fixed Percentage/ Amount Discount on Bill

This promotion type is to define schemes that can be applied as fixed percentage/ amount discount on each bill with or without any additional criteria. The discount percentage/ amount as per the definition will be displayed during billing. You cannot alter the values at the time of billing, as the discount type is fixed.

Discount Details

The Discount Details is as shown.

In terms of: Select Percentage or Amount to specify how the discount is going to be calculated. If you select Percentage, the Amount column in the grid will be disabled and vice-versa.

Based on:

Bill Value Range: Select if the discount varies according to the range of bill value. Click here to view the discount details captured based on Bill Value Range.

The Discount Details section changes to accept Bill Value Range as shown.

Enter the criteria to be satisfied by the total bill value to allow discount.

Bill Quantity Range: Select if the discount varies according to the range of billed quantity. Click here to view the discount details captured based on Bill Quantity Range.

The Discount Details section changes to accept Bill Quantity Range as shown.

Enter the criteria to be satisfied by the total bill quantity to allow discount.

None: This option is selected by default indicating that the discount is applied on each bill without any additional criteria.

Bill Level Discount: Enter the percentage of the bill value or the value to be given as discount.

3. More>>: Click to capture more details, if required. This button toggles between 3. More>> and 3. Less<<.

The Discount Details section changes as shown.

Subject To Max. Discount Allowed: Enter the maximum percentage of the bill value or value that can be allowed as discount for the bill. If you enter a number in Percentage, Amount will be disabled and vice-versa.
