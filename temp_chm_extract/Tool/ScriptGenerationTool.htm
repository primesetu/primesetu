Extension Script Generator Tool

Extension Script Generator Tool is used to generate the corresponding script which contains the SQL statements to be inserted in GenLookup table for configuring the required custom DLLs.

To generate scripts using this tool, double click the GenExtnScripts EXE.

The Extension Script Generator window is displayed.

image...

Select the required template to generate the corresponding script. The templates available are:

  • PDT/ Import Extensions

  • Transaction Extensions

  • Post-update Extensions
