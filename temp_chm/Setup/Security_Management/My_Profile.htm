My Profile

This option is used to change your profile. In this window, you can add your mobile number and e-mail ID.

How to reach: Setup > Supervisory Functions > Security Management

On clicking My Profile, the Security Management – My Profile window is displayed.

User ID: Displays the logged in user ID.

User Name: Displays the existing user name. Change if required.

Mobile No.: Enter your mobile number.

E-mail ID: Enter your e-mail ID.

Ok: To save the applied changes

Exit: To close the Security Management window
