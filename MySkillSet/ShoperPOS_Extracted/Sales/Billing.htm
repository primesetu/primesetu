POS Activities

The Billing option in the main menu leads to a window that allows all operations related to billing. These include generation of a new bill, cancellation of a bill, recording sales returns, reprinting a bill or sales return document, and change bill prefix for cash and credit transactions.

How to reach: Sales > Billing

 

The Billing window has the following components as shown.

Tool Button

Shortcut Keys

Functionality

 

Alt+1

Create a new bill

Alt+2

Void/ cancel a bill generated earlier

Alt+3

Record sales returns with reference to an earlier sale (Bill/ Cash Memo)

Alt+5

Record sales returns without reference to an earlier sale (i.e., when the customer does not have the copy of bill and when it is not possible to trace out the bill through which the sale was made earlier)

Alt+6

 Reprint a bill or sales return document

 

—

Change the bill prefix for cash and credit transactions, when more than one bill prefix is catalogued

 

Note:

•  Only one among the New Bill, Return and Return without reference can be used at a time.

•  Alter Prefix can be used only when no other functions are in use.

All functions are not applicable in all transactions. See the chart for the list of functions and its applicability.

Transaction Prefixes

The prefixes to be used for billing (here, Cash Transactions and Credit Transactions) under a Product and Service Bill as defined in the Bill Prefix are displayed. If you carry out billing without defining the bill prefixes, you will get appropriate messages.
