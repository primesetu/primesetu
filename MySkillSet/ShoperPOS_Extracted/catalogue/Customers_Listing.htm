Customers Listing

This report shows the details of the selected customers including code, name, price group to which the customer belongs, personal details and contact details.

How to reach: Catalogue > Listings > Customers

The Customer Details - View window is displayed.

 

Report Options

Party Type: Select the party, i.e., Customer, Vendor or Chainstore, from the list

Select Specific or All to identify the customers whose details are required. If you select Specific, you need to select the customers from the list. Else, click to browse.

View in Spread Format: Select to view the report in a spreadsheet format.

View: Click to view the customer details

Exit: Click to close the window
