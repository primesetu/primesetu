Excise Invoice Generation

Generally, an Excise Invoice is printed at the time of Sales or any form of outward transactions of the manufactured goods for the first time, from showrooms/ warehouses/ godowns. In Shoper 9 POS, you can generate or print the excise invoices for transaction where excise duty is collected. The Excise Duty is applicable in Sales or any form of outward transactions of the goods. The Excise Duty details are applied in these transactions as defined in the Excise Duty catalogue.

There are few parameters introduced under the category named Excise under Setup > General > System Parameters to manage some of the necessary settings of the Excise Duty Invoice to be generated/ printed in Shoper 9. The Excise Duty Invoice functionality in Shoper 9 can be accessed by setting the value as 1 for the parameter Whether Excise Duty is Applicable. By default, the value of this parameter is set to 0 (zero).

How to reach: Sales > Excise Invoice Generation

The Excise Invoice window is displayed. In this window you can execute the following functions.

New: Click to generate a new Excise Invoice. Click here for more information.

Edit: Click to edit an existing Excise Invoice. On clicking this option Excise Invoice Edit window is displayed. Specify the transaction type, document prefix and document number of the Excise Invoice to be edited. The corresponding details are displayed. Make necessary changes and click Ok.

Reprint: Click to reprint an existing Excise Invoice. On clicking this option Excise Invoice Reprint window is displayed. Specify the transaction type, document prefix and document number of the Excise Invoice to be reprinted. The corresponding details are displayed. Make necessary changes and click Ok.

Exit: To exit from Excise Invoice window
