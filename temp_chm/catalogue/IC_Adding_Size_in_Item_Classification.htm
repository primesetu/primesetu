Adding Size in Item Classification

Size Catalogue

Item Classification Catalogue (Size Catalogue) Window is displayed.

 

Copy From: Enter the Product and Brand when the sizes used for the specified combination are applicable to the current Product-Brand combination being entered.

Go: Click this to copy and display size details of entered Product-Brand combination.

Size Grid: Displays size details of Product-Brand combination from which they are copied. Irrespective of whether Copy From is opted or not, new size details can be entered. If Copy From is not used then enter the applicable sizes.

Previous: To move back to the Combo Specific page

Ok: To save the entered details

Cancel: To cancel the details entered.
