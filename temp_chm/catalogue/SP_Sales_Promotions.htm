Sales Promotions

The sales promotion catalogue is the repository of various promotion schemes that can be applied at a POS location. Multiple schemes can be defined to be implemented during the same period. Implementation is based on the priority assigned to each scheme. If there are two valid schemes with the same priority, then the one created last would be applied.

How to reach: Catalogue > Define Sales Promotions

The Sales Promotion Catalogue window is displayed. In this window you can execute the following functions.

Add: Click to add a new promotion scheme. Click here for more information.

View/Manage: Click to view or manage an existing promotion scheme. Click here for more information.

Exit: To quit the Sales Promotion catalogue option.
