HO Chain Stores

This option is used to record details regarding Head Office and chain stores. The option to allow or restrict Goods Inwards and Outwards from the location can be specified in this window.

How to reach: Catalogue > HO Chain Stores

The H.O/Stores Catalogue window is displayed. In this window you can execute the following functions.

Add: Click to add new chain store details. Click here for more information.

Edit: Click to edit existing details. Enter the code/ name or press F2 and select from the list displayed. Make necessary changes as explained in Add option. Click Ok to confirm modification.

Delete: Click to delete existing details. Enter the code/ name or press F2 and select from the list displayed. Click Ok to delete the details.

Exit: To quit this option.
