Counter-wise Details

The Counter-wise report provides counter-wise transaction information for all cashiers or a specific one for a specified period. A detailed report or an abstract based on payment mode can be generated. All types of billing transactions such as sales, sales returns, receipts, pay outs, and void sales are considered in this report.

Detailed report gives information of each transaction done at different nodes along with details of mode of payment used for each transaction. Summary report gives the total sales at each node for a particular day by different payment mode. It also gives total number of bills generated at each node for a particular day as well as total sales value.

How to reach: Reports > Cash > Counter-wise Details

The Counter Wise Report window is displayed.

Date Range: By default, the From Date and To Date are the current Shoper date. You can enter any date range but the To Date cannot be greater than the Shoper date.

Criteria Selection: Select Tender - Detail Report or Tender - Summary Report to specify the type of report to be generated.

Cashierwise Selection: Select All Cashiers or Specific Cashier to generate the report accordingly.

View: To view the report

Exit: To close the window
