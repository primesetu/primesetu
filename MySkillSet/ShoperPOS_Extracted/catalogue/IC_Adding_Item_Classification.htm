Adding Item Classification

The Item Classification Catalogue window is displayed.

 

Product/ Brand: Can either be keyed in or selected from catalogue browse window which opens on pressing F2. If the required ID is not already catalogued in the General Lookup, the same can be inserted by pressing Insert key. On pressing Insert key, the ID and description are accepted and inserted in the General Look Up.

Item Type

Consignment Item: Select if the combination is a consignment item for which the payment is made to the vendor after selling the items.

Billable Items: Select if the items combination is for sale. Unselect if the items such as gift items which are not for sale but whose inventory have to be tracked.

Service Item: Select if the combination is a service item.

Size & Tax Group

Size Group: Select the size group from the list catalogued in General Look-Up.

Product Tax: Select the type of tax from the list catalogued in General Look-Up.

Super Classification

Department: Select the Super Classification-1 (the caption of which is Department in this installation as shown in the above screen) catalogued in General Look-Up.

Buyer: Select the Super Classification-2 (the caption of which is Buyer in this installation as shown in the above screen) catalogued in General Look-Up.

Preferred Vendors

Primary Code: Enter the Vendor Id already catalogued in Vendor Catalogue. You may press F2 to browse for vendors catalogued and select the required vendor.

Primary Name: The name of the selected vendor is displayed.

Note: If required vendor is not available, then vendor can be created in vendor catalogue without closing the Item classification screen. The Item classification screen will be refreshed to display the vendor during browse.

Mark-Ups Applied

Retail Price: Enter the percentage mark-up over cost price to arrive at the retail price.

Dealer Price: Enter the percentage mark-up over cost price to arrive at the dealer price.

In the size grid enter the applicable sizes and choose the pivotal one, if any.

Note: In System Parameters, if Batch / Grade or Location is enabled for Specific items Classification then Batch / Grade or Location has to be manually configured for the corresponding class12 combo.

Grade Details

Enable Grade: Select to enable grade for the corresponding Class12Combo

Price Applicable: Select if different prices are applicable for batches.

 Location Details

Enable Location: Select to enable location for the corresponding Class12Combo.

Batch No. Details

Batch No. Applicable: Select if batch number needs to be considered for the corresponding Class12Combo

Maintain Batch-wise Prices: Select to accept batch wise prices for the corresponding Class12Combo.

Mfg. Date Applicable: Select to accept manufacturing date for the corresponding Class12Combo

Format: Select the format in which the manufacturing date has to be displayed.

Exp. Date Applicable: Select to accept expiry date for the corresponding Class12Combo

Format: Select the format in which the expiry date has to be displayed.

Accept Shelf Life:  Select to accept shelf life duration for the corresponding Class12Combo

Allow Expired Items in Sales: Select to allow sales of expired items for the corresponding Class12Combo

Allow Expired items in Sales Return: Select to allow accepting of expired items during Sales Return

Stop Sales of Items before days of expiry: Enter the number of days to stop selling of items reaching expiry date. It means, if this is set as 3, three days before reaching the expiry date, it will stop in billing. These items will not appear in Billing.

Delivery Instructions: The options here are displayed only when System Parameters, Service Type Captions (1 to 5) under Delivery Process are configured.

Door Delivery: Select to deliver the items to the customer’s door.

Installation Required: Select to provide installation of sold item at the customer’s place.

Demo Required: Select to provide a demo of the items’ working to the customer.

Wall mount required: Select to mount the item on the wall at the customer’s place.

One Year Warranty: Select to record the warranty for one year on the items sold to the customer.

Note: The above selections are made only if the parameters, Service Type Captions (1 to 5) are configured under WH Functions under System Parameters.

Grid Mode: Click this button to add entries in spreadsheet mode.

Normal Mode: Click this button to revert to normal view from grid view.

Ok: To accept the entered information

Cancel: To close the screen without saving.
