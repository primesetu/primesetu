Change Password

This option is used to change your password. A Super User can change the password for any user without entering the old password.

How to reach: Setup > Supervisory Functions > Security Management

On clicking Change Password, the Security Management - Change Password window is displayed.

User ID: Enter / Browse the User ID for which the password has to be changed.

Note: To change the password of a user other than the currently logged in user, you should have supervisory rights.

User Name: The name corresponding to the ID is displayed.

Old Password: Enter the current password of the specified login ID.

New Password: Enter the new password for the specified login ID.

Note: The password can also be blank.

Confirm Password: Re-enter the new password for confirmation. If retyped password does not match the earlier entry, a message Incorrect Password is displayed.

Ok: To save the changes in the password

Cancel: To clear the selection

Exit: To close the Security Management window
