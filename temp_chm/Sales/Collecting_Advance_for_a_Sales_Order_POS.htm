Collecting advance for a sales order

This option allows you to collect advance payment for a sales order made. The details of the advance payments are captured through the Cash Receipts window displayed on clicking Payment button on the sales order generation window.

The Cash Receipts window is displayed.

Document No.: The document number of the Sales Order is displayed in this field.

The type of order (i.e., Sales Order) is displayed at the top right corner of the window.

Net Payable: The total value of items entered in the Sales Order is displayed in this field.

The payment can be made through various payment modes. For a single payment, you can use more than one payment modes. The details of the payment made get displayed in the Payment Details grid.

Mode: Select the mode of payment from the drop down list.

Code: Select the appropriate code for the chosen payment mode, from the drop down list.

Note: If credit card is selected as the payment mode, the card number and authorisation slip number are to be entered. If cheque is the selected as the payment mode then the cheque number and date are mandatory.

Description: Enter the details of the payment.

Units: Enter the amount received. This is applicable only for Cash payment.

Exchange Rate: The currency exchange rate as configured is displayed in this field. This is applicable only in the case of Cash payment.

Value: Enter the amount received by each mode of payment. The amount (i.e., Units * Exchange Rate) is displayed in the case of Cash payment.

Ok: To save the payment details along with the sales order entered

Cancel: To cancel and clear the grid for fresh entry

Exit: To exit from the payment window without saving the payment details
