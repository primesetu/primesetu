Tally Printing Configuration

Tally printing can be configured as per user specification.

Configuration Name: Select the configuration from the list or enter a new configuration

Template Name: Enter the name of the template

TCP Location: Browse and select the required TCP

Printer Name: Select the printer

Print Mode: Set the print mode

No. of Copies: Enter the number of copies required

Enable Preview: Select to preview before printing the transactions

Set as Default: Select to set the configuration as default for the selected transaction

Edit the parameter name and value from the displayed list or enter the values for a new configuration.

Enter the Parameter Value as Yes to enable the parameters and No to disable.

New: To create a new configuration

Save: To save the settings for the configuration

Delete: To delete the configuration

Close: To exit the window
