Stock Taking based on Item Classification

To enter Items, classification-wise, select Item Class.

The items grid can be filled in three ways:

  • Manually keying in the five basic item classifications

  • Importing the file containing the Items.

  • By pressing F2 , browsing through the Item Classification Browse window and selecting the required value
