Sample 1 - Extension at Detail Level of Billing

In billing, while entering the quantity of an item, display a message requesting to place an order for the selected stock item, if the balance quantity of the stock item reaches the reorder level set for the same. Additionally, display the remaining quantity of the stock item in the same message.

The details of the extension to be enabled to facilitate the requirement are given below.

Extension Type: Pre-update extension at the Detail Level of Billing.

Transaction: Sales

Exe Name: SR130000

Project Name: ReorderLevel

Class Name: clsReorderLevel

Function: ShowDllForm1

DLL Name: ReorderLevel.dll

Records to be inserted in General Lookup (GenLookUp) table

SQL statements to insert records in GenLookUp table are:

Insert Into GenLookUp (Recid,Code,Descr,Flag) Values (14500,'2100+2+1','ReorderLevel.clsReorderLevel',1)

Insert Into GenLookUp (Recid,Code,Descr,Flag) Values (14600,'2100+2+1','Y1',1)

Output

The Billing window is displayed.

Here, each time when you select the stock item 000002 during billing, if the balance quantity of the item is Equal to or Less than 40 (Balance quantity <= 40), you will be alerted with this message.
