View / Manage Sales Promotions

The View / Manage Sales Promotions window is as shown.

Select Operation: Select the required operation from the drop down list. The operations provided are View Sales Promotions, Activate Sales Promotion(s), De-Activate Sales Promotion(s), Change Applicable Period For Sales Promotion(s), Archive Sales Promotion(s) and Change Priority For Sales Promotion(s)

Filter criteria: Enter any of the values in this group to recall the existing sales promotion definitions. You may enter values for multiple fields to filter and retrieve the relevant definitions that satisfy all the conditions specified.

Code: Enter the sales promotion code to retrieve a specific definition to view/ modify.

Description Like: Enter a part of the description of the scheme/s you want to view/ modify. All definitions that satisfy the given description would be retrieved.

Priority No.: Enter a number to recall all definitions with the specified priority.

Definition Type: Select the definition type as Discount or Offer.

Definition Level: Select Item Level or Bill Level.

Definition Name: Select the required definition name from list that is displayed based on the previous selection.

Definition F/V: Specify whether the discount/ offer is fixed or variable.

Start Date: Select the start date of the discount/ offer schemes.

End Date: Select the end date of the discount/ offer schemes.

Happy Hours: Specify whether happy hours are applicable or not.

Defined For: Identify the customer/s or groups for whom the schemes were defined.

Customer Code: Enter the customer code if the schemes you want to view/ manage were defined for specific customer.

Active/In-Active: Specify whether the definitions are active or not.

Show: To display the definitions that satisfy the selection criteria specified

All definitions that satisfy the criteria mentioned will be displayed in the grid. If you ave selected any operation to modify the existing definitions, you can make changes appropriately in the grid.

Clear: To cancel all the selections made

Update Changes: To save the changes made
