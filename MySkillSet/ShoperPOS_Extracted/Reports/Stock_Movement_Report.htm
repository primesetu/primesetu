Stock Movement Report

The Stock Movement report provides information on opening stock, goods received, goods returned to vendors, goods sold, sales returns, closing stock and closing stock value. A detailed report or a summarised one for all the items in stock, or for only those items transacted during the selected period, can be generated. The report may be sorted based on product or brand, as required.

How to reach: Reports > Stock > Movement

The Stock Movement window is displayed.

Product:  Select All or Specific. If you have chosen Specific, select the product from the list.

Brand:  Select All or Specific. If you have chosen Specific, select the brand from the list.

Select Only Transacted Items or All Items to generate the report of transacted items alone or of all items.

Select Only Consignment Items or Exclude Consignment Items to generate the report for consignment items alone or for all items other than consignment items.

Select Detail or Summary. Detail report gives information about each class1, class2, subclass1, subclass2 and size combination or stock number wise grouped as per your selection. Select Summary to view only the summary of transactions product/ brand wise or style/ shade wise.

If you have selected a detailed report, select Stock Number Wise or Style / Shade / Size to list the details accordingly. If you have selected a summary report select Product / Brand or Style / Shade to list the values accordingly.

Date Range: By default, the From and To dates are the current Shoper date. You can enter any date range but the To date cannot be greater than the Shoper date.

Display Negative Balances: Select to display the items with negative stock quantity.

Display Zero Balances: Select to display the items with zero stock quantity.

Primary Sorting Order: Select Product or Brand to sort the report accordingly.

Value Based On: Select Cost Price or Selling Price to display cost prices or selling prices in the report.

Note: The Cost Price can be hidden by the administrator based on user-wise restriction.

 

View: To view the report

Print: To make a hard copy of the report

Exit: To close the window
