HSN Mapping in Tally Interface

HSN Code is used to identify a commodity uniquely across countries. This code is used in India to define VAT rate for a commodity. HSN codes are not usually predefined in any application as all codes are never used in any business. To enable Shoper 9 to post VAT information to Tally 9, this option provides the facility to identify the HSN Code for each SKU.

How to reach: Setup > General > Tally Interface – HSN Mapping

The Tally Interface - HSN Mapping window is displayed. In this window you can map items to HSN codes or view the existing mappings.

New: Click to map items to HSN codes. The window has two tabs Item Details and HSN. On Item Details tab, select Based on item classification or Based on stock no, depending on which you want to assign HSN codes. Click the option name you have selected for more information.

View: Click to view the existing HSN code mapping. Click here for a sample of the View HSN Code window.

Click to close the window.
