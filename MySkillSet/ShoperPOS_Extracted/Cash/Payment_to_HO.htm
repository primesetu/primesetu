Payment to HO

The Franchisee has to pay cash and credit card sales proceeds to HO on a regular basis. Using Payment to HO option, the details of the payments made to HO can be recorded. This screen also shows the current outstanding amount of the franchisee.

How to reach: Cash > Franchisee A/C > Payment to HO

The franchisee payments window is displayed.

In this window you can execute the following functions.

Entry: Select to make a new payment entry to HO. For each entry, the Transaction Number will be different for that particular Transaction Date.

Cancel: Select to undo or cancel a previously recorded transaction.

Curr.Outstanding: Displays the current outstanding amount of the franchisee.

Transaction Account Type: Select the transaction type from the drop down list.

Transaction Number: Transaction number is auto generated and is incremented by 1 for each new entry.

Transaction Date: Displays the current Shoper date.

Transaction Mode: Select the mode of transaction from the drop down list. Depending on the selected transaction mode, the corresponding transaction details frame will appear on the right side of the screen.

If the transaction mode is selected as Cheque, then the cheque transaction details window is  displayed as shown.

Enter the transaction details like Bank Code, Bank Location, Cheque Date and Cheque No. for the payment made to HO.

If the transaction mode is selected as Cash, then the cash transaction details frame displays as shown.

Enter the transaction details like Transaction Currency, Currency Units and Currency Conversion Rate for the payment made to HO.

Transaction Amt: Enter the amount paid to HO or automatically fill the amount to be sent to HO based on configuration of the System parameter,  Prefill Balance in Payments to HO.

On clicking Ok, a window with a message for confirmation is displayed.

Ok: To save the entered details and proceed with the transaction.

Cancel: To close the window without saving the details entered.
