Stock Balance - Style / Model wise Report

The Style/ Model wise Stock Balance report provides stock information based on item sub-classification1 (Style/ Model). An important feature of this report is that it gives you the option to select items, based on the range of current stock quantity being all, equal to, less than, greater than or in a range of value/s set by you.

How to reach: Reports > Stock > Balance – Style/ Model wise

The Stock Balance -Style window is displayed.

Product:  Select All or Specific. If you have chosen Specific, select the product from the list.

Brand:  Select All or Specific. If you have chosen Specific, select the brand from the list.

Style: Select All or Specific. If you have chosen Specific, select the style from the list.

Breakup On Analysis Code: Select Yes or No. Accordingly report grouping will be introduced or not.

If you have selected Yes, based on the value selected from the list Fibre, Finish, Colour Base, Styling or Usage (item analysis codes) you may select All or Specific. If Specific is selected, select the corresponding value from the list.

Report Type: Select Report by Summary to generate a summary report or select Report by Detail to generate a detailed report. On selecting Report by Detail, the options Report Group By Select Quantity Range and Quantity Filter Criteria are activated.

Report Group By: Select StockNo, Style, Style/Shade, Style/Size or Style/Shade/Size to group the report accordingly.

Quantity Filter Criteria: Select Total Qty Wise to specify that the total quantity of  selected items should satisfy the specified criteria. Otherwise, select SKU Qty Wise to specify that the specified quantity criteria have to be satisfied by each SKU.

Select Quantity Range: Select All, Equal To, Lesser Than, Greater Than or Between to include items satisfying the specified stock quantity, in the report. If you have chosen Equal To, Lesser Than or Greater Than, enter the quantity in the field. If you have chosen Between, enter the lower and upper limit in the respective fields.

Code: Select to generate report with item classification code.

Description: Select to generate report with item classification description.

Report Criteria: select All Items, Items with Positive Bal or Items with Negative Bal to include the corresponding items in the report. If you have chosen All Items, select Skip Zero Balance to avoid including items with zero balance and/ or select Skip Negative Balance to avoid items with negative stock balances.

View: To view the report

Exit: To close the window
