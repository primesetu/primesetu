Adding Communication Details for HO

The HO Communications Details tab on the Configure Synchronisation window is as shown.

Head Office

Code: Enter the Head Office code.

Name: Enter the Head Office name.

Note: The Head Office code entered should be unique for each HO.

Shoper version 7.2 or lower: Select if the Shoper HO is Shoper 7.2 or lower version.

Only Base system: Select if Shoper HO installation is only the base system.

Mode: Choose the required mode of communication. The selected mode will be used for data transfer between the showroom and HO. Click the corresponding links to view the related fields to be filled.

Online

Server Name: Enter the Server Name and path in the following format Http://<Server name>#/<Virtual Dir> in this field.

Use SIS: Select to use SIS.

SSL: Select to permit connections only when the web server is running/ accepts connection in SSL (Secure Socket Layer).

User Name: Enter the user name.

Password: Enter the password.

 

E-mail

Head Office: Enter the E-mail Id of Shoper HO in this field.

Read Mails received after the last download: Select to read only the mails received after the last download.

 

FTP

Server Name: Enter the FTP Server Name in this field.

User Name: Enter the user name required for accessing the FTP Server.

Password: Enter the password required for accessing the FTP Server.

HO Upload (IN) Folder: Enter the name of the FTP folder to which this showroom has to send data.

HO Download (OUT) Folder: Enter the name of the FTP folder from which this showroom has to download data.

Passive: Select if your FTP server is passive.

SSL: Select to permit connections only when the FTP server is running/ accepts connection in SSL (Secure Socket Layer).

 

Direct Copy

HO Upload (IN) Folder:  Enter the name of the folder to which this showroom has to upload data. The files in this folder have to be sent to HO.

HO Download (OUT) Folder:  Enter the name of the folder from which this showroom has to download data. The files in this folder are sent from HO.

Files List: These fields are active in all the modes and hold the file type format specification used for upload and download.

To Upload: This field gets populated automatically.

To Download: This field gets populated automatically.

Upload Retry Count: This field is active only when Online or FTP is selected. Enter the number of retries to be attempted if there is a failure during the upload process.

Upload Retry Interval: This field is active only when Online or FTP is selected. Enter the interval duration in seconds between the retries.

File Split-size in KB: This field is active only when Online is selected. If you prefer to split the file, enter the size of the file in KB.

Communication mode is active: Select to make the mode of communication active.

Import from File: In case multi-brand outlet is enabled, this button is active. Click to import MBO Configuration from file. On clicking, file selection window opens. Select the configuration file. The values in the file are displayed in the HO Communication Details window. Click Apply.  The Item Classification Mapping window opens. Click here for more information.

Check Communication Mode: Click to test whether the system is able to communicate using the specified mode of communication (configuration and permissions).  Click here for more information.

Apply: To save the selection and entries

Clear: To clear all the values entered

Cancel: To cancel the selection
