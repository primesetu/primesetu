Edit Sales Promotions

This option is to modify the existing fixed or variable discounts. You can modify the items list, discount percentage or amount as well as the limit and applicability period of the promotions.

How to reach: Catalogue > Sales Promotions > Edit

The Sales Promotions Edit window is as shown.

Sales promotion code: Press any alphabet or number key, F2 or click to open the Browse window to select the required sales promotion.

Description: The description of the selected sales promotion is displayed. You may modify this, if required.

Modify operation:

Items: Select to modify the list of items under the sales promotion.

The Modify Items tab is as shown.

Add Items: Select to add more items to the list of items under the promotion. Enter the item details to be added to the list. Alternatively, press F2 to open the Browse window and select items.

Remove Items: Select to remove items from those existing in the promotion. The existing items are displayed. Select the required items.

Define Exclude List: Select to remove some items from the list of items under the promotion.

View: Select to view the list of items under the promotion.

Validate Items: Select to ensure that the items entered in the grid are existing ones.

Load: Click to load item details from a file. Click here to learn more.

Configure: Click to identify the attributes to be displayed in the grid. Click here to learn more.

The Item Selection Options are as shown.

Item Classification: The item classifications that are enabled in the system are displayed.

Code: Select to display the code of the corresponding item classification in the grid.

Desc.: Select to display the description of the corresponding item classification the grid.

You may select Code or Desc. or both.

Discount: Select to modify the discount percentage/ amount.

Item Level Discount Details: Displays the discount details of the selected sales promotion. Change the percentage or amount as required.

Applicability Period: Select to modify the applicability period of the promotion.

From: Displays the start date of the promotion. Change the date as required.

To: Displays the end date of the promotion. Change the date as required.

Selected sales promotion details:

Type: Displays the sales promotion type as defined.

Level: Displays the level at which the sales promotion is applicable.

Nature: Displays whether the sales promotion is fixed or variable.

Save: To save the modifications done

Clear: To clear the sales promotion details displayed

Exit: To close the window
