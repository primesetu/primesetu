Customer Mailer

This option is used to communicate with the customers through mailers and has provisions to perform the following functions :

  • Preparation of a list called distribution list comprising the customers to whom the mailer has to be sent whenever required

  • Compose the mail to be communicated

  • Print the mail to be sent by post

  • Send the mail electronically with a provision to additionally attach documents

How to reach: Catalogue > Customer Mailer

The CUSTOMER MAILER window is displayed.

Manage Distribution List

List ID: Displays the ID of all the available distribution lists

List Name: Displays the distribution list name

Creation Date: Displays the date of creation of the distribution list

Last Execution Dt: Displays the date on which the distribution list was last used

New: Click to add new distribution list. Click here for more information.

Edit: Click to edit the selected distribution list. Select Insert List to display the distribution list. Make necessary changes as explained in Add option. Click Ok to confirm modification.

Delete: Click to delete the selected distribution list.

View: To generate a report of the selected distribution list. On clicking View, the Condition Detail window is shown.

Exit: To quit the option
