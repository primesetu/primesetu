Other Details for Purchase Order/ Indent Generation

The Other Details tab is as shown.

Trailer Level Tax Details: Specify the tax details at document level.

Description: The Line level/ Item level values entered for Add-On, Deduction and Discounts are displayed in the respective columns. These values cannot be altered. The tax applicable at the document (PO) level can be entered here as a percentage or amount in these columns.

Per (%): Enter the tax rates wherever required. Calculated amounts will be displayed in the next column.

Amount: Enter the tax amounts wherever required. If you have entered the rate, the calculated values will be displayed here.

Note: If Tax/Duty are entered at item level, then entry is not allowed at Trailer level.

Shipment Types: Select the shipment type from the list.

Mode of Transport: Select the mode of transport from the list.

Document Remarks: Enter comments, if any.

Previous: To move to the Generation tab

Save: To save the purchase order

Clear: To undo the values entered
