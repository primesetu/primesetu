Item Classification

This option is used to classify a particular item, by specifying Product, Brand, Size Group, Preferred Vendors etc.  

How to reach: Catalogue > Item Classification

The Item Classification Catalogue window is displayed.

Initially the window is in browse mode, by default and you can move across the entries in the catalogue using the navigation buttons in the window. You can also toggle between record-by-record view and grid view using the Form and Grid buttons respectively.

Add: Click to record new Item Classification. Click here for more information.

Edit: Click to edit an existing Item Classification. Enter the Product and Brand code or press F2 and select from the list displayed. Make necessary changes as explained in Add option. Click Ok to confirm modification.

Delete: Click to delete an existing Item Classification, provided the same is not used in items master.

Form: Click to accept/ enter Item Classification

Grid: Click to display/ list the Item Classification Product and Brand. In case you want to edit/delete, select the required Product/ Brand combo and then click Edit or Delete button. The catalogued information will be displayed in the form.  

The navigation buttons help in viewing the records in sequence, in both, Form and Grid window. In Add mode these buttons are not activated.

Click the relevant button to display First, Previous, Next and the Last record.

Exit: To exit the Item Classification catalogue
