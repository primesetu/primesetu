Formats

The available formats are:

ASCII (Comma delimited), Excel (Spreadsheet), HTML (web-publishing), JPEG (Image), PDF (Portable Document Format), XML (data interchange). By default, XML (data interchange) is selected. Based on the format selection, the required fields appear on the export configurations sub-screen.

Other fields that appear based on the export format selected are:

Excel (Spreadsheet)

Output Sheet Name: By default, name of the report appears as sheet name. You may change this, if required.

Update existing file: Select Yes or No. If you select Yes, the existing file is updated with the content of the report.

Excel (Spreadsheet) Formatting: Select Yes or No. If you select Yes, you may choose to export the report with all colours as in Excel.

with Colour: Select Yes or No to export the report accordingly.

HTML (web-publishing)

Resolution: Select the required resolution from the Resolutions list. The available resolutions are: 1024 x 768 Recommended, 1280 x 1024, 640 x 480 Not Recommended and 800 x 600. By default, 1024 x 768 is selected.

JPEG (Image) & PDF (Portable Document Format)

Page Size: Select the required size from the Paper Sizes list. The available sizes are: User defined, A3, A4, A5, Legal and Letter.

Page Width, Height (in mms): The page width and height appear based on the page size selection.
