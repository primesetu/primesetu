New Customer Mailer

Clicking New displays the following window.

Manage Distribution List

List Name: Enter a name for the distribution list you are creating.

All Customers: Select to include all the customers into the distribution list.

Note: Once All Customers option is selected, the buttons Inset List, Delete List and Clear List are disabled. But you can either use the option Save List or Cancel.

Specific Customers: Select to include specific customers in the distribution list.

Insert List: Click to add filters so that only those customers satisfying the conditions set are included in the distribution list.

Filter Type: Select the required value from the list

Field Name: Select the field from the list on which you want to set the filter conditions.

Criteria: To set criteria click on  button. The window to select value is displayed as shown.

Join: Select And or Or from the list as required, click Insert List if you want to add more conditions.

Delete List: To remove the selected filter criteria  

Clear List: To clear all the filters

Save List: To save the distribution list

Cancel: To clear the entries made

Exit: To quit the option

Communication

Use this option to compose a Mailer which can be either printed or sent through e-mail.
