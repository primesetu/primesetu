Local License

You can activate or reactivate a license on a stand alone computer for the selected product.

How to reach: Start > Programs > Tally License Server > Tally Admin > Local License

The Local License window is as shown.

Product: Select the required product from the list.

License Path: The path from where you have launched the TallyAdmin tool is displayed. To activate the license on the specified path, enter the path or alternatively browse to select the required path.

Activate: To activate a Single Site or Multi Site license. Click here to learn about license activation process.

Re-Activate: To reactivate a Single Site or Multi Site license. Click here to learn about license reactivation process.

Proxy Configuration: Configure the proxy server to connect to the Internet (if required). Click here to learn more.
