Export

The export configurations sub-screen is as shown.

Yes: Select to export the report

No: Select to change export configurations

Language: Select Default (All Languages) or Restricted (ASCII Only). If you select Default (All Languages), the language set in Tally.ERP 9 is used. In the case of Shoper 9 reports, it is English.

Format: Select the required format from the Format list. Click here to learn more about formats.

Export Location: By default, the location appears as <Shoper 9 application folder\SupportCentre>. You may change this, if required.

Output File Name: By default, the report name appears as the output file name. Type of the output file varies based on the export format selection.

The applicable type of files (extensions) are:

Export Format

Change the file name, if required.
