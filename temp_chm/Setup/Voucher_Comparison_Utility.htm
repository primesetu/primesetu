Voucher Comparison Utility

This utility will enable you to compare the information between Shoper and Tally. The utility will generate a report displaying what should be the value in Tally for data posted from Shoper for a given period. You can compare vouchers in Tally for modifications.

With options to display voucher wise details and compare vouchers with Tally, this utility provides details of difference in transactional values of Shoper and Tally.

How to reach: Setup > General > Tally Interface Utilities > Voucher Comparison Utility

The Tally Interface Utilities – Tally Synchronisation Manager window is displayed.

 

The list of mappings created for the showroom is displayed. Select the mapping for which you need to generate a voucher comparison report.

Ok: To proceed with the change masters process

Exit: To abort the change masters process

The Voucher Comparison Utility window is displayed as shown.

From Date: Select the date from which report details are to be included.

To Date:  Select the date till which the report details are to be included. The default date selected is the current Shoper date. You can enter any date but it cannot be greater than the Shoper date.

Show Voucher-wise Details: Select to display voucher level details of each transaction in the report

Compare with Tally: Select to display detailed comparison of each voucher in Shoper and Tally. This option is available for selection only when the option Show Voucher Wise Details is selected.

Note: Tally.ERP 9 should be running and online to be able to compare vouchers with Tally.

 

Ok: To view the report

Exit: To exit the window
