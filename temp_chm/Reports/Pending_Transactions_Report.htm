Pending Transactions Report

The Pending Transactions report provides information on all pending, billed or cancelled transactions of suspended bills, sales advice slips, sales order slips, and so on, for the selected period. You can generate a detailed report or a summary report.

How to reach: Reports > MIS > Pending Transactions

The Pending Order(s) Report window is displayed.

Report For: Lists all types of pending transactions that are handled by Shoper 9. Select All or Selected to identify the transactions to be included in the report. If you have chosen Selected, select the required transactions from the list.

Select Date: By default, the Start and End are the current Shoper date. You can enter any date range but the End date cannot be greater than the Shoper date.

Report Option: Select Detail or Summary to view the corresponding report. Summary report shows the over all pending amount of the selected transaction type. Detailed report gives all information about each transaction like doc no, doc prefix, etc. Select Pending, Billed or Cancelled, or any combination of these to include the corresponding records in the report.

Ok: To view the report

Exit: To close the window
