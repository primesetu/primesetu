Packing Slip — Add

The Packing Slip window is as shown.

Packing Slip No.: Select the document prefix. The document number starts as defined in Bill Prefix for each series and gets incremented automatically.

Note: You have to define the prefixes in Bill Prefix option and carton details in Cartons Specification.

Carton Type: Select the type of carton.

Carton Capacity: Displays the capacity of the carton entered during carton type creation.

Date: Displays Shoper date.

Pack For: Select Customer, Showroom, Vendor, Sales Order or Dispatch Advice from the list. The Packing Slip is generated based on these selections.

Note: You can generate Packing Slip by selecting None which can be consumed for any Customer, Showroom or Vendor. This has to be configured in System Parameter.

Configuration: Click to display Column Selection and Quantity Entry screen.  

Column Selection: Select from the list of attributes which will form the grid header.

Scan (LSQ): Select to accept the quantity based on LSQ.

Manual: Select to enter the quantity manually.

Apply: To save the selection

Cancel: To cancel the selection

Load PDT: Select to load SKU from PDT (text file) and populate the Item Details grid.

Dimension: Displays the dimension of the carton entered during carton type creation.

Weight: Enter weight in Kilograms.

Remarks: Enter comments, if any.

Total Quantity: Displays the total quantity of items selected.

Ok: To accept the entries and print the Packing Slip as per System Parameter configuration

Cancel: To quit the entries

 

Note: You can print Packing Slip with a barcode (containing the details of Transaction prefix and Packing Slip number)as per System Parameter configuration.
