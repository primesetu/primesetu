Transport Receipt Entry

This option allows you to record the details of the transporter for an invoice.

How to reach: Sales > Transport Receipt Entry

The Transporter Details Update window is displayed.

Document Details

Transaction Type: Select Bill or Return as the transaction type.

Prefix: Enter the document prefix.

Number: Enter the document number.

Date: The invoice date of the specified document is displayed.  

Item Details: Details like Stock No, Description, Quantity and Rate of the items in the specified document are displayed in this grid.

Document Net Values: The net values of Add-Ons, Deductions, Discounts, Tax, Quantity, and total invoice value of the specified document are displayed in the respective fields.

Transporter Details

Mode of Transport: Select the mode of transportation of the items, from the list. The list contains the transportation modes that are catalogued.

LR Number: Enter the LR number for the items.

Transporter Name: Select the transporter name, from the list. The list contains the transporter names that are catalogued.

LR Date: Select the date on which the LR is prepared.

Save: To accept the transporter details entered

Exit: To exit the Transporter Details Update window without saving the details entered
