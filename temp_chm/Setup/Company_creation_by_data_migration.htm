The Data File and Log File path will depend on the destination path provided during installation. Shoper Files

In Dir: The In Dir path will depend on the destination path provided during installation. (I.e. if the destination path provided during installation is D:\Shoper9 then the In Dir path will be D:\Shoper9\In). You can browse and change the path, if required.

Out Dir: The Out Dir path will depend on the destination path provided during installation. (I.e. if the destination path provided during installation is D:\Shoper9 then the Out Dir path will be D:\Shoper9\Out). You can browse and change the path, if required.

Company Creation by Data Migration

On clicking Next, the Migration Options selection window is displayed as shown.

From Database: Select this option to migrate an existing database.

Select Database: Select the database to be migrated.

From Backup: Select this option to migrate backup taken from earlier version of Shoper.

Select Backup: Click to browse and select the backup (zip file) to be migrated.

Note: Ensure that the backup (zip file) contains the security database (Vasecdb file).

Shoper 9 migration option supports migration only from Shoper 7.2 Release 2.85 onwards.

Cancel: To exit company configuration window.

Database Name: By default the database name is Shoper9xxx, where xxx is the company code. The database name can be changed, if required.

Advanced: To confirm/ change the details like Server Name, User ID, Password, Data File path, Log File path and Shoper files details to be stored.
