Cash Transaction Report

The Cash Transaction report provides information on all types of cash transactions for the selected period. It can be generated date-wise, tender-wise, transaction-wise or cashier-wise. All types of billing transactions such as sales, sales returns, receipts, pay outs, and void sales are considered in this report. You may generate the report for any one type of transaction too.

How to reach: Reports > Cash > Cash Transaction

The Cash Transaction Summary - Report window is displayed.

Report Option: Select Date, Tender, Transaction or Cashier to group the report accordingly.

By default, the From Date and To Date are the current Shoper date. You can enter any date range but the To Date cannot be greater than the Shoper date.

Tender: Select Specific or All Tenders. If you have selected Specific, select the tender form the list.

Transaction Type: Select Specific or All. If you have selected Specific, select the transaction type form the list.

Cashier: Select Specific or All. If you have selected Specific, select the cashier form the list.

View in Spreadsheet format: Select to view the report in a spreadsheet format.

View: To view the report

Exit: To close the window
