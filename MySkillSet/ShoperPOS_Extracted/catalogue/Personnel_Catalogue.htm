Personnel Catalogue

This option is used to catalogue details of personnel such as employee code, name, mobile number, e-mail id, address, etc.

How to reach: Catalogue > Personnel > Personnel Catalogue

The Personnel Catalogue window is displayed. In this window you can execute the following functions.

Add: Click to  to add personnel details. Click here for more information.

Edit: Click this button and enter the employee code to recall an existing record. Alternatively, you can press F2 to open the Browse window and select the code. You can modify all the existing details, except the code. Follow the steps as in Add to change the values.

View: Click this button and enter the employee code to recall an existing record. Alternatively, you can press F2 to open the Browse window and select the code.

Exit: To quit this option
