─── 1. PRODUCT IDENTITY ────────────────────────────────────────
Product Name     : Smriti OS
Category         : Retail ERP Software
Version          : 3.0 (May 2026)
Inspired By      : Shoper 9 (Tally's Retail Management Software)
Tagline          : "Memory, Not Code."
Target Market    : Single & Multi-store Retail Businesses (India)

─── 2. PRODUCT VISION ──────────────────────────────────────────
Deliver a retail ERP faster, more flexible, and more modern than
Shoper 9 — preserving the familiar Shoper 9 workflow that retail
businesses already know and trust.

─── 3. DEPLOYMENT MODES ────────────────────────────────────────
Mode          | Description                          | Target Customer
White Label   | Smriti OS runs ON TOP of Shoper 9   | Existing S9 license holders
Standalone    | Fully independent ERP (S9-inspired)  | New customers — no S9 needed

─── 4. CORE PRODUCT AREAS ──────────────────────────────────────

─── 6. ROLES ───────────────────────────────────────────────────
Role             | Level        | Description
Super Admin      | System       | Full access — unrestricted
HO Admin         | Head Office  | Full HO — manages all stores
HO Manager       | Head Office  | View/reports only
Store Manager    | Store        | Full retail ops
Cashier          | Store        | POS billing + returns only
Stock Manager    | Store        | Inventory — no billing
Purchase Manager | Store        | Purchase/GRN only
Accountant       | Store        | Reports/day-end — no transactions
Custom Role      | Any          | User-defined flexible permissions

─── 7. ACCESS LEVEL DEFINITIONS ────────────────────────────────
No Access   → Menu hidden completely
View Only   → No add/edit/delete buttons
Read-Write  → No delete button
Full Access → All buttons + export visible
