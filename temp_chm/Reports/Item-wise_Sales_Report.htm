Item-wise Sales Report

The Item-wise Sales report provides sales information for the selected period, based on item information. It lists the items that were sold or returned during the period selected along with their sales, return, net quantity and values. You can generate either detailed or summary report Item-wise or Style-wise (subclass1).

How to reach: Reports > Sales > Item-wise Sales

The Sales Report window is displayed.

Select Item-Wise Sales Report or Style - Wise Sales Report.

Product: The first field of basic classification of the item attribute (here it is named as Product) is shown. Select All or Specific. If you have chosen Specific, select the required values from the list.

Brand: The second field of basic classification of the item attributes (here it is name as Brand) is shown. Select All or Specific. If you have chosen Specific, select the required values from the list.

Date Range: By default, the From date and To date will be the current Shoper Date. Select any date range for which you want to generate the report. However, the To date cannot be greater than the Shoper Date.

Type Of Report: Select the type of report to be generated. It may be Summary report or Detail report. In both the cases, the report can be generated based on any of the following options:

  • Based on Code: If you want to view only Item code in report

  • Based on Desc: If you want to view only Item description in report

  • Both (Code & Desc): If you want to view both Item code and description in report

Items: Select All Items, Only Consignment Items or Exclude Consignment Items.

Primary Sorting: Sorting Order of the report can be specified either on Product (Class1) or Brand (Class2).

Size Group: Select All or Specific.

Breakup: Select Vendor Wise, if you want to generate report for a Specific vendor or for All vendors. If All option is selected, then the sales information will be provided vendor wise for all vendors. And if Specific option is selected, then the vendor list box will be activated and you can select the required vendor code for report generation.

View: To view the report

Print: To print a hard copy of the report

Exit: To exit the window
