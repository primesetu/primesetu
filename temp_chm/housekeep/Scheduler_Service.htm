Scheduler Service

Once you install Shoper 9 POS, on the server, the menu item Start > Programs > Shoper 9 > Scheduler Service Starter is created and the service is added to Windows Startup group.

To start the service on the server for the first time, choose Start > Programs > Shoper 9 > Scheduler Service Starter.

When the service is running, the Shoper Agent POS icon will be displayed in the Notification Area (System Tray). Right clicking this icon will display a pop-up menu. This menu displays three options for the service -  Start, Stop and Exit.

Start: Choose to start the service. In case the service is running and is in a Stopped state, you may use this option to change the state of the service to Started.

Stop: Choose to stop the service. The Shoper Service window is displayed. Here you need to log in using the super user ID and password, as only a super user can stop the service. This will change the state of the service to Stopped.

Exit: Choose to exit  the service. The Shoper Service window is displayed. Here you need to log in using the super user ID and password, as only a super user can exit the service. This will close the service.

Note: When a scheduled job is being executed, you cannot use any of these options.

Once the service is closed using Exit, you need to choose the menu option Scheduler Service Starter to run the service again. However, on restarting the machine, the service will start running.

In case the service is in the Stopped state, and the machine is restarted, the service will continue to be in the Stopped state.

Note: Only one instance of the service can be running for a Shoper 9 POS installation.
