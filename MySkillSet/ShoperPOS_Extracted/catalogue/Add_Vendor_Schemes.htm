Add Vendor Schemes

The Schemes Definition appears as shown.

 

Scheme Code: Enter/type the ID for the scheme.

Description: Enter/type the description of the scheme.

Start Date: Select/enter the date of opening the scheme.

End Date: Select/enter the date of closing the scheme.

Scheme On: The scheme can be defined based on sale or purchase of items.

For e.g.: A manufacturer/distributor can define a scheme which entails the retailer to get additional incentives (commission/free quantity/points redeemable) based on the sales of item(s).  

Similarly, a scheme based on purchase will entail the retailer to get incentives provided a certain amount/quantity of purchases are made by it.

  • Select Sale/Purchase as required

Scheme Criteria

Based on: The basis of the schemes to be defined.

Quantity: Select if the primary condition is quantity of stock items sold/purchased by the retailer to qualify

Value: Select if the primary condition is value of stock items sold/purchased by the retailer to qualify

None: Select if there is no condition set in the scheme.

In Terms of: Select the required incentive that will be paid/given to the retailer.

This depends on the primary selection based on Quantity, Value or None.

  • Quantity & None: Flat Amount/Flat Qty/Amount per pc/Percentage Value/In Points

  • Value: Flat Amount/Flat Qty/Percentage Qty/In Points

Scheme Applicability:

Common for all: Select to apply the scheme if it is common for all retailers.

Vendor/Customer Specific: Select to apply the scheme is for a particular vendor/customer.

Points Slab: This option is used when the incentive is points earned for either sale/purchase of items.

  • Click this button to display the Points Slab Details window

   

  • Import Point Slab from scheme: Enter any saved/existing scheme to import the details in the grid.

Configure: Click this button to configure the fields for display in the grid.

  

  • Select/Deselect the fields.

Apply: Click to accept the selection/de-selection.

Ignore: Click to close the window.

The Grid displays the scheme details configured. It may be a simple scheme or a combination of two or more schemes. The connectors of schemes are And & End conditions.

   

OK: Click this button to accept the defined scheme.

Cancel: Click this button to stop/terminate the definition.
