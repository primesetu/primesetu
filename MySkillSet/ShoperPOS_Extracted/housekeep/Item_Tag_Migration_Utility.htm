Item Tag Migration Utility

This is a menu to migrate the existing stock (without the serial number configuration) to the serial number configuration.

In POS/Distributor environment the process happens in two ways.

  1. When serial number configuration under Catalogue is defined. (Please refer Serial No. Configuration page for more information)

OR

  1. By the manual migration process under Housekeeping as described below.

How to reach: Housekeeping > Item Tag Migration Utility

 

The White Goods Migration window is as shown.

 

Migrate: Click this button to migrate the stock items to the serial number configuration.

End: Click this button to stop/terminate the process.

A message box is displayed after clicking the Migrate button.

 

OK: Click to close the message window.

In a chain stores environment with Head Office (HO):

If the Head Office (HO) configures the serial number information of the stock items and sends to its chain stores;

  • The serial number configuration is stored in a temporary table and is applied and migrated during the OPEN Day/ day begin process. It is essential that the stock quantity should exist in this case.

  • If the stock balance is zero, the serial number configuration gets applied during synchronisation. However, the manual migration has to be done after synchronisation.
