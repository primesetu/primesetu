Printing Templates

This option is used to select the printing formats related to Sales (Billing) and Goods Outwards.

How to reach: Setup > General > Printing Templates

Default Printing Formats available

Sl.No.

Code

Description

1

Shoper40ColDS

40 Columns Double Spread - Two copies of the same bill, each of 40 columns width is printed in compressed mode on the pre-printed stationery with perforation between each copy. Page throw happens depending on the number of lines printed under headers, items block and footers. Hence the length of the bill is limited.

2

Shoper40ColPOS

40 Columns POS - If Shoper 9 is installed on a POS machine then the bill will be printed on a continuous stationery which is 40 column wide. The item details are printed in between the header and footer.

3

Shoper40ColSS

40 Columns Single Spread – This is similar to 40 columns double spread with only one copy of the bill being printed. Generally, the stationery will have two parts with carbon sheet embedded. The carbon copy is used by the retailer for future use.

4

Shoper80ColRetail

80 Columns Retail - This covers full width of 80 Columns of the paper used with Dot Matrix Printer. The font size of the characters will be bigger than those printed in the 40 columns format since the characters are not compressed. Due to availability of space, more characters can be printed for each field. Additionally, Bill Caption, like Cash Bill, Return Bill; Customer Name, Address etc., can also be printed.

5

ShoperDistributor

ShoperDistributor – Using this template the bill of 80 columns width is printed in compressed mode. It provides a detailed distributor invoice in columnar format.

6

ShoperInkJet

ShoperInkJet - This is applicable when you opt to print the bill on an Inkjet or Laser printer. The format is set for printing on A4 size paper.

 

The Printing Template Selection window is displayed.

 

 

Printing Template Selection

Template Folder Path: Displays the default path of the templates related to the each of the formats mentioned.  This cannot be modified.

Transaction Type: Select any transaction type, i.e., Sales, Inwards or Outwards.

The grid next to Transaction Type displays the allowed transactions for the selected type. Select the billing format required for each transaction by ticking the check box available on left side.

Transaction Type

Available Transactions

Sales

Sales, Sales Return, Void Sales

Inwards

Purchase, Transfer In, Approval Return, Miscellaneous Receipt

Outwards

Purchase Return, Transfer Out, Approval Issue, Miscellaneous Issue

 

You may also select an alternate template for conditionally printing the document under each of the available transactions by clicking Conditional Template to display the window as shown:

 

Select a required format (obviously, other than the one selected earlier) and provide condition under which it has to be used for printing. (e.g., Shoper80ColRetail)

Field ID: Select the required field from the list. (e.g., Customer Price Group Code)

Operator: Select the appropriate operator from the list (e.g., =)

Conditional Value: Enter one of the possible values for the selected Field ID (e.g., CPG)

Done: To accept the entries

Cancel: To quit the option

 

Save: To save the selected printing format for all transactions with respect to one transaction type

Help: To display Help window

Exit: To quit the option
