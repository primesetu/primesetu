Load from PT File

When the distribution center despatches goods to its POS location, it also sends a Purchase Transaction file electronically which contains item details, Doc.Ref.Number, Doc Date and Supplier ID/ Chain store code.

This PT file is created during goods outward activity from the distribution center to the POS location. Since the items classifications and other attributes are common for all the chains, this file is useful during goods inward at the POS location.

Note:

1. The PT file should be available in the Shoper In directory.

2. Shoper 9 does not allow to load a file more than once.
