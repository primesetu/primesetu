Edit Users

The window for editing existing users is displayed.

User ID: Displays the existing User Id’s. Cannot be edited.

User Name: Displays the existing user names. Change if required.

Password: If you want to change the password, enter a new password.

Confirm Password: Enter the password again to confirm.

Admin User: Select to provide administrator rights (Full rights) to this user.

Group: Select to change the group assigned to the user.

Group Id: Select to assign the user to group.

Group Name: Displays the existing group name.

Add: To add new group(s)

Ok: To confirm selected group(s)

Cancel: To cancel the selection

Company: Select to change the company assigned to the user.

Comp Code: Select to assign user to the company

Applicable for All Companies: Select to assign user to all the companies created and also the companies created in future.

Ok: To confirm selected company

Cancel: To cancel the selection

Deactivate User: Select to deactivate the user.

Account Expiry Date:Displays the expiry date of the user login. Change if required.

Country Code: Displays the Country code. Change if required.

Mobile No.: Displays the mobile number. Change if required.

E-mail ID: Displays the e-mail ID. Change if required.

Ok: To confirm the editing of the user(s)

Cancel: To abort editing of the user(s)

Exit: To close the Security Management window
