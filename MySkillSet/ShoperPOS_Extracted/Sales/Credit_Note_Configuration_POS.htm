Credit Note Configuration

In Shoper, credit notes are generated to handle the amount payable to the customer when there is a sales return or in cases of exchange where there is a balance amount due to the customer. Balance value comes into picture:

  • While exchanging a bill, if the Exchange value is more than the Net amount

  • While redeeming a credit note, if the value of a credit note is not completely utilised

The parameters explained below can be set to facilitate effective handling of balance value. In the case of chain stores, HO decides the template for billing functionality and hence some of the parameters are being blocked from modification. This ensures consistency of billing functions across POS locations.
