Applying Promo Details in Manual Mode

In this window you can view/ apply the promo/s that can be applied to the current bill. The window, displayed below the item grid, lists the active promos and the application of the promo on the selected items.

The highest priority promo will be applied by default. You may change this and select another promo or avoid giving any promos.

If it is an offer and if the number of items scanned exactly matches the sum of buy quantity and get free quantity, it would be chosen. In case the number of items scanned does not exactly match the sum of buy quantity and get free quantity but satisfies the buy quantity, an alert message is displayed. Click here to learn more.

In manual mode, you can view the details of promo/s chosen and change it, if required.

Code and description of all active promos are displayed on the left hand side of the window. The promo/s applied in the current bill is marked with a tick mark. The items on which selected promo/s are applied, along with their description and quantity, and details of discount/ offer applied are listed on the right hand side. To view details of application of the selected promo/s in the current bill, select the promo name in the list on the left hand side, by clicking the row.

Note: The details of Last Piece Discount, if any, will not be displayed in this window.

You may deselect the chosen promo and select another/ none, if required. All active promos that can be applied are enabled to select.

Reset to Default Promos: To cancel all the selections you made and revert to the default suggestion given by the system.

Apply Sales Promo: To apply the selected promo/s.

Cancel Sales Promo: To close the sales promo selection window and go back to item-scanning mode.

Press F6 (while the Item Level Promotional Details tab is displayed) to display Bill Level Promotional Details tab. This is editable.

Note: Press F6 to toggle between Item Level Promotional Details tab and Bill Level Promotional Details tab.

 

Price-Offs: Price factors, if any, applied at the time of billing is displayed here.

Bill Net Value: Total bill amount after applying item level promos, if any.

Discount Code: The discount code selected by default and description is displayed to the right of this field.

Calculated on: Bill Net Value is displayed by default as the value on which discount is calculated. Change the value, if desired.

Max. Allowed: Maximum discount allowed as per the selected promo is displayed.

Discount %: Displays the percentage of discount applicable as per the selected discount code/ scheme, subjected to the maximum allowed (if defined). If the item level discount selected is a fixed one you cannot edit it. On the other hand, in the case of a variable discount, if it is defined as:

  • Percentage: You may change the percentage of discount for the selected item

  • Amount: The computed percentage based on the discount amount is displayed

Discount Amt.: Displays the discount amount applicable as per the selected discount code/ scheme, subjected to the maximum allowed (if defined). If the item level discount selected is a fixed one you cannot edit it. On the other hand, in the case of a variable discount, if it is defined as:

  • Amount: You may change the amount of discount for the selected item

  • Percentage: The computed amount based on the discount percentage is displayed

Additionally, if you have defined the variable discount as Both, you may enter either percentage or amount. If you enter percentage, the corresponding amount will be displayed and vice versa.

Reason: Select/ enter a reason for discount as per the configuration. The reasons listed are Festival Discount, Stock Clearance Discount and Generally Allowed Discount.

Remarks: Enter any additional remarks, if required.

Apply Sales Promo: To apply the selected Bill Level promo selected

Cancel Sales Promo: To cancel the selected Bill Level promo

 

Note: You can prioritise the calculation of item level and bill level discounts, if configured. Click here for more information.
