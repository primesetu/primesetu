Upload

The upload configurations sub-screen is as shown.

Upload to: Select FTP Site or Web Page (HTTP/HTTPS) from the Upload to list.

URL: Enter the server URL.

Secure Server: Select Yes or No.

User Name (if any): Enter the user name, if any, for upload.

Password (if any): Enter the password, if any.

Format: Select the required upload format. Default format is XML (Data Interchange). Click here to learn more about the formats.

In Upload to field, if you select FTP Site, other fields relevant for the chosen format appear.

Destination File: The report name appears as the file name, by default. File type (extension) depends on the format selected.

Some additional fields may appear based on the export format selected. Click here to learn more.
