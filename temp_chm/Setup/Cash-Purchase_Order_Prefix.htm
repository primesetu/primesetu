Cash / Purchase Order Prefix

This option is used to define the Doc Prefixes and Doc number for all the Cash transactions and Purchase Order transactions. The Cash transactions include Cash Receipts, Void Cash Payouts, Cash Payouts and Void Cash Receipts. The Purchase Order (PO) transactions include PO, Pre approved PO, Indent, and Pre approved Indent.

How to reach: Setup > General > Cash/ Purchase Order Prefix

The Cash / Purchase Order Prefix window is displayed.

Select Purchase Order from the drop down list.

Current Year Suffix: Displays the Suffix used in the current year. This cannot be altered.

Grid

Trn Name: Displays the transaction name. This field cannot be altered.

Trn Type: Displays the transaction Id associated with transaction. This field cannot be altered.

Doc Prefix: Displays the Document Prefix defined in the Prefix definition. The document prefix can be altered.

Doc Number: Displays the last documented number. The document number can be altered.

Save: To consider the changes made

Exit: To quit the option
