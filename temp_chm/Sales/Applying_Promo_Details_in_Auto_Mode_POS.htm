Applying Promo Details in Auto Mode

In this window you can view the promo/s that can be applied to the current bill. The window, displayed below the item grid, lists the active promos and the application of the promo on the selected items.

The highest priority promo will be applied by default. You cannot change this and select another promo.

If it is an offer and if the number of items scanned exactly matches the sum of buy quantity and get free quantity, it would be applied. In case the number of items scanned does not exactly match the sum of buy quantity and get free quantity but satisfies the buy quantity, an alert message is displayed. This message varies based on the setting whether the selection of free items are mandatory or optional.

In auto mode, you can only view the details of promo/s applied. This is non-editable.

Code and description of all active promos are displayed on the left hand side of the window. The promo/s applied in the current bill is marked with a tick mark. The items on which selected promo/s are applied, along with their description and quantity, and details of discount/ offer applied are listed on the right hand side. To view details of application of the selected promo/s in the current bill, select the promo name in the list on the left hand side, by clicking the row.

Note: The details of Last Piece Discount, if any, will not be displayed in this window.

Reset to Default Promos, Apply Sales Promo and Cancel Sales Promo are disabled as no user interaction is possible.

Press F6 (while the Item Level Promotional Details tab is displayed) to display Bill Level Promotional Details tab.

Note: Press F6 to toggle between Item Level Promotional Details tab and Bill Level Promotional Details tab.

In auto mode, the tab is as shown. You may enter a reason and remarks.

Price-Offs: Price factors, if any, applied at the time of billing is displayed here.

Bill Net Value: Total bill amount after applying item level promos, if any.

Discount Code: Displays the discount code selected by default and description is displayed to the right of this field.

Calculated on: Bill Net Value is displayed by default as the value on which discount is calculated.

Max. Allowed: Maximum discount allowed as per the selected promo is displayed.

Discount %: Percentage of bill level discount applied as per the selected scheme is displayed.

Discount Amt.: Bill level discount amount applied as per the selected scheme is displayed.

Reason: Select/ enter a reason for discount as per the configuration. The reasons listed are Festival Discount, Stock Clearance Discount and Generally Allowed Discount.

Document Remarks: Enter comments, if any, about the document.

Apply Sales Promo and Cancel Sales Promo are disabled as no user interaction is possible.

The parameter Apply Bill Level Discount First under Billing gives you the flexibility to prioritise item level and bill level discounts as per your requirement.

Note: You can prioritise the calculation of item level and bill level discounts, if configured. Click here for more information.
