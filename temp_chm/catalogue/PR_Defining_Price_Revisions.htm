Defining Price Revisions

The Price Revision Catalogue window is as shown.

Filter Criteria

The items for which prices have to be revised can be selected based on Stock Number or the basic item classifications namely Product, Brand, Style, Shade, Size or a combination of any of these item classifications. Check the box to the left of the field name and click All to select all values under the field or click Range and select the required values from the list. After selecting the Stock Number if you select any item classification, the stock number entered will be cleared and vice versa.

List matching items: To apply the criteria and display the items those satisfy the criteria, in the grid.

Note: Alternatively, you may choose the item details from a PDT file.

Price Revision Details

Effective From: Double click to display a calendar and choose the date from when the new price revision has to be effective.

Doc No.: System generated serial number for the price revision document.

Retail Price: Click the box to modify the retail price. The Revised column under Retail Price in the grid is enabled for editing. Select the price change criteria as one among Up by, Down by and Fixed from the list on the right. From the next list select either Percentage or Amount by which the retail price has to be changed and accordingly enter the percentage or amount by which the price has to be changed in the next field. Notice that if Fixed has been selected by you, then the next list is prefilled with Amount (i.e. Percentage is not allowed) and retail price will be replaced by the amount specified in the next box.

Dealer Price: Click the box to modify the dealer price. The Revised column under Dealer Price in the grid is enabled for editing. Select the price change criteria as one among Up by, Down by and Fixed from the list on the right. From the next list select either Percentage or Amount by which the dealer price has to be changed and accordingly enter the percentage or amount by which the price has to be changed in the next field. Notice that if Fixed has been selected by you, then the next list is prefilled with Amount (i.e. Percentage is not allowed) and dealer price will be replaced by the amount specified in the next box.

Load From PDT File: Click to load item details from a PDT file. Click here to learn how to load the details.

Apply: Click to compute the revised prices and display the values in the grid.

The revised retail and/or dealer prices as per the criteria you have specified is displayed in the grid. If you want to make further changes to prices of certain items, you may do that in the grid.

Clear: To undo all selections made under Filter Criteria and specify new filters. If you click Clear at any stage before clicking Save, the filter conditions as well as the records displayed in the grid get cleared, paving way for the preparation of a new price revision document.

Save: To save the values entered or changed.
