Bill-wise Items Report

The Bill-wise Item List provides sales information of all the items included in each bill, for the selected period. It lists all the bills (including void and return bills) along with the item details. The report can either be grouped on bill date or on customer. In the case of report grouped by bill date, it is further grouped according to transaction type and bill number. In the case of report grouped by customer, it is further grouped according to bill date, transaction type and bill number. Group totals are displayed for all numeric values selected for display.

How to reach: Reports > Sales > Bill-wise Items

The Bill Wise Item List window is displayed.

Date Range: By default, the from date and to date will be the current Shoper date. You can select any date range but the to date cannot be greater than the Shoper date.

Select Salesman: Select All or Specific. If you have selected Specific, select the salesman name from the list.

Column Selection: Select the columns to be displayed in the report. Some of the columns are selected by default. You may unselect these, if required. However, the columns Sales Promo Code, Stock No., Qty, Value and Doc Rate cannot be unselected. Use and to change the order of display of fields in the report.

Report option: Select Natural Order or Sort By Promo Code. If Natural Order is selected, items are listed in the order in which it is entered in the bill. Sort By Promo Code sorts and groups the item list according to the sales promos applied.

Customer Selection: Select All or Specific. If you have selected Specific, select the customer name from the list.

Select All Bills or Consider Bills With Sales Promo to choose the type of bills. If you have selected Consider Bills With Sales Promo, the list of sales promos will be enabled to choose from.

Sales Promo Selection: By default all sales promos will be selected. Select the promos to identify the bills with those promos applied and are to be included in the report.

Clear All/ Select All: Click to unselect/ select all the sales promos. If you have chosen Clear All, then choose the required ones. If you have chosen Select All, then unselect the required ones.

Bill Type: Select the types of bills Cash, Credit or Both to be included in the report.

Group By: Select Bill or Customer based on which the report needs to be grouped.

View: To view the report

Clear: To cancel the selections done

Exit: To close the window
