Head Office – Franchisee Details

This tab displays the details about the Franchisee and its HO.

HO Code: Enter the code of the HO of the franchisee.

HO Name: Enter the name of the HO of the franchisee.

Franchisee Code: Enter the franchisee account code.

Franchisee Name: Enter the name of the franchisee.

HO Code - Franchisee: Enter the identification code given by the HO to the franchisee.

Agency Cheque Received in HO's Name for Credit Card sales: Select the check box, if required.

Payments to HO By Cash - For Cash Sales Only: Select the check box, if required.

Update: To update and save the changes made. You are prompted to provide the values for the mandatory fields before the information is saved into the database.

Exit: To exit from the Franchisee Details window.
