PDT/Import Extensions – Procedure

Procedure to Create PDT Extensions

There are three steps to create an extension supported by Extension Framework in Shoper 9 POS:

  • Creating Custom DLL

  • Deploying Custom DLL

  • Configuring Custom DLL details in GenLookUp table

Creating Custom DLL and deploying it at POS location

To know how to create and deploy Custom DLL, refer Appendix

Mandatory function to be present in a Custom DLL while creating a new option is:

Public Function PDTFile(StrExeName As String, StrConString As String, StrCmdLine As String, StrTrnType As String, StrFileFormat As String, StrDelim As String, StrExtraParam As String) As String

Recording details in General Lookup (GenLookUp)

The Rec Id (Record Id) 16000 is maintained in GenLookUp table for configuring the customised DLL.

There are seven records to be added in GenLookUp table. Each record consists of Recid (16000), Code and Description. These seven records capture seven properties of an extension.

The seven properties and their respective descriptions to be captured are:

Property Code

Description

Remarks

01

DllName.ClassName

Dll name and the Class name

02

Caption

Extension caption

03

N/Y

Whether the option is invisible or not

N -> Visible

Y -> Not visible (invisible)

04

Exe name

EXE name of the transaction where the extension is to be enabled

05

Int/Ext

Whether the extension is in internal functions (e.g., Purchase Order, PT file loading and PDT loading in Inwards) or in external functions.

New extensions are always external.

06

File format

  1. StockNo

  2. StockNo#ActQty

  3. StockNo#Rate#ActQty

  4. StockNo#DocQty#ActQty

  5. StockNo#Rate#DocQty#ActQty

For example, if the file consists of:

  • Stock number and Quantity, then the value is 2#;

  • Stock number, Rate and Quantity, then the value is 3#;

07

Parameter value/ NULL

Any parameter value to be passed to the custom DLL

In GenLookUp table, Code is inserted as a combination of Extension No. and Property Code separated by a ‘+’ symbol. For example, if the Extension No. is 1 and the Property Code is 01, then the corresponding Code is 1+01.

Alternatively, you can use Extension Script Generator Tool to generate the corresponding script which contains the SQL statements to be inserted in GenLookup table for configuring the required custom DLLs.
