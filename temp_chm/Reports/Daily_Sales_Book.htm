Daily Sales Book

The Daily Sales Book provides sales information on all items, consignment items or all items other than consignment items for selected products/ brands for a given period. It is grouped under Item Classifications, irrespective of the bill it is associated with.

How to reach: Reports > Sales > Daily Sales Book

The Daily Sales Book window is displayed.

Product Selection: The first field of basic classification of the item attribute (here it is named as Product) is shown. Select All or Specific. If you have chosen Specific, select the required values from the list.   

Brand Selection: The second field of basic classification of the item attributes (here it is name as Brand) is shown. Select All or Specific. If you have chosen Specific, select the required values from the list.

Date Range: By default, the start date and end date will be the current Shoper Date. Select any date range for which you want to generate the report. However, the end date cannot be greater than the Shoper Date.

Item Selection: Select All Items, Only Consignment Items or Exclude Consignment Items.

Primary Sorting: Sorting Order of the report can be specified either on Product (Class1) or Brand (Class2).

Report Breakup: Report can be grouped based on item attributes. It can either be first two levels of item attributes, which is a summary information (Product and Brand combination) or first three level of item attributes (Product, Brand and Quality combination), which will be a detailed report. Report can also be generated Date-wise.

View: To view the report

Exit: To exit the window

 

Note: The Bill Value does not include the round off values because the round off is done at bill level and this report is based on Item Level.
