Transaction Details with Image Report

Provides master information or transaction details along with the images of the items. This feature is useful in retails, especially dealing with jewellery, footwear, electronic goods, etc. where design, pattern, style, model are given importance.

How to reach: Reports > MIS > Transaction Details with Image

The Image Report Selection window is displayed.

Select Master or Transaction.

Selecting Master will generate the report displaying item master information. Here the Image Report Selection window has two tabs, Master(s) and Display Fields. Click the tab names to learn more.

Selecting Transaction will generate the report displaying transaction details. Then, the Image Report Selection window has two tabs, Transaction(s) and Display Fields. Click the tab names to learn more.

View: To display the Image Printing Report

Clear: To cancel the report criteria selected

Exit: To close the window
