Goods Outwards

Goods Outwards refers to transactions such as purchase return, inter-showroom transfers, and miscellaneous issues. You can use this option to record details of outward transactions.

Note: Extension Framework provides a certain degree of customisation in Goods Outwards.

 

How to reach: Stock > Goods Outwards

The Goods Outwards window is displayed. In this window you can execute the following functions, based on configuration.

Add: Click to do a new goods outwards transaction. Click here for more information.

Edit: Click to modify an existing goods outwards transaction details. Select Transaction Type, Doc. No. Prefix and Doc. No. to display the existing outward transaction document. Follow the steps as in the Add option to make necessary changes.

You can edit the documents only if the document date is same as Shoper date. When you edit a document, the existing document number is voided and a new document number with the changes is generated.

Delete: Click to delete an existing goods outwards transaction details. Select Transaction Type, Doc. No. Prefix and Doc. No. to display the existing outward transaction document. Verify the details in the selected document and click Ok.

You can edit the documents only if the document date is the same as Shoper date.

Reprint: Click to reprint an existing goods outwards transaction details. Select Transaction Type, Doc. No. Prefix and Doc. No. to display the existing  outward transaction document. Click Ok.

Exit: To close the window
