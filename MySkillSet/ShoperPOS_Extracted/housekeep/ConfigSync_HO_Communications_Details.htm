HO Communication Details

The HO Communications Details tab on the Configure Synchronisation window is as shown.

In this window you can execute the following functions.

Add: Click to add communication details for a different HO. Click here for more information.

Edit: Click to modify an existing communication details specified for an HO. Enter the Head Office code and the details are displayed. Make the necessary modifications and click Apply to save the changes made.  

Delete: Click to delete an existing communication details configuration. Enter the Head Office code and the details are displayed. Check the details and click Apply to confirm deletion.

Exit: To quit the Configure Synchronisation option
