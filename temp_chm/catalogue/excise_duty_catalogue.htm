Excise Duty Catalogue

Excise duty is an indirect tax on specific goods or commodities produced for sale within a country or on licenses granted for specific activities. This catalogue can be used to define excise duty details applicable for the merchandise in your store.

How to reach: Catalogue > Excise Duty

The Excise Duty Catalogue window is displayed. In this window you can execute the following functions.

Add: Click to create a new excise duty catalogue. Click here for more information.

Edit: Click to edit an existing excise duty catalogue. On clicking Edit, the Excise Duty Document Browse window appears. Select the catalogue to be edited. Make necessary changes and save.

Delete: Click to delete an existing excise duty catalogue. On clicking Delete, the Excise Duty Document Browse window appears. Select the catalogue to be deleted. Verify if the selected catalogue is to be deleted and confirm deletion by clicking Ok on the Excise Duty Catalogue window.

Exit: To quit the Excise Duty Catalogue
