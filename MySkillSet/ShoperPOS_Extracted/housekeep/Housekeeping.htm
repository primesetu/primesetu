Housekeeping

Housekeeping menu allows you to perform various activities. Some of these activities are conducted occasionally for the smooth functioning of Shoper 9 POS. Some other activities are performed to optimise database. In addition to this, housekeeping menu provides you with the necessary synchronisation activities in the case of chain stores.

Various options available under this menu are:

  • OPEN Day

  • CLOSE Day

  • Backup

  • Restore

  • Compact

  • Database Tuning Utility

  • Purge Data

  • Delete Temporary Tables

  • Delete Backup File

  • Item Tag Attachment Utility

  • Item Tag Migration Utility

  • Alerts Management

  • Communication Configuration

  • Schedule for Reports

  • Activity Log

  • Synchronisation

  • Data Import

  • Data Export Masters

  • Back-end Data
