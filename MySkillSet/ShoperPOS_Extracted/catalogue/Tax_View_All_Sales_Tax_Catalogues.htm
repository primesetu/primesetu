View All Sales Tax

The Sales Tax Catalogue window with the View All tab is as shown.

Effective From: Displays the date from which the sales tax is effective/ applied.

Source Tax Type: Displays the source of goods.

Destination Tax Type: Displays the destination of goods.

Product Tax Group: Displays all the product tax groups selected.

Tax Description: Displays all the tax description entered in Tax Description field or copied from Product Tax Group.

Comp. No. : Tax may be made up of multiple components like Sales Tax, Surcharge, Cess, etc. Enter the number of such components.

Comp. No.: Displays the serial number of each component.

Tax Rate: Displays the rate of tax for the component.

Comp. Desc: Displays  the description for the component.

Computed On: Displays the item based on which the tax component is computed.

Clear: To go to Tax Details tab

Print: To print the Sales Tax Catalogue Listing
