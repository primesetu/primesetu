Communication Details for Mail

The Communication Details window is as shown.

Configuration Name: Select the required configuration, if already created.

Create: Select, if you want to create a new communication configuration. On selecting this option, the Communication Configuration window is displayed.

To: Enter the mail IDs to which the report has to be sent. Use semi-colon (;) to separate the IDs.

Cc: Enter the mail IDs to which copy of the report has to be sent. Use semi-colon (;) to separate the IDs.

Subject: Specify the subject line to be used in the mail, when sending reports.

Format: Select PDF, Excel, CSV or HTML (Direct View), to identify the format in which the report is to be sent.

Message: Specify the message to be sent as body of the mail.

Active: Select to specify that the current configuration is active and available for communication.

Ok: To save the details entered

Cancel: To close the Communication Details window
