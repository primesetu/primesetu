Denomination Details

The Denomination Details window is as shown.

Denominations Used In:

Transaction Name: Lists the transactions in which the denominations can be used.

Active: Shows whether the corresponding transaction is currently allowed to use the denominations being catalogued.

Denomination Details:

Caption: Enter the description of the currency denomination. These are displayed during transactions.

Denomination: Enter the numeric value (face value) of the corresponding denomination.

Active: While cataloguing a denomination, by default, it is marked as active. In case any denomination is not to be allowed in transactions, deselect it.

Ok: To save the denomination details

Cancel: To close the window
