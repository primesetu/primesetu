TallyAdmin Tool

TallyAdmin is an administrative tool that helps in managing license for a single product or different products from Tally. This tool is more useful when you are using a multi user version of Shoper 9.

A License Server is required if you are using a multi user version of Shoper 9.

During License Server installation TallyAdmin tool gets installed.

The system where you use TallyAdmin tool needs internet connectivity.

Note: In case you are using multi user version of Shoper 9, License Service can be installed on a machine which does not have internet connectivity, thus safe guarding it from external attack. Then you can use TallyAdmin tool from a different machine for managing the license.

 

How to reach: Start > Programs > Tally License Server > Tally Admin

The TallyAdmin window is displayed. The version of TallyAdmin tool and the list of products that are compatible with the current version of the tool are displayed.

The TallyAdmin window has four tabs. Click the tab name for more details.

  • License Server

  • Local License

  • License Service

  • File Format Conversion: This tab is not currently relevant for Shoper 9 users.
