Customer Catalogue

This option is used to create customer database. Once created, browsing can be done on various attributes of a customer like Name, Classifications, Profiles etc., during billing, to associate a sale to the required customer. This will help in generating customer related reports like Customer Offtake reports.

How to reach: Catalogue > Customer

The Customer Catalogue window is displayed.

Add: Click to record the details of a new customer. Click here for more information.

Edit: Click to edit an existing customer details. Enter the Customer Code to display the customer details. Make necessary changes as explained in Add option. Click Ok to confirm modification.

Delete: Click to delete an existing customer details. Enter the Customer Code to display the customer details. Click Ok to confirm deletion.

Note: While using the Edit or Delete option, press F2 to browse and select the customer code from the Advanced Customer Search browse.

Exit: To quit the customer catalogue option
