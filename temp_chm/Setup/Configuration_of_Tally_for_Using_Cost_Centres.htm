Configuration of Tally for Using Cost Centres

To enable Cost Centres in Tally:

  1. Go to Gateway of Tally

  2. Select the required company

  3. Press F11:Accounting Features

  4. Set Maintain Cost Centres to Yes

  5. Set More than ONE Payroll/Cost Category to Yes

  6. Accept to save

To create a Cost Category in Tally:

  1. Go to Gateway of Tally

  2. Select the required company

  3. Select Accounts Info. > Cost Categories > Create

The Cost Category Creation window is displayed.

  1. Enter ShoperStore in Name

  2. Set Allocate Revenue Items to Yes

  3. Set Allocate Non-Revenue Items to Yes

  4. Accept to save

To create a Cost Centre in Tally:

  1. Go to Gateway of Tally

  2. Select the required company

  3. Select Accounts Info. > Cost Centres > Create

The Cost Centre Creation window is displayed.

  1. Enter ShoperStore in Category

  2. Enter Shoper Company Code in Name

  3. Ensure that Primary is selected in Under

  4. Accept to save

A Cost Centre has to be created for each showroom.
