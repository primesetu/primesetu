Communication Configuration

This option is used to configure different modes of communication for alerts.

How to reach: Housekeeping > Communication Configuration

The Communication Configuration window is displayed. In this window you can execute the following functions

Add: Click to add the details for a communication mode. Click here for more information

View: Click to view an existing communication details defined for alerts. Select the configuration name and the details are displayed

Edit: Click to edit the existing communication details defined for alerts

Clear: Click to clear the entries made.

Exit: To quit the Communication Configuration option
