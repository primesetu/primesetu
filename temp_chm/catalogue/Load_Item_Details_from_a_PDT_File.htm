Load Item Details from a PDT File

The Select File window is as shown.

File Format: This is to identify the fields in the PDT file. Stock Number is selected by default and cannot be unselected. You may optionally select Retail Price and/or Dealer Price, which are the revised prices stored in the file. In case you do not have Retail Price and Dealer Price in the PDT file, you may specify the values in the Price Revision window.

Effective From: Click the list button to display a calendar and choose the date from when the new price revision has to be effective. If you are specifying the date here, you may specify the same in the Price Revision window.

Delimiter: Specify the character/ symbol used to separate the fields in the PDT file. If you do not specify any character, the default value ; (semi-colon) is considered as the delimiter.

Format: The format of data in the PDT file is displayed here based on the selections you have done in File Format. The order of fields cannot be changed.

File Name: Enter the path and name of the PDT file (.txt file). Alternatively, select the file name by clicking the button.

Load: To select the item details from the PDT file

Cancel: To cancel the selections made
