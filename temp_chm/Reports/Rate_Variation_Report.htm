Rate Variation Report

If the billing parameter Allow Rate Alteration in Billing is set to Yes, then you can change the rate during billing. The Rate Variation report provides information on items for which the rates have been changed during billing in the specified period. This report can be generated based on cash transactions, credit transactions, or both.

How to reach: Reports > MIS > Rate Variation

The Rate Difference window is displayed.

Date Range: By default, the From and To are the current Shoper date. You can enter any date range but the To date cannot be greater than the Shoper date.

Bill Type: Select Cash, Credit or Both to include the corresponding bills in the report.

View: To view the report

Exit: To close the window

 

Note: Shows only the bills in which the rate alteration was done during billing.
