HSN Mapping based on Item Classification

The Tally Interface - HSN Mapping window is displayed.

The grid shows five sheets based on which you can specify item selection criteria. The first sheet Product is active by default.

Click Select Product to select product codes from those existing in the Item Master. The Item Selection window is displayed. Select the required product Code or Desc and click Select. The selected product details are displayed in the grid.

Click the sheet name Brand to view the sheet. The Item Selection window is displayed. Select the required brand Code or Desc and click Select. The selected brand details are displayed in the grid. Note that the Select button changes with the sheet.

Similarly, you can select the other sheets and select the values as required.

Clear: To clear the selections made on the current sheet

Next: To view the HSN tab

Note: HSN code cannot be assigned for all items together. It need to be assigned based on any of the item classification codes.
