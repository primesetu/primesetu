Manage Users

This option is used to add new users, modify the details of existing users or deactivate existing users. You can also assign the user to groups and companies.

How to reach: Setup > Supervisory Functions > Security Management

On clicking Manage Users, the Security Management – User Management window is displayed.

Add: Click to add new users. Click here to learn more.

Edit: Click to modify the details of existing users. Click here to learn more.

Cancel: To clear the selection/ entries

Exit: To close the Security Management window
