Reset License

You can reset your license if you do not receive the unlock key after activation request. Resetting license will allow you to regenerate the activation request. This option is available after the activation request is sent.

The Manage License window with Reset License option is displayed as shown.

 

Reset License: To reset the license which will regenerate the activation request.

Proxy Configuration: To configure the proxy server to connect to the internet (if required).

List of License Servers: To list the License Servers accessible on the network.

Continue: To continue the reset process

A message requesting for Serial number is displayed.

OK: To continue

A message is displayed after resetting the license.
