Walk-in Details Report

The Walk-in report provides information on the number and percentage of walk-in entries that have been converted into sales on either weekly or daily basis. While the sales information is automatically picked up by Shoper 9, the walk-in counts have to be manually entered. This report provides information on the average transaction value and counts, against different reasons because of which the walk–ins could not be converted into sales. A walk-in report can be generated for a selected period. It helps to devise sales promotional plans or new schemes to convert more walk-ins into sales.

How to reach: Reports > MIS > Walk-in Details

The Traffic Report window is displayed.

Date Range: By default, the From and To are the current Shoper date. You can enter any date range but the To date cannot be greater than the Shoper date.

Select Option: Select Week Wise or Date Wise to display the walk-in details on weekly and daily basis respectively.

Week Start Date: Specify the start date of the week from when the data need to be included in the report. This option is enabled only if the Week Wise option is selected.

Transactions: Select With Void Bills or Without Void Bills to specify the type of bills to be included in the report.

Display Reasons By: Select Code or Description to group the report accordingly.

View: To view the report

Exit: To close the window
