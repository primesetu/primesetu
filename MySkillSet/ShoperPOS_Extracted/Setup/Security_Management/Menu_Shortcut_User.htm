Menu Shortcut – User

This option helps to restrict users from creating relevant menu options.

 The Security Management – Menu Short window is displayed.

 

Select: Click to browse and select the Menu option for which shortcuts needs to be created.

Shortcut Caption: Enter the desired shortcut caption.

Shortcut Type: Select the type of shortcut such as Right Panel, Left Panel or Icon

Icon: Place the desired icon in the Templates folder under Application folder and enter the image name so that desired icon will be displayed.

Ok: To confirm selection made

Cancel: To cancel the selection

Exit: To close the Security Management window
