Delivery Status Report

This report displays the status of items that are delivered/not delivered to the customers.

How to reach: Sales > Delivery Management > Delivery Status Report

The Delivery Status Report window is displayed.

 

Invoice Date Range: Enter/Select the opening and closing dates for which the report is to be generated. By default, the From and To are the current Shoper date. You can enter any date range but the To date cannot be greater than the Shoper date.

Filters:

Invoice No.: Enter the sales invoice no or press F2 to select the required sales invoice.

Customer Code: Enter the customer code or press F2 to select the required customer code.

Showroom Code: Enter the code of the showroom or press F2 to select the showroom code.

Show Delivered Items: Select to display details of delivered items.

Show Non Delivered Items: Select to display details of non- delivered items.

Sort On Invoice Number: Select to display the report based on invoice number.

Sort On Customer: Select to display the report based on customer code.

View: To view the report

Cancel: To close the window
