Manage License

This option is used to manage Shoper 9 licensing.

In Shoper 9 start-up screen select Manage License

The Manage License window is displayed as shown.

 

In this window you can do the following:

  • Activate License: To activate license for the first time for your organisation or to activate an additional site.

  • Reactivate License: To reactivate Shoper 9 license, if it is surrendered.

  • Configure Existing License: To share the license from a License Server in a multi user setup.

  • Buy Rental License: To rent Shoper 9 license by paying the requisite charges for a Month/Quarter/Year.

  • Work in Evaluation Mode: To open Shoper 9 in evaluation mode.

Note: Shoper 9 can be used in evaluation mode for 45 days with full features, functions and capabilities.

Proxy Configuration: To configure  the proxy server for connecting to internet, if required. Click here to learn more.

List of License Servers: To list the License Servers accessible on the network.

Continue: To continue with the selected licensing process.

 

Note: On invoking Shoper 9 if license is not activated, Manage License window appears. License can be activated using this window or may use Shoper 9 in evaluation mode by selecting Work in Evaluation Mode.
