Manage Vendor Schemes

The Schemes Definition window is displayed.

  

Operation Type: Select the appropriate value, Activate/Deactivate/Archive

  • Select the required scheme in the grid

  • Click OK to activate it

Deactivate: This process will render the scheme not available for selection.

Archive: This process will render the scheme saved/backed up for future use.

Show: Click this button to display the defined schemes.

Cancel: Click this button to stop/terminate the managing operations.
