Defining Personnel Incentives

The Personnel Incentives window is as shown.

Incentive Name: Enter a name for the scheme you are defining.

Applicable Period: Select Season, Week, Month, Date or Year from the list. The seasons need to be catalogued in the season catalogue and weeks need to be catalogued in the week catalogue.

Select the year from the next list. Based on the selection corresponding values are listed below. Select the required seasons, weeks, months or dates.

Items: Select All or Specific. If you have selected Specific, select the required item classification or Stock Number from the list on the left. The corresponding values are displayed in the grid on the right hand side. Make necessary selections.

Personnel: Select All or Specific. If you have selected Specific, select the required personnel’s from the list. Alternatively, you may click   to open the Browse window and select the required personnel.

Incentive Criteria (Based On): Select Total Value, Total Qty, Specified Bill Value or Highest Bill Value.

Total Value – the cumulative net sales value for the period

Total Qty – the cumulative net sales quantity for the period

Specified Bill Value – if a single bill values exceeds a specified limit

Highest Bill Value – the bill value which is the highest in the period

In case you have selected  Total Value, Total Qty or Specified Bill Value, enter the target value under Condition (Greater than).

Incentive Values:

In Terms Of: Select Percentage or Amount.

Percentage Of: If you have selected Percentage, select the value on which the percentage is to be calculated. These values vary based on the incentive criteria.

Total Value – Total Value and Above Target

Total Qty – Total Value

Specific Bill Value – Bill Value and Above Target

Highest Bill Value – Bill Value

Value: Enter the percentage or amount against Value.

Apply: Click to display the details in the grid.

The values displayed in the grid can be modified, if required.

OK: Click to save the details.

Cancel: Click to clear the values selected/ entered in the window.
