General

General submenu allows you to configure various features as per your business requirement. Additionally, it facilitates necessary configuration for Tally posting and certain Tally interface utilities.

Various options available are:

  • System Parameters

  • Transaction Prefix

  • Stock Number Methodology

  • Printing Templates

  • Flat File Value

  • Cash/ Purchase Order Prefix

  • Tally Interface Mapping

  • Tally Interface – HSN Mapping

  • Print Engine Configuration

  • Entry Panel Column Configuration

  • Tally Interface Utilities

  • Batch & Grade Configuration
