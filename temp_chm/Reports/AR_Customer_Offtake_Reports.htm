Customer Offtake Reports

How to reach: Reports > Analysis Reports > Customer Offtake

Summary Report  

The summary report provides information on sales made to customers for the selected period. In detail this reports will show purchasing pattern.

Detailed Report

The detailed report provides bill-wise transaction details of customers.

Brand-wise Report

The Brand-wise report provides detailed sales information of different customers. This report is used to keep track of the items purchased by customers for a given period.
