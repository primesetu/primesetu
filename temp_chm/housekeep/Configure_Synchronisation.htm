Configure Synchronisation

This option is used to set the mode of communication for different HO(s) during the synchronisation process between POS and HO.

How to reach: Housekeeping > POS-HO Synchronisation > Configure Synchronisation

The Configure Synchronisation window is displayed. In this window you can execute the following functions.

Check Communication Mode: Click to test whether the system is able to transfer and receive data using the selected mode of communication. Click here for more information.

Add: Click to add communication details for a different HO. Click here for more information.

Edit: Click to modify an existing communication details specified for an HO. Enter the Head Office code and the details are displayed. Make the necessary modifications and click Apply to save the changes made.  

Delete: Click to delete an existing communication details. Enter the Head Office code and the details are displayed. Click Apply to confirm deletion.

Exit: To quit the Configure Synchronisation option.
