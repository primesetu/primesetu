Item Identification - New

The Item Identification window is displayed.

Prefix and No.: The fields, Prefix and No. are populated automatically with details.

Person in Charge: Select the person from the drop down box. Person in charge information can be recorded in Personnel Catalogue where the Type (person) will be selected as Person In Charge Types.

Invoice Prefix/No.:Press F2 and select the bill prefix and the bill number from the Browse window.

The Browse window is displayed.

 

Previous: To view the previous page

Next: To view the next page

Ok: Click to accept the selected documents into the Item Identification window

Cancel: Click to close the window without accepting the selected details

 

Showroom Code: The code of the billing Showroom is displayed.

Stock No.: Press F2 to select the item from the Browse window. This will be enabled only if the selected bill has more than one stock number.

The Browse window is displayed.

Previous: To view the previous page

Next: To view the next page

Ok: To accept the selected items into the item identification window

Cancel:To close the window without accepting the selected details

 

Item Description: Displays the item description of the selected stock number.

Location: Press F2 to select the location from where the item have to be delivered.

The Browse window is displayed.

Select the required service location.

Note: In the Item Identification stage, the location will be always current company location. If the billing location is doing the item identification, then only the local locations of that store will be displayed here. If the service location is doing item identification, then the local location of the service location company only will be displayed here.

Ok: Click to accept the selected service location

Cancel: Click to close the window without accepting the selected details

The Stock No., Item Description and Location fields gets populated.

Qty: Enter the required quantity of item.

Item Tag Information: Click F2 to select the Serial No. / IMEI No.

The Item Tag Information window is displayed.

 Serial No. / IMEI No.: Click F2 to select the Serial No. / IMEI No. or enter Serial No./IMEI No. and press tab.

Note: You can configure up to 5 columns by configuring the required System Parameter under Item Tag category.

If it is a local delivery process, i.e., billing location itself is delivering and serial no. is set to mandatory in the all transactions, then the serial no. entered while billing will be populated automatically while doing IID.

If it is a remote delivery and srl. no. is set to mandatory, then during IID, the srl. no. is assigned.

 

 

The Browse window for selecting the Serial No. / IMEI No. is displayed.

Ok: To accept the selected Serial No. / IMEI No.

Cancel:To close the window without accepting the selected details

 

 

The selected stock items details are displayed in the grid.

Show Item Tag: Select to display the item tag information.

The Item Tag Information window is displayed for the selected Item.

Close: To close the window.

Hotkeys:

Del: To delete row

Esc: To clear entry grid

Double click:To edit row

Save: To save and print the identification details.

Cancel: To clear the selected details.
