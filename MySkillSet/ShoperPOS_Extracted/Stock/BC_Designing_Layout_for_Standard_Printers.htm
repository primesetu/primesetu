Designing Layout for Standard Printers

This option is used to design the layout for the barcode label to be used on standard printers before printing. You can configure the page size, page format and number of rows and columns to be printed in a label.

How to reach: Stock > Barcode > Design Layout for Standard Printers

The Layout Design for Standard Printers window is displayed.

Load Label Design:

Select Label Design File: Enter the file name. Alternatively, click to browse and select the .btf file created in Barcode Label Designer.

The fields File Name, File Path, Unit of Measurement, Width and Height are populated when the label design file is selected.

Label Design Preview: Click here for a preview of the label design. Click Close to exit the Label Design Preview window.

Page Set Up:

Page Size: Select the page size. The page size can be A3, A4, A5, A6, Letter, Legal or Custom.

Page Height: Displays the height of the selected page in the unit of measurement selected.

Page Width: Displays the width of the selected page in the unit of measurement selected.

Side Margin: Enter the value for the side margin of the page. The selected unit of measurement would be used.

Top Margin: Enter the value for the top margin of the page. The selected unit of measurement would be used.

Horizontal Pitch: Enter the horizontal pitch for two labels. If there is no gap between the labels then the horizontal pitch is equal to the label width.

Vertical Pitch: Enter the vertical pitch for two labels. If there is no gap between the labels then the vertical pitch is equal to the label height.

Number Across: Enter the number of columns to be printed in a page. The number entered here when multiplied by the sum of label width and horizontal pitch should not exceed the page width, i.e., Side Margin + (Width * Number Across) + [(Horizontal Pitch - Width) * (Number Across - 1)] should not be more than the page width.

Number Down: Enter the number of rows to be printed in a page. The number entered here when multiplied by the sum of label height and vertical pitch should not exceed the page height, i.e., Top Margin + (Height + Number Down) + [(Vertical Pitch - Height) * (Number Down - 1)] should not be more than the page height.

Layout Design Preview: Click here to preview the layout design.

Saved File Name: The layout design file name, which is open, is displayed.

Saved File Path: The path of the layout design file, which is open, is displayed.

Save: To save the configuration as a .blf file. If you have modified an existing file, it gets saved in the same file. Otherwise, it creates a new file.

Save As: To save the configuration as a new .blf file. This is useful when you want to create a new layout design using an existing barcode layout file as the base.

Open: To load a saved .blf file and you can modify the parameters.

Clear All: To clear all the values entered.

Exit: To close the layout design process.
