Style Catalogue Listing

This option is used to generate a report to display summarized information of the inventory for the set filters at the Style level (Item SubClass1). You have a provision to drill down the report to the next lower level of hierarchy, up to the Stock Number (SKU) level in the following order:

Item SubClass1 (Style) > Item SubClass2 (Shade) > Size/UOM > Stock Number (SKU)

How to reach: Catalogue > Listings > Style Catalogue

The Style Catalogue View window is displayed.

Report Filters: Apply the required filter criteria on the report at four item classification levels namely, Super Classification1 (Department), Super Classification2 (Buyer), Item Classification1 (Product) and Item Classification2 (Brand). The availability of filters and the captions displayed vary depending on the item classifications selected for each installation.

Note: The Code and Description displayed under each item classification is dependent on the values catalogued under respective category at Catalog > General Lookup.

To apply filters,

  • Check the box to the left of the item classification title (example, Department) or press Alt+1 to display all the catalogued values.

  • Click Select All to generate report for all the values displayed.

  • To generate a report only for the required values, select the box to the left of each of the value.

Note: Repeat the process for each of the item classification, as required. To apply filter, remember to use the digit displayed on the left side of each classification along with Alt key.

Code: To display item classification codes in the report.

Description: To display item classification descriptions in the report.

Both: To display both item classification codes and descriptions in the report.

Merge Rows: To make the report more readable by displaying unique values in different columns of the adjacent rows.

View: To generate the report for the applied filters.

Press F5 or click   to hide/ show the filter pane, to display the report with or without the frame containing the selected filter conditions and action buttons.

To drill down a report, place the cursor on the value of a particular row of the report for which the next level report is required and press Enter or Double Click. The drilled down reports are displayed in separate tiled windows. Press Esc to move the cursor control to the previous higher level report in the hierarchy.

Note: The mouse pointer will change to a pointed hand in the columns of the Summary screen wherever drill down is possible.

 

Clear All: To undo all selections made and specify new filters.

Exit: To quit the Style Catalog View option.

The sample reports at different levels of hierarchy for a given filter condition are shown.

Summary Level report: Double click on any cell in the grid to drill down

 

Drilled down report for a selected Sub Class1 (Style): Double click on any cell in the grid to drill down.

  

Press Ctrl+1 to move to Summary Report.

Drilled down report for a selected Sub Class2 (Shade): Double click on any cell in the grid to drill down.

Press Ctrl+2 to move to Level 1 (Item SubClass1) report.

Drilled down report for a selected Size/UOM: Double click on any cell in the grid to drill down.

Note: The preview of images in Style Catalogue View is available if the system parameter Image ID Present In Itemmaster under the category Item Master is selected.

 

Press Ctrl+3 to move to Level 2 (Item SubClass2) report.
