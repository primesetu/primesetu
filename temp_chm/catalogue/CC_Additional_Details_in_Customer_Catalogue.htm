Additional Details in Customer Catalogue

The Additional Details tab in the Customer Catalogue window is displayed.

 

Enter the values for fields pertaining to Payment Details, Price and Tax Factors, LST/CST Details, Shoper Details and Tax Form Details.

Back: Go to Retail Details tab to change the details furnished earlier.

Ok: To save the record to the customer database. Where Customer code is set to be auto-generated, the code so assigned to the customer is displayed for future reference.

Cancel: To quit the option.
