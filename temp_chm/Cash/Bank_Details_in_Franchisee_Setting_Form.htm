Bank Details

This tab displays the details about the bank, which is taking care of all the transactions of the franchisee with its HO.

Note: The opening balance can be edited according to the system parameters setting. The system parameter Franchisee Year Opening Setting under the category Franchisee has to be set to 0 or 1 to make opening balance editable or 2 to make it non editable. The Opening Balance can be altered only if the shoper date is not having any payment related transaction to HO.

Pay-In Slip Name: Enter the caption that gets printed in the Pay-in Slips.

Bank Name (Chq Deposit): Enter the name of the bank in which all cheques of the franchisee get deposited.

Bank Address:Enter the address of the bank.

Franchisee Opening Setting

Opening Date (Shoper Date): Select the Shoper Date on which this form is created.

Cash Opening Balance: Enter the initial cash outstanding amount.

Credit Opening Balance: Enter the initial card outstanding amount.

Update: To update and save the changes made. You are prompted to provide the values for the mandatory fields before the information is saved into the database.

Exit: To exit from the Franchisee Details window.
