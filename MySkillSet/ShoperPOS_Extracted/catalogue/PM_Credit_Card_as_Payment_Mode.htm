Credit Card as Payment Mode

Payment Code: Enter the code for credit card, e.g. CC.

Applicable: Select/Deselect as per requirement.

Issue Agency: Enter the name of the agency that has issued the Credit Card. Press F2 to select from the list of agencies catalogued earlier. If the agency is not in the list then press Insert key or click the button to open the Agency Catalogue window and catalogue the new agency details.

Submission Agency: Select if the credit card used in billing are to be submitted to the specified agency.

Submission Online: Select if the specified agency allows online authorisation.

Commission Rate: Enter the commission rate charged by the agency when the credit card is submitted for realisation.

 

Number of Components: Captures the number of additional components required for accounting the amount received from the credit card servicing agency.

For example:

  • Credit Card number while accepting Credit Card as the mode of payment in Billing process

  • Batch number during credit card Submission process

  • Cheque number during Realization process

The captions to be displayed for accepting these components in various screens and the type of field to be captured for each component is to be provided in the Submission Details tab.

Note: The number of fields and captions in the credit card Billing, Submission and Realisation vary based on the payment mode catalogue settings. There can be a total of 7 components in the payment mode.

Billing: Enter the number of additional fields to be captured during billing process.

Submission: Enter the number of additional fields to be captured during submission process.

Realization: Enter the number of additional fields to be captured during realisation process.

Caption: Enter relevant captions to be applied at each point.

OK: To save the values entered.

Cancel: To clear the values entered. Clicking again on this button cancels the selected operation.

Exit: To close the window
