Creation of PT File

This option helps you to create flat file pertaining to the data of selected transaction type. These flat files can be used to avoid entering the item details again in Shoper transaction. For example, Purchase transaction file can be used to populate the items grid in goods inward option.

How to reach: Housekeeping > Data Export Masters > Create PT File

The Flat File Creation window is displayed.

Transaction Type: Select a transaction type from the list, i.e., Sales Invoice, Transfer Out, Purchase Return, Approval Issue, Miscellaneous Issues, Sales DC or Approval Issue DC for which the flat file has to be created.

Range Selection: Creation can happen for a range of document numbers or dates. Depending on the selection made, the date range or document range appears.

Document Number: Select to opt for document number as the criteria for creation of the flat files.

Document Date: Select to opt for document date as the criteria for creation of the flat files.

Note: Depending on the transaction type selected, a check box - Customer, Chain Store or Vendor is displayed. Selecting this allows you to create the file for Specific or All the Customers, Chain Stores or Vendors as the case may be.

 

Create: To create the flat file in the Shoper-out directory. The name of the file created will be displayed and it changes based on the transaction type selected.

Exit: To quit this option
