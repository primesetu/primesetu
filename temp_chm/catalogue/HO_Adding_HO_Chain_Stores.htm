The Mailing Address window is displayed.

The number of address columns on this screen depends on the number of parameters configured under the category Types of Address in General Lookup. The default is Billing Address. You may modify the existing one or add any other address parameters as required. In the given Mailing Address window, the default parameter is modified to Mailing Details. The captions Address1, Address2, Address3, Address4, Address5 or Default Address can be changed by setting the corresponding system parameters under the category Customer, if required.

The H.O./Store Code and Name are displayed in the Code and Name fields respectively. The Code and Name displayed cannot be altered.

Enter the remaining address details as required.

You may set this address or one of the addresses (if you have more than one parameter configured), as Default Address. This address would be used for all correspondences.

Ok: To confirm and save the mailing details of Head Office/Store

Cancel: To clear the details entered

Adding HO Chain Stores

The H.O/Stores Catalogue window is displayed.

H.O/Stores Code: Enter code for H.O./Stores and the name.

H.O/Store Type: Select the type of store from the list, i.e., Head Office, Ware House, Distribution Point, Retail Store.

Address: Click on Enter button to record contact information about H.O./Store. On clicking this option, the Mailing Address window is opened to enter the mailing details of the Head Office/chain stores.

Check if Goods Inwards from this location is allowed: Select if Goods Inwards or in general any inward movement of stock from the selected H.O./Store to the POS location is allowed.

Check if Goods Outwards is allowed to this location: Select if Goods Outwards or in general any outward movement of stock to the selected H.O./Store from the POS location is allowed.

Note: These two check boxes will be inactivated if the entered H.O./Store type is Head Office

 

Details of Shoper

The information under this heading is applicable if the H.O./Store whose details are being catalogued is also a Shoper user. These details help in making the data exchange easier.

Comp Code: Enter the Shoper company code of the H.O./Store

Environment: Select Retail or Distributor depending on the environment in which H.O./Store operates.

Flat File Format: Select the appropriate format from the list in which the flat file has to be created.

Delimiter: A semicolon is set as the default delimiter for creation of data files. You can change the delimiter, if required. This field will be disabled, if the Flat File Format selected is either DOS Format or GUI Format.

Buying Factor/ Selling Factor: Enter the factor (e.g., 10) to be applied as a rate or amount on different prices while creating flat files for any type of transaction. The factor set here is used in the option Setup > General Setup > Flat File Value.

Ok: To confirm and save the details entered

Cancel: To clear the details entered
