Multiple Price Setting

You can set multiple prices for all items or for the items having the class1 + class2 combinations for which multiple-wise prices are required, as set in Product + Brand Combo Settings. Multiple prices can be set for specified items also.

How to reach: Masters > Define Multiple Prices > Multiple Price Setting

The Multiple Prices window is displayed. In this window you can execute the following functions.

Add: Click to create a new set of prices for the required items. Click here for more information.

View: Click to view the multiple prices report. Click here for more information.

Manage: Click to manage the multiple prices report. Click here for more information.

Cancel: To undo all selections made and select a new function from the Multiple Prices window

Exit: To quit the Multiple Prices setting process
