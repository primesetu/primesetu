Price List

It is basically a report that gives detailed price list of all products present in the item master.  If there exists multiple prices for an item, all the prices are listed in the report.

The Price List window gives options to generate the price list for specific or all Products and Brands.

How to reach: Catalogue > Listings > Price

The Price List window is displayed.

 

Report can be generated based on Product and Brand.

Report Options

Product:

All: Select to generate price list of all available products.

Specific: Select to generate price list of a specific product.

Please select a Product: Select an item from the list displayed. This option is activated only if Specific is selected.

Brand:

All: Select to generate price list of all available brands.

Specific: Select to generate price list of a specific brand.

Please select a Brand: Select an item from the list displayed. This option is activated only if Specific is selected.

View: Click to view the price list for the specified products/ brands

Exit: Click to close price list generation form
