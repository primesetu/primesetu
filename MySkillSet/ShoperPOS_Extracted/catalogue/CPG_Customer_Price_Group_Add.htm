Adding Customer Price Group

The Customer Price Group window is as shown.

 

Code and Description: Enter customer price group code and a meaningful description for the entered code.

Payment Terms: Select the type of payment terms from the list.

Dest. Tax Type: Select a destination-wise tax type code from the list. The values in the list are those catalogued in General Lookup. This can be set only when the system parameter Destination-wise Tax Grouping Applicable under the category Tax is enabled.

Credit Days: Enter the number of days in which the credit has to be cleared by the customers in this group.

Credit Limit: Enter the credit amount allowed to the customers in this group.

Transactions Allowed:

Credit Invoice: Select to allow credit sales to the customers in this price group.

Cash Invoice: Select to allow cash sales to the customers in this price group.

Tax Exclusive Invoice: Select if items have to be billed as tax exclusive for the customers in this price group.

Misc. Issue: Select to allow miscellaneous issues to the customers in this price group without invoicing.

Ok: To confirm and save the details

Cancel: To clear the details entered
