Salesperson Sales Summary

The Salesman-Wise Sales Report provides details of the quantity and value of sales, sales returns and sales cancellations for all salesmen or a particular salesman. This report is available in detailed and summary formats. It may be used to analyse the performance of salesmen, fix targets and reward performance.

How to reach: Reports > Sales > Salesperson Summary

The Salesman-Wise Sales Report window is displayed.

Select Type: Select Product, Service or Both to include the corresponding items in the report.

Date Range: By default, the From and To dates are the current Shoper date. You can enter any date range but the To date cannot be greater than the Shoper date.

Type Of Report: Select Summary or Detail to generate the corresponding report. Detailed report gives item level sales and returns information for each salesman whereas summary report shows total sales made by each salesman.

View: To view the report

Exit: To close the window
