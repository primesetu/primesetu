Menu Access Control – Node

This option helps to restrict nodes from accessing relevant menu options.

The Security Management – Menu Access Control node window is displayed.

Node ID: Enter / Browse the Group ID for which the permissions have to be restricted.

Node Name: The name corresponding to the ID is displayed.

Menu Descriptions: Displays the Shoper 9 menu tree. The options that can be currently accessed by the selected group are checked while the inaccessible items are unchecked.

You may click to expand all options, to shrink the tree structure, to select operations or double click to clear the selections made.

Ok: To confirm selection made

Cancel: To cancel the selection

Exit: To close the Security Management window
