Sales Order Conversion

Sales Order Conversion is provided at both Retail and Distributor environment. This option allows you to convert a Sales Order to a Sales Delivery Challan, Approval Issue Delivery Challan or Sales Advice Slip. The options, Sales Delivery Challan and Approval Issue Delivery Challan are available only in a Distributor environment.

Conversion can be done only for valid sales orders. Cancelled sales order can not be converted. From a sales order document, one or more line items can be selected and converted to a Sales Advice Slip. For example, from a sales order raised for 10 line items, you can generate a sales advice slip for seven line items and the balance three line items can be opened for conversion. Partial utilisation of converted document is not allowed.

If a converted document is cancelled, then the corresponding sales order can be reopened and reused for conversion once again.

If a converted document is consumed in billing and later if that bill is cancelled, the converted document can be reused in billing once again.

How to reach: Sales > Sales Order Conversion

The SALES ORDER Conversion window is displayed.

Sales Order

Doc Prefix: Enter the sales order prefix or select the required sales order from the Browse window that displays on pressing F2.

Doc Number: Enter the sales order number or press F2 to browse and select the document number. If the browse search was used for selecting the sales order at the Doc Prefix field, then the document number is displayed by default.

Doc date: Displays the document date of the sales order. This field cannot be edited.

Convert to: Select the type of the transaction to which the sales order is to be converted. The default transaction type can be set using General Lookup. In a retail environment, Sales Advice Slip is selected by default.

Customer Details: The customer details like customer code, customer name and address as captured in the sales order, are displayed in the respective fields.

Item Details: The item details like Stock Number, Item Description, Rate, Quantity and Sales Staff as captured in the sales order, are displayed in the grid.

Total Qty: The total quantity of items in the sales order is displayed in the bottom left corner of the SALES ORDER Conversion window.

Total Value: The total value of the sales order is displayed in the bottom left corner of the SALES ORDER Conversion window.

OK: This option allows you to confirm the conversion operation and displays the details of the conversion, i.e., the document prefix and the document number after conversion.

Clear: To clear the information captured for conversion

Exit: To exit the SALES ORDER Conversion window

OK: To confirm the sales order conversion.

When you convert a sales order to a sales advice slip, the converted sales advice slip can be recalled in billing.

The converted Sales Advice Slip when recalled in Billing is displayed.
