Alerts Management

An effective alert system is an inevitable part of a software solution for the smooth handling of it. This option is to ensure an efficient alert system which takes care of different types and modes of messages appropriate for different events or activities performed in Shoper 9. In a chain store environment, you can configure the alert details specific to each showroom at HO and send them to the respective showrooms.

The current functionality of alert management in Shoper 9 allows you to configure alert details for:

  • Price revision import/ Sales promotion import – sends alert based on configuration, when a new price revision/ sales promotion details are imported

  • Login password – sends alert based on configuration, when the user chooses to retrieve password.

How to reach: Housekeeping > Alerts Management

The Alerts Management window is displayed. In this window you can execute the following functions.

Add: Click to add the mode of communication for the alerts for a functionality. Click here for more information.

Delete: Click to delete an existing mode of communication defined for alerts for a selected functionality.

Save: Click to save the entries made.

Exit: To quit the Alert Management option
