E-Mail

The e-mail configurations sub-screen is as shown.

Yes: Select to export the report

No: Select to change export configurations

E-Mail Server: Select e-mail server from the Common Mail Servers list. The available e-mail servers are: User defined, Gmail, Hotmail and Yahoo Mail.

Server Address: The server address appears based on the E-mail Server selection.

The applicable server addresses are:

Server Address

E-Mail Server

<enter the address>

smtp.gmail.com:465

Gmail

smtp.live.com:25

Hotmail

smtp.mail.yahoo.com:465

Yahoo Mail

Use SSL: Select Yes or No. Default is: No for User defined and Yes for others.

Use SSL On Standard Port: Select Yes or No. Default is: Yes for Hotmail and No for Yahoo & Gmail. This field is disabled for User defined and you need to set Yes in Use SSL to enable this.

From: Enter the name of the sender.

From E-Mail Address: Enter the e-mail address of the sender.

Authentication User Name:  Enter the user name, if the mail server requires authentication.

Password: Enter the password of the authentication user.

Format: Select the required format from the Format list. Click here to learn more about formats.

To E-Mail Address: Select the e-mail address from the Mail Addresses list. If you select New Address, the New Address sub-screen appears. Enter the e-mail address.

CC To (if any): Select the e-mail address from the Mail Addresses list. If you select New Address, the New Address sub-screen appears. Enter the e-mail address. You may also select None from the list.

Subject: By default the report name appears. Change, if required

Additional Text (if any): Enter any additional details, if required. You can enter a maximum of 251 characters in this field.

Information sent: Select As Attachment (Recommended for Printing) or For Direct viewing in Mail from the Mail Sending list.
