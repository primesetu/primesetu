Hotkeys

Alt + H: To show and hide the list of hot keys

F4: To delete an item in the grid

F7: To save a transaction

Up/Down arrow: To move up and down in the grid

Ctrl + T: To toggle between Item Description and Item Classification in the grid

Ctrl + Alt + H: To trigger the header level extension, if any

Ctrl + Alt + D: To trigger the detail level extension, if any

Ctrl + Alt + F: To trigger the footer level extension, if any
