Goods Inwards/ Outwards Prefix Definition

This option is used to define the document prefix and number for all the transaction types applicable during goods inwards and goods outwards.

How to reach: Setup > General > Goods Inwards / Outwards Prefix

The Doc Prefix / Doc Number / Control Number selection for GOR and GIR window is displayed

Select Inwards/ Outwards from the dropdown list

Current Year Suffix: Displays the current year as suffix

You can alter the default Doc Prefix and Doc Number if required for any Trn Type displayed in the grid

Save: To consider the changes made

Exit: To quit the option
