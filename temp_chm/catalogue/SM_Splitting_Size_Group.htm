Splitting Size Group

The Size Catalogue Management window is as shown.

All the available size groups are displayed in the grid. Choose the size group to be split. Select the size codes to be moved to the new size group.

New Size Group ID: Enter the ID for the new size group

New Size Group Description: Enter description for the new size group

Split: Click to display the new size group details in the grid above. All Product-Brand combinations and the size groups for the combinations are displayed in the grid below.

To associate a Product-Brand combination with the newly created size group, identify the Product-Brand combination and select the New Size Group ID. The corresponding Size Group Description is displayed.

Apply: The new size group is applied to the selected Product-Brand combination

Cancel: To cancel the selected functionality

Next: To rearrange the size codes

 

Note: When you split a size group, a new group is created with the selected subset of size codes. If there are masters associated to the size group, the original group will remain in tact. Otherwise, the size group will be split with discreet size codes.

The Size Catalogue Management window changes as shown.

Size Group List: The available size groups are displayed. Select the size group for which the size codes have to be rearranged. The corresponding Size Codes and Serial Numbers are displayed on the right hand side.

Select size code/s to be moved up or down. Click Move Up to move the selected codes up or click Move Down to move the selected codes down. Repeat until the required sequence is arrived.

Add: To add a new size code at the end of the list

Insert: To add a new size code above the selected code

Delete: To remove the selected size code, if not used so far

Previous: To move to the Split Size Groups screen

Next: To select the applicable size codes

The Size Catalogue Management window changes as shown.

Select the size group to be modified from the list. The corresponding Product-Brand combination and the currently applied size codes are displayed on the right hand side. Choose the relevant size codes for each Product-Brand combination. Colour change (Not Applicable, Applicable and Applicable + Pivotal) indicates the status of each size code.

Cancel: To cancel the selected functionality

Previous: To move to the Rearrange Sizes screen

Save: To save the new size group
