Print Mail Communication

 

Distribution List: Select the required distribution list of mail recipients   

Select Fields: Place the cursor at a required location in the body of the mail and select the field from this list..

Add: Click to add the selected field where the cursor was placed. The value of this field for each of the recipient is replaced from the database in the mail

 

Select to format the text, i.e., font, font size and text alignment.

Insert File: To insert text from word file into the body of the letter

Save: To save the document in word format in the directory of your choice.

Print: To print the letter

Cancel: To clear the entries made

Exit: To quit the option
