Load Inward to Outward

The Load Inward to Outward window is as shown.

Note: The process of loading inward doc to outward is known as cross-docking.

Transaction Type: Select the transaction type. The available transactions are Purchase, Transfer In and Misc Rcpts.

Inward Doc. Prefix: Enter the prefix of the inwards document.

Inward Doc. No.: Enter the number of the inwards document.

Ok: To fill the details in the outwards grid and close this window

Exit: To close this window without copying the details to the grid
