Node-wise Details

This option generates a sales report detailing sales node wise. A detailed report or a summary report can be generated. In the case of a standalone installation, this report gives you collection details at the counter. When there are multiple counters, this report gives collection details at each counter individually.

How to reach: Reports > Sales > Node-wise Details

The Node Wise Report window is displayed.

 

Date Range: By default, the From and To dates are the current Shoper date. You can enter any date range but the To date cannot be greater than the Shoper date.

Criteria Selection: Select Node – Detail Report or Node – Summary Report to generate a detailed or summary report.

View: To view the report

Exit: To close the window
