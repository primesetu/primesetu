Trip Sheet Creation

This menu is used to create and print the trip sheet for door delivery of stock items to customers from the store or remote locations.

How to reach: Sales > Delivery Management > Trip Sheet Creation

The Trip Sheet window is displayed.

Prefix: The prefix of the trip sheet is displayed by default.

No.: The number of the trip sheet document is generated and displayed by default.

Note: The date and time of creation is displayed.

Expected Delivery Date: Select to create the trip sheet for a time period.

From:  Select/enter the starting date of the period.

To: Select/enter the end date of the period.

Showroom: Select to create the trip sheet for a particular showroom.

Select All: Select to create the trip sheet for all showrooms.

Locality: Select to create the trip sheet for delivery to a particular locality.

Select All: Select to create the trip sheet for all delivery to all localities.

 

Transport Details: Select the appropriate details relating to the transport of stock items to the customer.

Driver Name: Select the name of the vehicle driver configured in the system parameters.

Vehicle Type: Select the type of vehicle used as configured.

Vehicle No.: Select the registration number of the vehicle delivering the stock items to the customer.

Person In charge: Select the person who is responsible to deliver the stock items to the customer.

Remarks: Type/enter the remarks in this field.

List Matching Items: Click to display the details of the trip sheet.

The Trip Sheet window is displayed.

   

Note: The fields, Print Invoice and Print DC are displayed if the system parameter Option Required to Print DC on Saving Trip Sheet and Option Required to Print Invoice on Saving Trip Sheet is set to Yes.

Print Invoice and Print DC in the display are selected by default. You may deselect them if required.

Reprint: Click to reprint the trip sheet after creation.

The Reprint window is displayed.

Trip Sheet Reprint

Doc Prefix: Type/enter the prefix of the trip sheet to be reprinted.

No.: Type/enter the number of the trip sheet to be reprinted.

The invoice and reference doc details are displayed.

Print Invoice: Select if you want to print the original invoice. The invoice may be local or remote.

Print DC: Select if you want to print the Delivery Challan.

Print: Click this button to reprint the trip sheet.

Exit: Click this button to close the Reprint window.

Save: To save and print the trip sheet.

Exit: To close the window.
