Report Designer

A Report Designer is used to configure the required fields of a report which are to be shown in the Tally window based on customer requirements. You can create and save different combination of particulars  and value fields as per the requirement. The Admin user at Shoper 9 HO/POS can assign the saved configuration to other users.

How to reach: Reports > Analysis Reports > Report Designer

Reports for the following  can be configured using the Report Designer in Shoper9 POS.

  • Sales

  • Stock

  • Sales and Stock

  • Stock as on Date

  • Tender

  • Slips
