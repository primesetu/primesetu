Configure Existing License

This option is used when you want to share license from the License Server in a multi user setup. In case of single user it is used when the license is in a different folder. The client installations can use this option to connect to the server and access the license running on the server.

In Shoper 9 start-up screen select Manage License > Configure Existing License

The Configure Existing License window is displayed with options to configure license for:

  • Multi-User: Select in case of multi user and to share license from the License Server.

  • Single-User: Select in case of single user and the license is in a different folder.

Submit: To enable the license.

 

Note: When you restart the application, the license details are displayed in the Info Panel.
