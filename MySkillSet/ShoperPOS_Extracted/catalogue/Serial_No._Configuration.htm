Serial No. Configuration

This option is used to configure the facility to accept serial number information of stock items during the inward, outward, billing, sales order and returns. This option will be useful in processing the above operations relating to white goods retail operations.

How to reach: Catalogue > Serial Number Configuration

The Serial Number Configuration window is displayed.

 

Item Classification: The stock item classifications, Product & Brand and the Applicable column are displayed for configuration.

Product: Click F2 to select the product code.

OK: Click to accept the selection.

Cancel: Click to terminate the selection.

Brand: Click F2 to select the brand code.

OK: Click to accept the selection.

Cancel: Click to terminate the selection.

Note: You can click the keys Alt + F2 to select product + brand combination for configuration.

Applicable: Select/deselect the product + brand combination for configuration.

Transactions: Select All or Specific transactions to be configured based on requirements.

All: If this option is selected, a default list of transaction types with mandatory boxes selected is displayed.

 

 

Specific: If this option is selected, select/deselect the required transactions and the mandatory boxes.

Note:

1. The column captions under Serial Nos are configured based on the requirements of the particular retail business. The columns are configured in the Item Tag Category under System Parameters.

2. Selection of a transaction is compulsory but it need not be made mandatory.

3. The configuration of the first column is important which affects the other columns.

 

Configure: Click this button to configure the serial number columns for any new product + brand combination.

Column Caption: The captions of the columns are displayed.

Data Type: Select Text/Number as required.

Min Length: Set the minimum length of the column, greater than the default value of zero.

Max Length: Set the maximum length of the column, limited to the default value of 32 characters.

Applicability: Select the values, Mandatory in All Transactions or Optional for the serial columns.

Display Attribute Properties: Select this option to display the minimum and maximum length of the serial number columns.

Apply: Click to save the configurations.

Close: Click to close the window without any change.

  

OK: Click this button to save the entire configuration. On clicking, a message, Data saved successfully, is displayed in the left bottom corner.

Cancel: Click this button to terminate the configuration.

Exit: Click this button to close the menu.
