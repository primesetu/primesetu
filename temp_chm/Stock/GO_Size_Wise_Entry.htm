Size Wise Entry

A sample size-wise grid for item selection is as shown.

Columns Selection: To modify the list of columns displayed in the grid, click Columns Selection. A list of optional fields are displayed. Select the required fields and click Close Selection.

The grid has columns to enter/ display item details and columns to enter quantity for different sizes. All size groups defined in the system are displayed as column headings.

Select the items by scanning/ entering item numbers. The item details are displayed. Once the item is selected, the corresponding size groups are highlighted and quantities can be entered against the required ones.
