Surrender License

This option is used when you want to surrender Shoper 9 license and reactivate the license later. It can be used in scenarios when you have planned a re-installation for changing the hardware (Hard disk or Mother Board), moving Shoper 9 from one system to another, changing the Operating System, etc.

This option is available only after activation of license.

The Manage License window with Surrender License option is displayed as shown.

Surrender License: To surrender the license.

Proxy Configuration: To configure the proxy server to connect to the internet (if required).

List of License Servers: To list the License Servers accessible on the network.

Continue: To continue the surrender process.

 

A confirmation message is displayed.

OK: To surrender and display the authentication window.

Cancel: To cancel the surrender process.

The authentication window is displayed as shown.

Your E-Mail ID: Enter the site/ account administrator E-Mail ID.

Your Tally.NET Password: Enter the Tally.NET password.   

The Manage License screen displays the options Activate License, Re-Activate License and Configure License.

Note: Shoper 9 license can also be surrendered using the Control Centre feature on the Tally web site. Use Tally.NET account to login to the Tally web site.

 

Caution: Using the license after surrendering is violation of the license regulations, which will lead to the cancellation of the license.
