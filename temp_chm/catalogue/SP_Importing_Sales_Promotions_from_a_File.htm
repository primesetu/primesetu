Importing Sales Promotions from a File

The Import Sales Promotions From File window is as shown.

Item Level: Select if the sales promo defined in the csv/ xls file is at item level.

Bill Level: Select if the sales promo defined in the csv/ xls file is at bill level.

Load File: Select the csv/ xls file in which the sales promo details are saved.

Help: To get help on how to define a promo definition in the csv/ xls file.

Sample Format: To view a sample of the promo definition in a .xls file. All the field names are displayed in the first row. Follow the guidelines given in the help to complete the promo and save the file. Enter the corresponding data in the rows below the header. Ensure that you do not alter the header. You may define multiple sales promos in a single file. Select this file name and import.

Import: To import promo details from the selected file.

Close: To close the window.
